# Installing uBO Scope

This guide provides clear, step-by-step instructions to install the uBO Scope browser extension on Chrome, Firefox, and Safari. Follow these actionable steps to add uBO Scope from each browser's official marketplace and verify a successful installation.

---

## 1. Prerequisites

Before installing uBO Scope, ensure you meet the following requirements:

- **Supported Browsers:**
  - Chrome (version 122 or later)
  - Firefox (version 128 or later)
  - Safari (version 18.5 or later)
- **Internet Connection:** Required to access your browser’s extension marketplace.

Ensure your browser is updated to the minimum version to enable uBO Scope compatibility.

---

## 2. Installation Instructions

### Chrome Installation

1. Open the Chrome browser.
2. Visit the Chrome Web Store page for uBO Scope:
   - [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)
3. Click **Add to Chrome**.
4. In the confirmation popup, select **Add extension**.
5. After installation, you should see the uBO Scope icon in the toolbar.

### Firefox Installation

1. Open the Firefox browser.
2. Go to the Firefox Add-ons page for uBO Scope:
   - [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)
3. Click **Add to Firefox**.
4. In the prompt, click **Add**.
5. Confirm any permission requests, then the uBO Scope icon will appear in your toolbar.

### Safari Installation

> **Note:** Safari extensions require Safari 18.5 or higher.

1. Open the Safari browser.
2. Open the **Safari Extensions Gallery** or visit the App Store to search for uBO Scope.
3. Locate uBO Scope by Raymond Hill.
4. Click **Get** or **Install** to add the extension.
5. Enable the extension via **Safari > Settings > Extensions**, if not enabled automatically.
6. The uBO Scope icon will appear in the toolbar once activated.

---

## 3. Verifying Successful Installation

After installation, ensure uBO Scope is properly installed and ready to use:

1. Look for the **uBO Scope toolbar icon** in your browser’s extension or toolbar area.
2. Click the icon to open the uBO Scope popup.
3. The popup should show the interface with a message like **NO DATA** initially.
4. Visit any website and reload the page.
5. Click the uBO Scope icon again; it should now display connection statistics for the active tab.

<Tip>
If the icon does not appear, check your browser’s extension manager and ensure uBO Scope is enabled.
</Tip>

## 4. Common Installation Pitfalls & Troubleshooting

<AccordionGroup title="Troubleshooting Installation Issues">
<Accordion title="uBO Scope Icon Does Not Appear">
- Confirm your browser version meets the minimum requirements.
- In browser extension/add-on settings, verify uBO Scope is enabled.
- If recently installed, try restarting your browser.
- Check for any browser policies or restrictions blocking extensions.
</Accordion>
<Accordion title="Unable to Find uBO Scope in Extension Store">
- Make sure you are searching in the official extension marketplace for your browser.
- Some regions or network restrictions may limit access; try a VPN if necessary.
- Check that your browser is fully updated.
</Accordion>
<Accordion title="Popup Shows NO DATA Even On Browsing">
- Wait a moment and reload the page you are on.
- Ensure network requests are occurring (try a different website).
- Confirm no firewall or privacy filters prevent the extension from monitoring.
</Accordion>
</AccordionGroup>
