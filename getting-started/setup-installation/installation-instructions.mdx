# Installing uBO Scope

Get up and running with uBO Scope quickly by installing it on your preferred supported browser. This guide walks you through the installation process for Chromium-based browsers, Firefox, and Safari, providing direct links to official extension stores and a summary of each store's installation steps.

---

## Supported Browsers for Installation

Before you proceed, ensure you are using one of these supported browsers:

- **Chromium-based browsers** (Google Chrome, Microsoft Edge, Brave, etc.) version 122 or later
- **Firefox** version 128 or later (including Firefox for Android)
- **Safari** version 18.5 or later

uBO Scope requires permissions such as `webRequest`, `storage`, and `activeTab` to report remote server connections accurately.

## Step-by-Step Installation Instructions

### 1. Installing on Chromium Browsers (Chrome, Edge, Brave)

1. Open your browser.
2. Navigate to the uBO Scope listing on the [Chrome Web Store](https://chrome.google.com/webstore/detail/ublock-origin-scope/).
3. Click **Add to Chrome** (or **Add to Edge/Brave**).
4. Confirm by clicking **Add extension** in the prompt.
5. Once installed, uBO Scope's icon will appear next to the address bar.

<Tip>
If your browser blocks extensions from less secure sources, ensure you only install from the official Chrome Web Store to avoid security risks.
</Tip>

### 2. Installing on Firefox

1. Open Firefox (desktop or Android).
2. Go to the uBO Scope page on [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ublock-origin-scope/).
3. Click **Add to Firefox**.
4. Accept any permission requests.
5. The extension icon will become visible near the address bar.

<Note>
On Firefox for Android, the extension will integrate smoothly with the mobile browsing experience, providing the same network connection insights.
</Note>

### 3. Installing on Safari (macOS)

1. Open the **App Store** on your Mac.
2. Search for “uBO Scope” or click the link provided.
3. Click **Get** and then **Install** to download the extension.
4. Open Safari and go to **Preferences > Extensions**.
5. Enable uBO Scope by checking its box.
6. The extension icon will appear in the Safari toolbar.

<Warning>
Safari requires macOS and Safari version 18.5 or later to support uBO Scope.
</Warning>

## Verification of Successful Installation

- The extension icon should be visible adjacent to the browser's address bar.
- Clicking the icon will open the uBO Scope popup interface.
- If prompted, grant the required permissions for the extension to function correctly.

<Check>
Test the installation by visiting any web page and observing the uBO Scope badge, which displays third-party connection counts.
</Check>

## Troubleshooting Common Installation Issues

- **Extension icon not visible:**
  - Ensure the installation completed successfully.
  - Check browser extension management to verify uBO Scope is enabled.
  - Restart the browser.

- **Permissions not granted:**
  - Some features depend on permissions like `webRequest`. Grant these when prompted.

- **Unsupported browser version:**
  - Verify your browser version is at least the minimum supported (Chrome 122+, Firefox 128+, Safari 18.5+).

- **Installation blocked by browser policies or security software:**
  - Ensure origins like official stores are trusted and accessible.

<Note>
Consult the [Installation Issues & Permissions](/getting-started/troubleshooting/troubleshooting-install) guide for detailed troubleshooting if these steps fail.
</Note>
