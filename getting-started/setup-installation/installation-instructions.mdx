# Installing uBO Scope

This guide provides you with clear, step-by-step instructions to install the **uBO Scope** browser extension from official stores. Whether you are using Chromium-based browsers, Firefox, or Safari, follow the relevant section below to complete the installation smoothly and securely.

---

## 1. Preparing for Installation

Before installing uBO Scope, ensure that your environment meets the prerequisites for a flawless experience:

- **Supported Browsers**:
  - Chromium-based browsers (Google Chrome, Microsoft Edge, Brave, etc.) version **122.0** or higher
  - Mozilla Firefox version **128.0** or higher
  - Safari version **18.5** or higher

- **Permissions Needed**:
  - The extension requires permissions for active tab access, storage, and network request monitoring through the `webRequest` API.

- **Connectivity**:
  - Ensure your browser can reach the respective official extension stores.
  - If your network restricts access to these stores, installation may fail.

<Tip>
Installing from official extension stores guarantees you receive the authentic and up-to-date version of uBO Scope.
</Tip>

---

## 2. Installing uBO Scope on Chromium-based Browsers

Follow these steps to install uBO Scope from the Chrome Web Store or any Chromium-based browser store:

### Step 1: Access the Chrome Web Store

- Open your Chromium-based browser.
- Navigate to the uBO Scope listing: [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

### Step 2: Add the Extension

- Click **Add to Chrome** or equivalent button.
- Review the permissions presented; uBO Scope needs access to:
  - Read and change data on websites you visit
  - Access browser tabs
  - Use storage
  - Observe network requests via `webRequest`
- Click **Add extension** to confirm.

### Step 3: Verify Installation

- Upon successful installation, you will see the uBO Scope icon on your browser toolbar.
- Hover over the icon to confirm it shows the title "uBO Scope".

<Tip>
If the extension does not appear on the toolbar, check your browser’s extension management page and pin uBO Scope manually.
</Tip>

### Common Pitfalls

- If you cannot access the Chrome Web Store, verify network restrictions or try installing via another Chromium-based browser.
- Avoid installing uBO Scope from third-party sites as they may be outdated or malicious.

---

## 3. Installing uBO Scope on Mozilla Firefox

Installation on Firefox requires similar steps, with attention to version compatibility and permissions.

### Step 1: Open Firefox Add-ons Site

- Launch Firefox browser.
- Navigate to uBO Scope on Firefox Add-ons: [uBO Scope on Firefox](https://addons.mozilla.org/firefox/addon/ubo-scope/)

### Step 2: Install the Extension

- Click **Add to Firefox**.
- Review the requested permissions:
  - Access to tab data
  - Storage
  - Monitor web requests with `webRequest`
- Confirm by clicking **Add**.

### Step 3: Confirm Installation

- After installation, the uBO Scope icon will appear in the toolbar.
- Click the icon to open the popup and familiarize yourself with the extension interface.

### Troubleshooting

- If Firefox prompts you for permission again later, ensure you have the latest browser updates installed.
- Disable other conflicting extensions temporarily if uBO Scope does not activate as expected.

---

## 4. Installing uBO Scope on Safari

Safari requires use of the App Store or manual packaging; the following instructions assume official App Store installation.

### Step 1: Open Safari Extensions

- Open Safari and access Safari Extensions via Safari menu > Extensions.
- Search for **uBO Scope** in the Safari Extensions Gallery or from the App Store.

### Step 2: Install and Enable

- Click **Get** to download and install the extension.
- After installation, enable uBO Scope in Safari's Preferences > Extensions.

### Step 3: Verify Icon and Permissions

- Look for the uBO Scope icon in your Safari toolbar.
- Safari may require you to grant permissions to control websites and monitor network requests.
- Follow on-screen prompts to allow necessary permissions.

<Tip>
Since Safari 18.5 and higher use Manifest V3, uBO Scope runs smoothly under this version but ensure you are updated.
</Tip>

---

## 5. Best Practices for Verifying Authentic Installation

After installing uBO Scope, verify you're running the official and latest version:

- Check the extension version through your browser’s extensions page; it should match or be newer than **1.0.1**.
- Confirm the author is listed as **Raymond Hill**.
- Verify the permissions requested match those outlined in this guide. Avoid extensions asking for excessive or unrelated permissions.

<Tip>
If in doubt, uninstall the extension and reinstall only from official stores using the provided links.
</Tip>
