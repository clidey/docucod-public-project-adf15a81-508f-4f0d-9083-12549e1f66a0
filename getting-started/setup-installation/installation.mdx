# Installation Guide for uBO Scope

Welcome to the definitive installation guide for uBO Scope, your essential privacy tool for monitoring remote server connections in your web browser. This guide walks you through installing uBO Scope across the three major browsers—Chrome, Firefox, and Safari—so you can start gaining real-time insights into your web traffic with confidence.

---

## 1. Where to Get uBO Scope

uBO Scope is distributed through the official web stores for each browser. Installing from these trusted sources ensures automatic updates and security compliance.

- **Chrome:** [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

- **Firefox:** [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

- **Safari:** Safari does not currently have a dedicated public listing; installation typically requires manual loading of the extension or using official Apple developer channels.


## 2. Installation Steps by Browser

### A. Chrome Installation

1. **Open the Chrome Web Store link:** Navigate to the official uBO Scope listing via the link above.
2. **Add to Chrome:** Click the 'Add to Chrome' button.
3. **Confirm the prompt:** Approve the required permissions including access to active tabs, storage, and network requests.
4. **Verify installation:** Once added, uBO Scope's icon will appear in your browser toolbar.

### B. Firefox Installation

1. **Open the Firefox Add-ons page:** Use the link above to visit uBO Scope's page.
2. **Click 'Add to Firefox':** Initiate the installation process.
3. **Grant permissions:** Confirm permissions such as webRequest and storage access.
4. **Look for the extension icon:** It should appear in your Firefox toolbar upon successful installation.

### C. Safari Installation

Installing uBO Scope for Safari may involve advanced steps as official store availability is limited:

1. **Obtain the extension files:** Clone or download the uBO Scope repository from [GitHub](https://github.com/gorhill/uBO-Scope).
2. **Open Safari Extensions:** Open Safari, go to Preferences > Extensions.
3. **Enable Developer Menu:** In Safari's Advanced tab, check 'Show Develop menu in menu bar'.
4. **Load the extension manually:** Use the Develop menu to select 'Show Extension Builder' and add the uBO Scope extension folder containing the `manifest.json` from the `platform/safari` directory.
5. **Launch and approve:** Enable the extension in Safari preferences.


## 3. Verify Successful Installation

After installing uBO Scope, verify it is active and functioning as expected.

- Look for the uBO Scope icon in your browser toolbar.
- When browsing, observe the badge count on the icon; it shows the number of distinct third-party domains connected per open tab.
- Click the icon to open the popup interface, where domains are categorized as "not blocked," "stealth-blocked," and "blocked."


## 4. Troubleshooting Installation Issues

If you encounter difficulties during installation or the extension does not appear to work correctly, try the following steps:

<Accordion title="Common Installation Issues and Fixes">

<Accordion title="Extension Icon Not Showing">
- **Ensure browser version meets minimum requirements:**
  - Chrome 122 or newer
  - Firefox 128 or newer
  - Safari 18.5 or newer
- **Restart the browser:** Sometimes changes require a restart to take effect.
- **Check extension settings:** Confirm that uBO Scope is enabled in the browser's extension manager.
</Accordion>

<Accordion title="Permissions Not Granted">
- uBO Scope requires permissions for `activeTab`, `storage`, and `webRequest`.
- If prompts are declined, uninstall and reinstall the extension, granting all requested permissions.
- On Firefox and Safari, check manual permission settings in your browser’s extension preferences.
</Accordion>

<Accordion title="Extension Not Recording Connections">
- Confirm you have active tabs with network activity.
- Make sure your browser is not blocking extensions from accessing web requests (some privacy settings or other extensions might interfere).
- Clear browser cache and try again.
</Accordion>

## 5. Additional Installation Tips

- **Use Official Sources:** Always install uBO Scope from official web stores or the trusted GitHub repository to avoid security risks.
- **Keep Your Browser Updated:** Newer browser versions enhance extension compatibility and security.
- **Check Badge Counts Regularly:** The badge reflects your real-time exposure to third-party connections — a lower number indicates better privacy.
