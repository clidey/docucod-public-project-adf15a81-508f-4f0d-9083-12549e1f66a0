# Installation Guide for uBO Scope

Welcome to the installation guide for uBO Scope, the browser extension that empowers you to monitor all remote server connections your browser makes. This guide provides clear, step-by-step instructions tailored to your browser to download, install, and ensure uBO Scope is ready for use.

---

## Supported Browsers and Compatibility

Before installation, verify your browser meets the minimum requirements:

- **Chromium-based browsers (Google Chrome, Microsoft Edge, Brave, etc.)**: Version 122 or newer
- **Mozilla Firefox**: Version 128 or newer
- **Safari**: Version 18.5 or newer

<u>Note:</u> Older versions may lack APIs required by uBO Scope to function correctly.

---

## Installation Steps

### 1. Chromium-based Browsers

#### Step 1: Open the Chrome Web Store
Click the following link to navigate directly to the uBO Scope extension page:

[Install uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

#### Step 2: Add Extension
- Click the **Add to Chrome** button.
- Confirm by selecting **Add extension** in the popup.

#### Step 3: Confirm Installation
- Look for the uBO Scope icon in your browser toolbar.
- The icon should be active without error indicators.

### 2. Firefox

#### Step 1: Open the Firefox Add-ons Site
Visit the official uBO Scope page here:

[Install uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

#### Step 2: Add to Firefox
- Click on **Add to Firefox**.
- Accept any permission requests that appear.

#### Step 3: Verify the Extension
- Look for the uBO Scope icon near the address bar.
- Ensure the extension is enabled.

### 3. Safari

#### Step 1: Access Safari Extensions
- Open Safari's menu > **Settings** (or Preferences) > **Extensions** tab.

#### Step 2: Enable uBO Scope
- Locate uBO Scope in the list and enable it.
- If you have the development version or manual installation, follow the developer instructions for installing via Xcode or manual load.

> **Note:** The Safari version requires Safari 18.5+ and may require enabling developer mode if installing outside the App Store.

---

## Post-Installation Checks

After installation:

1. **Confirm Toolbar Icon** appears.
2. **Click the icon** to open the popup interface, which should display `NO DATA` if you have not yet loaded any web pages.
3. **Navigate to any webpage**; uBO Scope will start tracking remote server connections and update the badge count.

## Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting Installation Issues">
<Accordion title="The uBO Scope icon doesn't appear after installation">
- **Check if the extension is enabled** in your browser's extensions management page.
- Try restarting the browser.
- For Chromium, ensure you installed from the official Chrome Web Store.
</Accordion>
<Accordion title="Permissions or access prompts during installation">
- uBO Scope requires access to read network requests (`webRequest` permission).
- Review the permission prompts carefully; denying permissions will prevent functionality.
</Accordion>
<Accordion title="Installation fails or the extension crashes immediately">
- Confirm your browser version meets the minimum requirements (see Supported Browsers).
- Check for conflicts with other extensions that also monitor network requests.
- For Safari, verify system compatibility and that you followed the manual installation steps if not using the App Store variant.
</Accordion>
<Accordion title="Badge count shows no updates after visiting websites">
- The extension uses browser network APIs; ensure no other blockers or privacy settings prevent this access.
- Try visiting different websites to trigger network requests.
- Disable conflicting extensions for testing.
</Accordion>
</AccordionGroup>

## Additional Tips

- Always install uBO Scope from official browser stores to guarantee authenticity and automatic updates.
- The badge count reflects the number of distinct third-party servers connected; a zero or low count is expected on simple or private pages.
- This extension functions independently of your content blockers and reveals actual network activity, including stealth-blocked connections.
