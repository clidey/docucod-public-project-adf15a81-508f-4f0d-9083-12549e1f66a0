# Installing uBO Scope on Chrome, Firefox, & Safari

This guide walks you through installing the uBO Scope browser extension on Chrome, Firefox, and Safari, ensuring you can start monitoring third-party network connections with confidence and ease.

---

## 1. Overview

uBO Scope is a lightweight extension that transparently reports network connections to remote third-party servers your browser makes during web page loads. Installing it correctly on your browser of choice unlocks real-time insights into web traffic and content blocking outcomes.

---

## 2. Prerequisites & Preparation

Before installation, verify you meet these requirements:

- Supported browsers:
  - Google Chrome (version 122.0 or higher)
  - Mozilla Firefox (version 128.0 or higher)
  - Safari (version 18.5 or higher)
- Internet access to download the extension from the official browser stores
- Sufficient permissions to install extensions in your browser

<Tip>
Always use official extension stores or verified manual packages to avoid security risks.
</Tip>

---

## 3. Installation Instructions

### 3.1 Installing on Google Chrome

1. Open the [Chrome Web Store](https://chrome.google.com/webstore).
2. In the search bar, type "uBO Scope" or navigate directly to the official extension page.
3. Click **Add to Chrome**.
4. Confirm by clicking **Add extension** when prompted.
5. After installation, the uBO Scope icon (a stylized shield) will appear next to your address bar.

### 3.2 Installing on Mozilla Firefox

1. Open the [Firefox Add-ons website](https://addons.mozilla.org).
2. Search for "uBO Scope" or visit the official extension page.
3. Click **Add to Firefox**.
4. In the permissions prompt, click **Add**.
5. The extension icon will be visible near the toolbar after completion.

<Tip>
On Firefox, you can also manually install using the `.xpi` file generated by building the package with the `make-firefox.sh` script if you prefer offline installation.
</Tip>

### 3.3 Installing on Safari

Installation on Safari requires macOS Safari version 18.5 or newer.

1. Open the Mac App Store or [Apple Safari Extensions Gallery](https://apps.apple.com).
2. Search for "uBO Scope" extension.
3. Click **Get** and then **Install**.
4. After installing, open Safari Preferences > Extensions.
5. Enable uBO Scope by checking the box next to its name.

<Tip>
For manual or development installation, build the Safari extension package using `make-safari.sh` on a Linux environment and follow Safari’s developer extension installation workflow.
</Tip>

---

## 4. Verifying Installation

After completing installation:

- Check that the uBO Scope icon appears in your browser toolbar.
- Click the icon and ensure the popup loads without errors.
- Visit a website and observe that the badge count updates with third-party connections.

<Check>
If the icon or badge doesn't appear immediately, restart your browser or check extension status in your browser’s extension manager.
</Check>

---

## 5. Troubleshooting Common Issues

| Issue                             | Solution                                                                                   |
|----------------------------------|--------------------------------------------------------------------------------------------|
| Extension icon not visible        | Ensure the extension is enabled in your browser’s extensions settings.                      |
| Installation blocked or failed    | Verify browser version compatibility and internet connection; try reinstalling.            |
| Badge count not updating          | Refresh the current tab or browse to a new website; check for conflicting extensions.      |
| Permissions prompt blocked        | Approve all permission requests during installation for full functionality.                |

<Tip>
Refer to the [Troubleshooting Common Setup Issues](../first-use-and-validation/troubleshooting-and-support) guide for detailed solutions.
</Tip>
