# Installing uBO Scope

Follow these clear, step-by-step instructions to install uBO Scope on your preferred browser using the official extension stores. This guide ensures you set up uBO Scope correctly and verify its successful installation.

---

## 1. Prerequisites & System Requirements

Before proceeding, ensure your environment meets the following:

- **Supported Browsers**:
  - **Firefox** version 128.0 or later
  - **Safari** version 18.5 or later
  - **Chromium-based browsers** (Google Chrome, Microsoft Edge, Brave, etc.) — version requirements vary; check latest store info

- **Permissions & Features**:
  - Installation via official browser extension stores is recommended for security and automatic updates.
  - uBO Scope requires permissions to monitor network requests (e.g., `webRequest` API), active tab access, and storage.

- **Network Requirements**:
  - Access to the internet to download the extension from stores.
  - No special firewall or port configuration needed.

> <Check>Ensure your browser is up to date to avoid compatibility issues during installation.

---

## 2. Installation Instructions

Choose your browser below and follow the respective installation steps.

### 2.1 Installing on Firefox

1. Open the [Mozilla Add-ons site for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. When prompted, confirm by clicking **Add** to grant necessary permissions.
4. After installation, you will see the uBO Scope icon appear in the toolbar.

### 2.2 Installing on Safari

1. Open the [Mac App Store](https://apps.apple.com/app/ubo-scope/idxxxxxxxx) or visit the Safari Extensions Gallery.
2. Search for "uBO Scope".
3. Click **Get** and then **Install**.
4. Open Safari's **Preferences**, navigate to **Extensions**, and enable uBO Scope.

> <Note>Because Safari requires explicit enabling after installation, ensure the extension is activated in Safari preferences.

### 2.3 Installing on Chromium-based Browsers

1. Go to the [Chrome Web Store listing for uBO Scope](https://chrome.google.com/webstore/detail/ubo-scope/xxxxxxxx).
2. Click **Add to Chrome** (or **Add to Brave/Edge** as applicable).
3. Confirm permissions by clicking **Add extension**.
4. The uBO Scope icon will appear in your browser’s toolbar upon successful installation.

> <Tip>Pin the uBO Scope icon to the toolbar for quick access by clicking the puzzle piece icon and selecting the pin next to uBO Scope.

---

## 3. Verifying Installation Success

After installation, verify the extension is properly installed and active.

### 3.1 Visual Confirmation

- Look for the uBO Scope extension icon in your browser's toolbar.
- Click the icon to open the popup dashboard.
- Initially, the popup will display "NO DATA" until you visit websites.

### 3.2 Testing Basic Functionality

1. Open a popular website (e.g., https://example.com).
2. Click the uBO Scope toolbar icon to open the popup dashboard.
3. Observe the "domains connected" count changing from "?" to a numeric value.
4. You should see lists under "not blocked", "stealth-blocked", or "blocked" based on connections made.

> <Check>If you see counts update and domain lists populate, uBO Scope is functioning correctly.

---

## 4. Troubleshooting Common Installation Issues

### Issue: Extension Icon Does Not Appear
- Verify browser version meets minimum requirements.
- Check if uBO Scope is enabled in the browser’s extensions/settings panel.
- Restart the browser after installation.

### Issue: Popup Shows NO DATA Permanently
- Ensure you have visited some websites since installation.
- Confirm that no other extensions are blocking uBO Scope’s access.
- Permissions must be granted; check extension permissions in browser settings.

### Issue: Installation Fails or Is Blocked
- Confirm you are installing from the official browser extension store.
- Review any antivirus or firewall settings that might block extensions.
- Try reinstalling the extension.

> <Warning>If issues persist, consult the [Troubleshooting Installation & Setup](https://documentation-site/getting-started/first-use-and-validation/troubleshooting-installation) page for more detailed guidance.
