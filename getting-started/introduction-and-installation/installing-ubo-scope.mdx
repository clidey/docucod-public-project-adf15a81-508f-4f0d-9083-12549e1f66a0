# Installing uBO Scope

This guide provides clear, step-by-step instructions to install uBO Scope on your preferred browser. Whether you use Chromium-based browsers, Firefox, or Safari, follow the relevant section to get the extension up and running quickly and hassle-free.

---

## Supported Browsers

uBO Scope supports the following browsers with minimum version requirements:

- **Chromium-based browsers (Chrome, Edge, etc.)** - Version 122 or higher
- **Firefox** - Version 128 or higher
- **Safari** - Version 18.5 or higher

Ensure your browser meets these requirements before proceeding with installation.


## 1. Installing on Chromium Browsers

Chromium-based browsers include Google Chrome, Microsoft Edge, Brave, and others. Follow these straightforward steps:

1. **Open your Chromium browser.**
2. **Go to the Chrome Web Store page for uBO Scope:**
   
   [Install uBO Scope from Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click on **Add to Chrome** (or similar depending on browser).
4. Confirm the installation by allowing necessary permissions.
5. Once installed, the uBO Scope icon will appear in your toolbar.

### Verification

- Click the uBO Scope icon to open the popup and verify it loads without errors.
- The badge on the toolbar should appear empty initially and update as you browse.

<Warning>
Make sure you are using Chromium version 122 or newer; older versions are unsupported and may cause installation or runtime issues.
</Warning>

---

## 2. Installing on Firefox

Firefox users can install uBO Scope directly from the official Firefox Add-ons repository.

1. **Open Firefox browser.**
2. **Navigate to the Firefox Add-ons page for uBO Scope:**

   [Install uBO Scope from Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click **Add to Firefox**.
4. Review the permissions requested, then click **Add** to install.
5. The uBO Scope icon will appear near your address bar.

### Verification

- Click the uBO Scope icon to open the interface.
- Check that the popup loads the summary of remote server connections.

<Warning>
Ensure Firefox version is 128 or above for full compatibility.
</Warning>

---

## 3. Installing on Safari

Safari support requires Safari version 18.5 or later. Note that Safari extensions use a slightly different installation process:

1. Open Safari browser.
2. Navigate to the **Safari Extensions Gallery** or **Safari Extensions preferences pane** to search for uBO Scope.
3. If uBO Scope is not published in Safari’s gallery, you may need to install it manually:

    - Download the extension package from the [uBO Scope GitHub repository](https://github.com/gorhill/uBO-Scope).
    - Use Xcode or Safari’s developer mode to build and add the extension.

4. Follow Safari’s prompts to enable the extension.

### Verification

- Confirm the uBO Scope icon is present in Safari’s toolbar.
- Click the icon to verify the popup loads with connection summary.

<Note>
Manual installation on Safari requires technical familiarity with Safari’s extension development tools.
</Note>

---

## 4. Post-Installation Tips

- **Pin the Extension:** Pin uBO Scope to your toolbar for quick access.
- **Check Permissions:** Ensure the extension has permissions to monitor all sites.
- **Upgrade Browser if Needed:** If you encounter installation errors, verify the browser version and update if it does not meet minimum requirements.

## 5. Troubleshooting Common Issues

<AccordionGroup title="Troubleshooting Installation Issues">
<Accordion title="uBO Scope icon does not appear after installing">
- Try restarting your browser.
- Check that the extension is enabled in your browser’s extension manager.
- Confirm you installed the extension for the correct browser profile.
</Accordion>
<Accordion title="Extension fails to load or popup is blank">
- Verify your browser version meets the minimum version.
- Clear browser cache and reload.
- Try disabling other extensions temporarily to check for conflicts.
</Accordion>
<Accordion title="Installation button is disabled or not available">
- Check internet connectivity.
- Use the direct links provided to access the official stores.
- For Safari, manual installation may be necessary if the extension is not listed.
</Accordion>
</AccordionGroup>
