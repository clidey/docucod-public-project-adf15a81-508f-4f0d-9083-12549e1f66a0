# Installation Instructions for uBO Scope

Get up and running with uBO Scope quickly by following these targeted instructions tailored for your browser environment. This guide walks you through the installation on Chromium-based browsers (like Chrome and Edge), Firefox, and Safari, guiding you to official sources and showing you how to verify that the extension is correctly installed.

---

## 1. Prerequisites

Before installing uBO Scope, ensure that your system meets these basic requirements:

- **Supported Browsers:**
  - Chromium-based browsers version 122 or later
  - Firefox version 128 or later
  - Safari version 18.5 or later

- **Permissions:** uBO Scope requires permissions to access network requests and storage:
  - `webRequest`
  - `activeTab`
  - `storage`

- **Network Access:** Internet access is required to download the extension from official stores.

- **Browser Settings:** Ensure that extension installation is enabled in your browser settings.

For details, refer to the [System Requirements](./system-requirements) page.

---

## 2. Installation Steps

### A. Installing in Chromium-Based Browsers

1. Open your Chromium-based browser (Google Chrome, Microsoft Edge, Brave, etc.).
2. Navigate to the uBO Scope extension page on the Chrome Web Store:

   [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click the **Add to Chrome** (or equivalent) button.
4. Confirm the installation when prompted by clicking **Add Extension**.
5. Once installed, the uBO Scope icon will appear in your browser toolbar.

### B. Installing in Firefox

1. Launch Firefox version 128 or higher.
2. Go to the Firefox Add-ons site for uBO Scope:

   [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click the **Add to Firefox** button.
4. When prompted, click **Add** to confirm the permissions and complete the installation.
5. The extension icon will appear to the right of the address bar.

### C. Installing in Safari

1. Ensure you are running Safari 18.5 or later on macOS.
2. Open the Safari Extensions pane via Safari > Settings > Extensions.
3. Search for "uBO Scope" in the Safari Extensions Gallery or App Store or install it via the developer’s distribution if available.

   _Note: Safari versions may require user permission for unsigned or developer extensions._

4. Enable the uBO Scope extension in the Extensions settings.
5. The extension icon will appear in the Safari toolbar.

---

## 3. Verifying Your Installation

Confirm that uBO Scope is ready to use by following these checks:

- The uBO Scope icon appears in your browser’s toolbar.
- Click the icon to open the popup panel. The panel should load and display information (or say 'NO DATA' if no network activity is yet recorded).
- Reload a website to activate network monitoring and observe the badge count updating with distinct third-party connections.

<Tip>
If the popup does not open or shows an error, restart your browser and try again.

If the badge remains empty after visiting multiple websites, verify that uBO Scope has the necessary permissions and is enabled.
</Tip>

---

## 4. Post-Installation Security and Usage Tips

- Only install uBO Scope from official extension stores to avoid tampered versions.
- Keep your browser and extensions up to date for security and compatibility improvements.
- Understand that uBO Scope collects no personal data but monitors network requests for transparency.

## 5. Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting Installation">
<Accordion title="Extension Icon Not Visible After Installation">
- Verify that the extension is enabled in the browser’s extensions page.
- Check if your browser hides icons; use the toolbar’s customize option to display it.
</Accordion>
<Accordion title="Unable to Install on Unsupported Browser Version">
- Confirm browser version meets minimum requirements (Chromium 122+, Firefox 128+, Safari 18.5+).
- Upgrade your browser to the latest version.
</Accordion>
<Accordion title="Store Page Not Loading or Installation Blocked">
- Check your internet connection.
- If in a managed environment, ask your administrator if extensions are restricted.
- Temporarily disable conflicting security software.
</Accordion>
<Accordion title="Safari Installation Steps Differ or Extension Not Listed">
- Refer to Apple’s official documentation for Safari extension installation.
- Ensure you are on macOS and Safari versions that support WebExtensions.
</Accordion>
</AccordionGroup>
