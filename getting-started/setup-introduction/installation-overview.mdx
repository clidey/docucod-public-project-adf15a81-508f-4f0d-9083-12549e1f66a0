# Installing uBO Scope

Install uBO Scope quickly and effortlessly on your preferred browser. This guide provides explicit steps for installing the extension on major browsers including Chromium-based browsers (like Google Chrome), Mozilla Firefox, and Apple Safari.

---

## 1. Supported Browsers and Versions

Before proceeding, ensure your browser meets the minimum requirements:

- **Chromium-based Browsers** (Google Chrome, Edge, Brave, etc.) ‚Äî Version 122 or higher
- **Mozilla Firefox** ‚Äî Version 128 or higher
- **Apple Safari** ‚Äî Version 18.5 or higher

Using unsupported browsers or versions will prevent successful installation or operation of uBO Scope.

---

## 2. Installing uBO Scope

uBO Scope is available publicly via official browser extension marketplaces for major browsers. Follow the instructions below to install from these trusted sources.

### 2.1 Install on Chromium-Based Browsers

1. Open your Chromium-based browser.
2. Visit the uBO Scope page on the **Chrome Web Store**:

   üëâ [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click the **Add to Chrome** button.
4. In the confirmation dialog, select **Add extension**.
5. Wait for the extension to download and install. You will see the uBO Scope icon appear next to your address bar.

<Tip>
Make sure your browser is updated to Chrome 122+ to avoid any compatibility issues during installation.
</Tip>

### 2.2 Install on Mozilla Firefox

1. Launch Mozilla Firefox.
2. Go to the uBO Scope page on the **Firefox Add-ons** marketplace:

   üëâ [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click **Add to Firefox**.
4. When prompted, click **Add** to grant permissions.
5. Upon completion, the uBO Scope icon will appear in your toolbar.

<Note>
Firefox requires version 128 or later to support uBO Scope properly. Ensure your browser is up to date.
</Note>

### 2.3 Install on Apple Safari

1. Open Apple Safari on your Mac.
2. Navigate to the **App Store**.
3. Search for **uBO Scope** or visit the developer's page if available.
4. Download and install the extension.
5. After installation, enable uBO Scope from Safari's preferences under the ‚ÄúExtensions‚Äù tab.

<Warning>
Safari requires macOS that supports Safari 18.5 or higher. If your system is outdated, installation may not complete or extension may not function.
</Warning>

## 3. Verifying Installation

After installation:

- Look for the uBO Scope icon on your browser‚Äôs toolbar. It typically looks like a small shield or the uBO Scope logo.
- Click the icon to open the popup interface.
- The popup should display information or say "NO DATA" if not yet actively tracking.

If the icon is missing or the extension does not function as expected, please review the troubleshooting section or consult browser extension management settings.

---

## 4. Tips and Best Practices

- Allow the extension necessary permissions when prompted to ensure accurate network monitoring.
- Keep your browser updated to benefit from the latest security and performance improvements.
- Avoid installing multiple similar network monitoring extensions simultaneously to reduce conflicts.
