# Installation Guide for uBO Scope

Welcome to the uBO Scope Installation Guide. This document provides clear, step-by-step instructions to get uBO Scope up and running on your browser, whether you use Chrome, Firefox, or Safari. Each section covers verified installation sources, manual installation options, and helpful tips for extending the lifecycle of the extension, including updating and removing it.

---

## 1. Supported Browsers and Prerequisites

Before installing, confirm your browser meets the minimum version requirements to ensure full functionality:

- **Google Chrome:** Version 122 or later
- **Mozilla Firefox:** Version 128 or later
- **Apple Safari:** Version 18.5 or later

<u>Permissions Required:</u> On installation, uBO Scope requests permissions for `activeTab`, `storage`, and `webRequest`. These allow it to monitor network requests precisely and store session data for accurate badge counts.

<Tip>
Make sure your browser is updated before installation to avoid compatibility issues.
</Tip>

---

## 2. Installing uBO Scope

Follow the installation steps below tailored to your browser.

### 2.1 Installing from Official Extension Stores

Installing uBO Scope from the verified extension stores is the most secure and reliable method.

<Steps>
<Step title="Install on Google Chrome">

1. Open the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome**.
3. Confirm any prompts to grant the required permissions.
4. Once installed, the uBO Scope icon will appear in your toolbar.

<Check>
Verify installation by checking that the toolbar icon is visible.
</Check>

</Step>
<Step title="Install on Mozilla Firefox">

1. Navigate to the [Firefox Add-ons uBO Scope page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. Confirm the permissions dialog.
4. The uBO Scope icon should now appear in your Firefox toolbar.

<Check>
Ensure your Firefox version is 128 or higher for smooth operation.
</Check>

</Step>
<Step title="Install on Apple Safari">

1. Open the Safari Extensions Gallery or App Store.
2. Search for **uBO Scope** (compatible with Safari 18.5+).
3. Install the extension and enable it in Safari’s Extensions preferences.

<Note>
Safari extensions may require additional confirmation under macOS System Preferences > Security & Privacy > Extensions.
</Note>

</Step>
</Steps>

### 2.2 Manual Installation (Advanced Users)

For users who need to build or sideload uBO Scope manually, for example for development or testing:

1. Clone the repository from GitHub: `git clone https://github.com/gorhill/uBO-Scope.git`
2. Navigate to the `tools` directory.
3. Run the packaging script for your platform (Linux environment assumed):

```bash
# For Chromium-based browsers
./make-chromium.sh
```

4. Find the packaged extension in the `build` directory as a ZIP file.
5. Load the unpacked extension in your browser’s developer mode.

<Tip>
Manual installation methods are for advanced users who want control over versions or modifications. Store installation is preferred for most users.
</Tip>

## 3. Updating uBO Scope

uBO Scope updates, if available, are delivered through your browser’s extension update mechanism:

- Chrome and Firefox automatically check for updates regularly.
- Safari updates may require manual confirmation through the App Store or Safari Extensions interface.

If you installed uBO Scope manually, update by repeating the packaging and reloading the extension.

<Warning>
Avoid disabling updates unless necessary. Old versions may lack critical features or security fixes.
</Warning>

---

## 4. Removing uBO Scope

To uninstall uBO Scope, follow the steps below according to your browser:

### Chrome

1. Right-click the uBO Scope toolbar icon.
2. Select **Remove from Chrome**.
3. Confirm removal.

### Firefox

1. Open Firefox menu > **Add-ons and themes**.
2. Click **Extensions**.
3. Find uBO Scope and click the three-dot menu > **Remove**.

### Safari

1. Open Safari > Preferences > **Extensions**.
2. Uncheck or uninstall uBO Scope.

<Tip>
Removing the extension clears its badge and popup data immediately. No leftover data remains in your browser session storage.
</Tip>

---

## 5. Verification after Installation

Once installed, verify that uBO Scope operates correctly:

1. Visit any website.
2. Observe the uBO Scope badge counter on the toolbar icon — it displays the count of distinct third-party domains your page connected to.
3. Click the toolbar icon to open the popup and view categorized domains: allowed, stealth-blocked, and blocked.

<Check>
If the badge is empty or the popup shows 'NO DATA', ensure your browser permissions were granted and the page is fully loaded.
</Check>

## 6. Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting">
<Accordion title="uBO Scope icon doesn’t appear after installation">

- Make sure your browser version meets the minimum requirement.
- Check if the extension is enabled in your browser’s extension management page.
- Reload the browser or restart your computer.

</Accordion>
<Accordion title="The badge count never updates or is 0 always">

- Confirm you granted the required `webRequest` permission during installation.
- Try visiting different websites.
- Clear browser cache and reload.

</Accordion>
<Accordion title="Manual installation fails or errors on loading unpacked extension">

- Ensure you are running the packaging scripts on Linux as expected.
- Verify the directory structure matches the manifest requirements.
- Consult logs in browser developer console for errors.

</Accordion>
</AccordionGroup>
