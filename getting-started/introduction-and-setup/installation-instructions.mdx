# Installing uBO Scope

This guide provides straightforward, step-by-step instructions to download and install uBO Scope, ensuring you are ready to monitor your browser’s third-party network connections quickly and efficiently.

---

## 1. Preparing to Install

Before you install uBO Scope, ensure the following prerequisites are met:

- **Supported Browsers:**
  - Google Chrome (version 122.0 or newer)
  - Mozilla Firefox (version 128.0 or newer)
  - Safari (version 18.5 or newer)

- **Permissions:** The extension requires permission to access browser tabs, storage, and network web requests.

- **Network Access:** uBO Scope needs unrestricted access to internet URLs, including HTTP, HTTPS, WebSocket (wss/wss), for full visibility of network traffic.

> <Note>
> Installing uBO Scope on unsupported or outdated browser versions may cause installation failures or incomplete functionality.
> </Note>

---

## 2. Installation Steps by Browser

Follow the instructions below to install uBO Scope from official sources.

### Google Chrome

1. Open Google Chrome and navigate to the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click the **Add to Chrome** button.
3. Confirm the permissions dialog by clicking **Add extension**.
4. Wait a few moments for the extension to install.

### Mozilla Firefox

1. Open Firefox and visit the [Firefox Add-ons uBO Scope page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. Review the permissions requested and click **Add**.
4. The extension will install shortly.

### Safari

1. Open the Mac App Store or Apple’s Safari Extensions Gallery (if applicable).
2. Search for "uBO Scope" and select the official extension.
3. Click **Install**.
4. Enable the extension in Safari's Extensions preferences if not auto-enabled.

---

## 3. Post-Installation Verification

After installing uBO Scope, follow these steps to verify the installation succeeded:

1. Look for the uBO Scope icon in your browser toolbar. It uses a distinctive logo (colored squares).
2. Click the icon to open the popup. Initially, the popup may display **NO DATA** until you load websites.
3. Browse to any webpage, then click the uBO Scope icon again to see real-time data about connections.
4. Confirm the badge on the icon shows a number representing distinct third-party servers detected.

<Check>
If these elements are present and reactive, uBO Scope is installed correctly.
</Check>

---

## 4. Troubleshooting Common Installation Issues

If you encounter problems during or after installation, try these solutions:

- **Extension icon not appearing:**
  - Check the browser’s extensions management page to confirm uBO Scope is enabled.
  - Make sure your browser version meets the minimum requirement.

- **Installation button disabled or missing:**
  - Verify network connectivity.
  - Restart your browser and try again.
  - Ensure you are using the official store URLs.

- **Permissions prompts not appearing or blocked:**
  - Disable other conflicting extensions temporarily.
  - Allow extension permissions manually via the browser’s extension settings.

- **Popup displays NO DATA indefinitely:**
  - Reload an active tab after installation to trigger data collection.
  - Check browser security or privacy settings that might block extensions from capturing requests.

<Tip>
For persistent issues, consult your browser's extension troubleshooting guides or consider reinstalling uBO Scope.
</Tip>

## 5. What to Expect Next

With uBO Scope installed:

- You will begin to see live counts of distinct third-party server connections as you browse.
- The popup window allows in-depth review of connection categories: allowed, stealth-blocked, and blocked.
- The toolbar badge updates dynamically to reflect network activity per tab.
