# Installation Guide for uBO Scope

uBO Scope is a lightweight browser extension designed to display all network connections initiated by your browser tabs. This guide provides clear, actionable steps to install uBO Scope from official browser stores for Chrome, Firefox, and Safari, ensuring you quickly start monitoring your web connections.

---

## Supported Browsers and Minimum Versions

Before installation, confirm that your browser meets the minimum version requirements:

| Browser    | Minimum Version |
|------------|-----------------|
| Chrome     | 122.0 and above |
| Firefox    | 128.0 and above |
| Safari     | 18.5 and above  |

Ensure your browser is updated to meet these requirements to guarantee full compatibility.

---

## Step-by-Step Installation

### 1. Installation via Official Browser Stores

Choose the browser you use and follow the corresponding section to install uBO Scope.

### Chrome

1. Open Google Chrome.
2. Visit the [Chrome Web Store page for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click ‘Add to Chrome’.
4. Confirm the installation by clicking ‘Add extension’ when prompted.

### Firefox

1. Open Mozilla Firefox.
2. Visit the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click ‘Add to Firefox’.
4. Confirm the permission requests and click ‘Add’.

### Safari

1. Open Safari on your Mac.
2. Navigate to the Mac App Store or extensions page (uBO Scope appears in the Safari extensions gallery).
3. Search for “uBO Scope” and click ‘Install’.
4. Follow Safari prompts to enable the extension after installation.

---

### 2. Verifying Installation

Once installed, verify uBO Scope is active and ready:

- Look for the uBO Scope icon in your browser’s toolbar (usually near the address bar).
- The icon shows a badge with a count indicating the number of distinct third-party domains connected by your active tab.
- Click the icon to open the popup panel that displays detailed network connection insights.

<Tip>
If the icon or badge doesn't appear, try restarting your browser or checking your browser’s extensions/settings menu to confirm the extension is enabled.
</Tip>

### 3. Post-Installation Tips

- **Permissions**: uBO Scope requires permissions for webRequest, storage, and activeTab to monitor network traffic effectively.
- **Host Permissions**: The extension has broad host permissions (http://*/*, https://*/*, ws://*/*, wss://*/*) to observe all network requests.
- **Privacy**: uBO Scope only monitors connections from the browser and does not collect or transmit your browsing data externally.

## Troubleshooting Common Installation Issues

<AccordionGroup title="Common Installation Issues and Resolutions">
<Accordion title="uBO Scope icon not visible in toolbar">
- Ensure the extension is enabled in your browser’s extension manager.
- For Chrome and Firefox, pin the extension icon to the toolbar for quick access.
- Restart the browser and try again.
</Accordion>
<Accordion title="Installation blocked or not proceeding">
- Verify you are visiting the official browser stores.
- Check your internet connection.
- Disable any other security software temporarily that might block extension downloads.
</Accordion>
<Accordion title="Permissions prompts missing or denied">
- Manually grant required permissions by going into your browser’s extension settings.
- Reinstall the extension if permissions do not prompt on first install.
</Accordion>
</AccordionGroup>
