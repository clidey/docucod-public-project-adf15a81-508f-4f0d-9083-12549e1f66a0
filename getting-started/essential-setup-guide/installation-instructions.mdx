# Installation Instructions

This guide provides clear, step-by-step instructions to install uBO Scope on the three supported browsers: **Google Chrome (Chromium)**, **Mozilla Firefox**, and **Apple Safari**. Simply follow the process for your browser to add the extension and verify it is installed correctly. No technical expertise is required.

---

## Supported Browsers

- **Google Chrome / Chromium** version 122 or newer
- **Mozilla Firefox** version 128 or newer
- **Apple Safari** version 18.5 or newer

Ensure your browser version meets these minimum requirements for compatibility.

---

## Step 1: Choose Your Browser

Identify which browser you will use with uBO Scope from the options below. Each browser has a dedicated installation approach.

---

## Step 2: Installing uBO Scope

### Installation for Google Chrome / Chromium

1. Open your Chrome or Chromium browser.
2. Navigate to the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click **Add to Chrome**.
4. In the confirmation dialog, click **Add extension**.
5. Wait a few seconds until the extension installs.

### Installation for Mozilla Firefox

1. Open Firefox.
2. Go to the [Firefox Add-ons uBO Scope page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click the **Add to Firefox** button.
4. When prompted, choose **Add** to confirm.
5. The extension will install and appear in your toolbar.

### Installation for Apple Safari

1. Ensure your Safari version is 18.5 or later.
2. Visit the **Safari Extensions** section in the Mac App Store.
3. Search for "uBO Scope" and select it.
4. Click **Get** or the install button to download and install.
5. Open Safari Preferences (`Cmd + ,`), navigate to the **Extensions** tab, and enable uBO Scope.

---

## Step 3: Confirming Installation

After installation, verify that uBO Scope is running correctly:

1. Look for the uBO Scope icon in your browser toolbar. It resembles a small shield or badge with the uBO Scope logo.
2. Click the icon. A popup window should appear showing "NO DATA" initially.
3. Navigate to any website (for example, https://example.com).
4. Click the uBO Scope icon again. The popup should update with domain connection data from the active tab.

If you see domain counts and connection statuses, the extension is installed and functioning correctly.

---

## Step 4: Initial Use

- After installation, browsing any webpage will trigger uBO Scope to monitor third-party remote server connections.
- The badge on the toolbar icon will update to indicate the number of distinct remote servers contacted.
- Clicking the icon opens a summary popup with detailed results.

---

## Tips and Best Practices

- **Keep your browser updated** to stay compatible with uBO Scope’s latest features.
- If the badge count never updates, **try refreshing the webpage** or restarting the browser.
- uBO Scope works alongside content blockers but **accurately reports network connections regardless of blocking**.

## Troubleshooting Installation Issues

<AccordionGroup title="Common Installation Problems and Solutions">
<Accordion title="Extension Icon Missing After Installation">
- Check your browser’s extension or add-on management page.
- Make sure uBO Scope is enabled.
- Restart your browser.
- Ensure your browser is the correct version.
</Accordion>
<Accordion title="Popup Does Not Show Data After Navigating to Websites">
- Confirm the extension has required permissions.
- Refresh the webpage after installation.
- Restart your browser to clear any loading issues.
</Accordion>
<Accordion title="Installation Button Disabled or Not Responsive">
- Check your internet connection.
- Ensure you are signed in to the browser’s extension store if required.
- Try clearing the browser cache or using private/incognito mode.
</Accordion>
</AccordionGroup>
