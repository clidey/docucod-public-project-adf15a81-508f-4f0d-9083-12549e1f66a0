# Installation: Chrome, Firefox, Safari

This guide provides clear, step-by-step instructions to install the uBO Scope browser extension on Chrome, Firefox, and Safari. Whether you want to install directly from each browser's official extension store or sideload the extension for development and testing purposes, this page will walk you through the process efficiently.

---

## 1. Prerequisites Before Installation

Before installing uBO Scope, ensure the following:

- You are using a supported browser version:
  - **Chrome:** Version 122.0 or newer
  - **Firefox:** Version 128.0 or newer
  - **Safari:** Version 18.5 or newer
- You have an active internet connection to access the extension stores
- You have the necessary permissions to install browser extensions


## 2. Installing uBO Scope from Browser Extension Stores

The most straightforward way to install uBO Scope is via the official extension stores of your browser.

### 2.1 Installing on Chrome

1. Open Chrome browser.
2. Navigate to the uBO Scope [Chrome Web Store page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the **Add to Chrome** button.
4. In the confirmation dialog, click **Add extension**.
5. Wait for the installation to complete; the uBO Scope icon will appear in your toolbar.

### 2.2 Installing on Firefox

1. Open Firefox browser.
2. Go to the uBO Scope [Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click **Add to Firefox**.
4. Confirm by clicking **Add** in the prompt.
5. Once installed, the uBO Scope icon will appear next to the address bar.

### 2.3 Installing on Safari

1. Open Safari browser.
2. Access the Mac App Store or Safari Extensions Gallery to find uBO Scope.
3. Since uBO Scope requires Safari 18.5+, ensure your macOS and Safari are updated.
4. Install the extension following the App Store instructions.
5. After installation, enable the extension by going to **Safari > Settings > Extensions** and checking the box for uBO Scope.


## 3. Sideloading uBO Scope for Developers or Testers

If you are a developer or want to test the extension directly without using a store, you can sideload uBO Scope in your browser.

### 3.1 Sideloading on Chrome

1. Download or clone the uBO Scope source code from the [GitHub repository](https://github.com/gorhill/uBO-Scope).
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked**.
5. Select the folder containing the `manifest.json` file located in `platform/chromium/`.
6. The extension will load, and the icon appears in the toolbar.

### 3.2 Sideloading on Firefox

1. Download or clone the source code.
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
3. Click **Load Temporary Add-on...**.
4. Select the `manifest.json` file from the `platform/firefox/` directory.
5. Firefox will load the extension temporarily, and the icon will appear.

> **Note:** Temporary add-ons are removed when Firefox restarts.

### 3.3 Sideloading on Safari

Sideloading on Safari requires creating a Safari Web Extension project or using Xcode.

1. Open the `platform/safari/` folder in Xcode.
2. Build and run the extension from there.
3. Enable the extension in Safari through **Safari > Settings > Extensions**.

> **Tip:** For detailed developer guidelines on Safari extension sideloading, refer to Apple's official documentation on Safari Web Extensions.


## 4. Verifying Successful Installation

After installation, ensure uBO Scope is running correctly:

- Look for the uBO Scope icon in your browser toolbar.
- Click the icon to open the popup.
- Confirm the popup shows network connection data when browsing.
- See the badge count indicating distinct third-party connections on the icon.
