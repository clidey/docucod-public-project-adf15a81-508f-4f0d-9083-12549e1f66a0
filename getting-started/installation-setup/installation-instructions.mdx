# Installing uBO Scope

Follow these straightforward, browser-specific steps to add uBO Scope to Chrome, Firefox, or Safari via their official extension stores. This guide will enable you to get up and running quickly with direct links and clear instructions for a smooth installation experience.

---

## Supported Browsers

uBO Scope is designed for modern browsers with support for browser extension APIs:

- **Google Chrome** (version 122.0 and above)
- **Mozilla Firefox** (version 128.0 and above)
- **Safari** (version 18.5 and above)

Make sure your browser is updated to meet the minimum version requirements. Refer to [System Requirements](/getting-started/installation-setup/system-requirements) for detailed compatibility.

---

## Step-by-Step Installation

### 1. Install in Google Chrome

1. Open Google Chrome browser.
2. Visit the official uBO Scope listing on the [Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the **Add to Chrome** button.
4. In the confirmation dialog, click **Add extension**.
5. Wait for the extension icon to appear in the toolbar.

### 2. Install in Mozilla Firefox

1. Open Mozilla Firefox browser.
2. Go to the uBO Scope page on [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click **Add to Firefox**.
4. Confirm the installation by clicking **Add** in the pop-up.
5. After installation, you should see the uBO Scope icon in the toolbar.

### 3. Install in Safari

1. Open Safari browser.
2. Navigate to the [Mac App Store Extensions section](https://apps.apple.com/us/story/id1505056550) or directly search for 'uBO Scope' if listed.
3. Click **Get** or install through the official Safari extensions listing provided by Apple.
4. After installation, enable uBO Scope in Safari’s Extensions preferences:
   - Go to Safari > Settings (or Preferences) > Extensions.
   - Check the box next to **uBO Scope** to activate.

<Note>
Safari extensions require manual enabling after installation through the system preferences to function.
</Note>

---

## Verify Installation

Once installed, confirm uBO Scope is active by:

- Locating the uBO Scope icon on your browser toolbar.
- Clicking the icon should open the extension’s popup panel showing "domains connected" with initial data or a loading state.

<Tip>
If you do not see the icon:
- In Chrome and Firefox, check the extensions list (`chrome://extensions` or `about:addons`) and enable uBO Scope if disabled.
- In Safari, ensure the extension is enabled in Safari > Settings > Extensions.
</Tip>

## Best Practices for Installation

- Install uBO Scope only from official browser stores to ensure authenticity and security.
- Keep your browser updated to enjoy all features and security patches.
- Regularly review extension permissions for your peace of mind.

## Troubleshooting Common Installation Issues

<AccordionGroup title="Installation Troubleshooting">
<Accordion title="Extension Does Not Appear After Installation">
- Verify you are using a supported browser version.
- Manually enable the extension from the browser’s extension or add-on manager.
- Restart your browser to refresh extension loading.
- Clear browser cache if loading problems persist.
</Accordion>
<Accordion title="Installation Button is Disabled or Inactive">
- Confirm your internet connection is stable.
- Check if your browser enforces policies restricting extension installation (common in corporate environments).
- Temporarily disable other conflicting extensions and retry.
</Accordion>
<Accordion title="Safari Installation Steps Are Unclear or Unavailable">
- Ensure you are running Safari 18.5 or later.
- Use the Mac App Store or Apple Extensions Gallery; official uBO Scope Safari listing may vary.
- Consult Apple's support for managing Safari Extensions on macOS.
</Accordion>
</AccordionGroup>
