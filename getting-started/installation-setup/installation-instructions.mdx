# Extension Installation Guide

Welcome to the installation guide for uBlock Origin Scope (uBO Scope). This guide walks you through installing the extension on supported browsers via official stores, ensuring a secure and simple setup to begin monitoring network connections immediately.

---

## Supported Browsers and Versions

uBO Scope supports the following platforms with minimum browser versions:

- **Chromium-based browsers (Chrome, Edge, Brave, etc.)**: version 122.0 or later
- **Mozilla Firefox**: version 128.0 or later
- **Apple Safari**: version 18.5 or later

<Check>
Verify your browser version before installation to ensure compatibility.
</Check>

---

## Installation Overview

You have several straightforward options to install uBO Scope:

- Installing directly from the official browser extension or add-on stores.
- For advanced users or developers, building the extension from source is possible (see developer documentation).

This guide focuses on the official store installation for a seamless, safe experience.

---

## Step-by-Step Installation

### 1. Install on Chromium-Based Browsers

1. Open your Chromium-based browser.
2. Navigate to the [Chrome Web Store: uBO Scope](https://chrome.google.com/webstore/category/extensions) (search for "uBO Scope" by Raymond Hill).
3. Click **Add to Chrome**.
4. Confirm the permissions prompt:
   - uBO Scope requires permissions: `activeTab`, `storage`, and `webRequest` to monitor network activity.
5. Wait for the extension to download and install.
6. Upon installation, the uBO Scope icon appears in your browser toolbar.

<Tip>
If the extension icon does not appear, check your browser's extensions menu to ensure it is enabled.
</Tip>

### 2. Install on Mozilla Firefox

1. Open Firefox.
2. Go to the [Firefox Add-ons site](https://addons.mozilla.org/en-US/firefox/).
3. Search for "uBO Scope" by Raymond Hill.
4. Click **Add to Firefox**.
5. Review the required permissions (`activeTab`, `storage`, `webRequest`) and accept.
6. The extension icon will show in the toolbar once installation completes.

<Warning>
Ensure you install only from the official Mozilla Add-ons site to avoid counterfeit extensions.
</Warning>

### 3. Install on Apple Safari

1. Open Safari (version 18.5+).
2. Open the **Safari Extensions** section in the App Store.
3. Search for "uBO Scope" extension.
4. Click **Get** and then **Install**.
5. After installation, enable the extension in Safari’s preferences under the Extensions tab.

<Note>
Safari's extension system requires manual enabling after installation.
</Note>

## Verifying Installation

Once uBO Scope is installed, confirm the extension is properly loaded:

- Look for the uBO Scope icon (logo) in your browser toolbar.
- Click the icon to open the popup interface; it should load without errors.
- The popup shows real-time reports of remote server connections.

<Check>
If any errors occur in loading the popup, please consult the troubleshooting section or ensure your browser version meets the minimum requirements.
</Check>

---

## First Activation and Usage Tips

1. Click the uBO Scope icon to open the popup.
2. Visit a website; uBO Scope will automatically report connections it detects.
3. Familiarize yourself with the displayed data:
   - Allowed connections
   - Blocked connections
   - Stealth-blocked connections
4. The popup updates live as you navigate.

<Tip>
Granting the minimal requested permissions ensures uBO Scope gathers full connection data without compromising browser security.
</Tip>

## Additional Installation Methods (For Developers)

If you want to build the extension yourself or customize it:

- Clone the source from the [GitHub repository](https://github.com/gorhill/uBO-Scope).
- Use the provided scripts in the `/tools` directory:

```bash
# Build Chromium extension package
./tools/make-chromium.sh [version]

# Build Firefox extension package
./tools/make-firefox.sh [version]

# Build Safari extension package
./tools/make-safari.sh [version]
```

- Follow detailed build and packaging instructions in the repository's README.

<Note>
These scripts require a Linux environment and necessary tools like `zip` and `jq`.
</Note>
