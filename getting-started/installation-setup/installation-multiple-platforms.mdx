# Installation (Chrome, Firefox, Safari)

This guide walks you through the step-by-step process of downloading and installing **uBO Scope**, the browser extension designed to reveal remote server connections, from your browser’s official extension store. Installation is quick and straightforward on all major platforms: Chrome (and Chromium-based browsers), Firefox, and Safari.

---

## 1. Before You Begin

### Supported Browsers and Versions
- **Chrome (and Chromium-based browsers)**: Minimum version 122.0
- **Firefox**: Minimum version 128.0
- **Safari**: Minimum version 18.5

<Tip>
Make sure your browser is updated to at least these minimum versions to ensure compatibility and smooth operation.
</Tip>

### Permissions Required
uBO Scope requires the following permissions to function properly:
- Access to active tabs
- Storage access for settings
- `webRequest` API access to monitor network connections

These permissions will be requested during installation.

---

## 2. Installing uBO Scope in Your Browser

Follow the instructions below relevant to your browser.

### Chrome (and Chromium-based browsers)
1. Open the Chrome Web Store link for uBO Scope:

   [Install uBO Scope from the Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

2. Click the **Add to Chrome** button.
3. Review the permissions requested, then click **Add extension** to confirm.
4. The extension will download and install automatically.
5. Once installed, the uBO Scope icon will appear on your browser toolbar.

<Note>
If you do not see the extension icon, click the puzzle piece icon to open the Extensions menu and pin uBO Scope for easy access.
</Note>

### Firefox
1. Navigate to the official Firefox Add-ons page for uBO Scope:

   [Install uBO Scope from Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

2. Click the **Add to Firefox** button.
3. When prompted, review the permissions and click **Add**.
4. The extension will install and its icon will appear in the toolbar.

<Note>
If the icon is not visible immediately, open the Firefox menu and use the "Manage Extensions" option to pin uBO Scope.
</Note>

### Safari
1. Open Safari and visit the Mac App Store (Safari Extensions category).
2. Search for "uBO Scope" or use the direct link below if available.
3. Click **Get** or **Install** to add the extension.
4. After installation, go to Safari Preferences > Extensions and enable uBO Scope.

<Warning>
Safari extensions require explicit enabling in the Preferences pane after installation.
</Warning>

---

## 3. Verifying Your Installation

After installation, confirm that uBO Scope is active and ready:

1. Locate the uBO Scope icon in your browser toolbar.
2. Click the icon to open the popup interface.
3. You should see a popup describing the extension status and connection monitoring details.

<Tip>
If the extension icon shows a badge count (a number), this indicates the number of distinct third-party remote servers detected in current webpage connections — a core feature of uBO Scope.
</Tip>

---

## 4. Tips for a Smooth Setup

- **Browser Profiles:** If you use multiple browser profiles, install uBO Scope separately in each profile where you want to monitor network connections.
- **Restart Browser:** After installation, restart your browser to ensure all permissions and background services initialize correctly.
- **Permissions Review:** If any functionality seems limited, revisit your browser’s extension permissions to verify `webRequest` and related permissions are enabled.

<Check>
Make sure your browser is not blocking the extension or restricting its permissions via security or privacy settings.
</Check>

## 6. Troubleshooting Common Installation Issues

<AccordionGroup title="Common Installation Issues and Fixes">
<Accordion title="Extension Icon Missing After Install">
- Check if the extension is enabled in the browser’s extensions menu.
- Pin the icon from the extensions dropdown.
- Restart the browser.
</Accordion>
<Accordion title="Permissions Not Granted">
- Reinstall the extension to trigger permissions prompts.
- Manually adjust permissions in your browser’s extension settings.
</Accordion>
<Accordion title="Extension Not Monitoring Connections">
- Confirm your browser version meets minimum requirements.
- Disable conflicting content blockers or proxy extensions temporarily.
- Restart your browser.
</Accordion>
</AccordionGroup>

<Note>
If problems persist, consult the official troubleshooting guides linked in the documentation or seek support via the repository issues.
</Note>
