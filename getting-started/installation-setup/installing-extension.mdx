# Installing uBO Scope

This guide walks you through the detailed, browser-specific steps to install the uBO Scope extension. uBO Scope is a powerful tool designed to reveal all remote server connections initiated by webpages you visit, helping you gain insight into your network exposure and privacy.

Whether you use Chrome, Firefox, or Safari, this guide provides direct links, visual cues, and clear instructions so you can add uBO Scope quickly and securely.

---

## Supported Browsers and Requirements

Before starting, ensure your environment meets the following:

- **Google Chrome or Chromium-based browsers**: Version 122.0 or higher
- **Mozilla Firefox**: Version 128.0 or higher
- **Safari**: Version 18.5 or higher

The extension requires these browsers because it relies on specific APIs exposed only in these versions or above.

<Tip>
If you are using another browser or an older version than listed, uBO Scope installation is not supported, and functionality may be limited or unavailable.
</Tip>

---

## Installation Steps by Browser

### Installing on Google Chrome or Chromium-based Browsers

1. Open the [Chrome Web Store page for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click the **Add to Chrome** button.
3. Review the permissions requested (active tab access, storage, webRequest API) and click **Add extension**.
4. Wait for the confirmation message indicating the extension was added.
5. The uBO Scope icon will appear near the address bar.

<Note>
This installation requires Chrome version 122.0 or higher due to the use of Manifest Version 3 and service worker background scripts.
</Note>

### Installing on Mozilla Firefox

1. Navigate to the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. When prompted, verify the permissions and click **Add**.
4. The extension installs and its icon will be visible next to the address bar.

<Note>
Firefox version 128.0 or newer is required because uBO Scope uses modern WebExtensions APIs including Manifest Version 3 compatibility.
</Note>

### Installing on Safari

1. Open the [Safari Extensions Gallery](https://apps.apple.com/app/ubo-scope/id) or visit the Mac App Store if uBO Scope is listed there.
2. Download and install uBO Scope.
3. Enable the extension in Safari under **Preferences > Extensions**.
4. Confirm that the uBO Scope icon appears in your toolbar.

<Warning>
Safari requires version 18.5 or above for proper support. Older versions will not run uBO Scope.
</Warning>

## Verifying Successful Installation

After installation:

- Look for the uBO Scope icon in your browser toolbar.
- Click the icon to open the popup interface, which will initially display 'NO DATA' if no tab data is collected yet.
- Navigate to some websites to begin seeing remote server connection counts in real time.

<Check>
If the icon is missing or nothing happens when clicking it, try restarting your browser or checking your extension settings to ensure uBO Scope is enabled.
</Check>

## Best Practices During Installation

- **Use official web stores or trusted repository links** only for installation to guarantee safety and latest updates.
- **Review permissions carefully:** uBO Scope requires permissions to access tab data and intercept network requests to function properly.
- **Keep your browser updated** to ensure compatibility and security.
