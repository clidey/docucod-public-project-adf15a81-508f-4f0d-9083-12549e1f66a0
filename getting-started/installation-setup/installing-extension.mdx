# Installing uBO Scope

Welcome to the installation guide for **uBO Scope**, a lightweight browser extension that transparently reveals all network connections a webpage makes. This page provides clear, step-by-step instructions on how to install uBO Scope via official browser add-on stores for Chrome, Firefox, and Safari, along with guidance to verify your installation and ensure smooth operation.

---

## 1. Supported Browsers & Compatibility

uBO Scope supports the latest versions of major browsers with specific minimum versions and permissions requirements:

- **Google Chrome (Chromium-based browsers)**
  - Minimum version: 122.0
  - Installation via Chrome Web Store
  - Requires permissions: `activeTab`, `storage`, `webRequest`

- **Mozilla Firefox**
  - Minimum version: 128.0
  - Installation via Firefox Add-ons
  - Requires permissions: `activeTab`, `storage`, `webRequest`

- **Safari**
  - Minimum version: 18.5
  - Installation via Safari Extensions
  - Requires permissions: `activeTab`, `storage`, `webRequest`


<Tip>
Check your browser version before installation to avoid compatibility issues. On most browsers, you can find the version number under "About" in the menu.
</Tip>

---

## 2. Installing via Official Browser Store

Installing uBO Scope from your browser's official extension or add-on store guarantees you get the latest, secure, and verified version.

### Chrome Installation

1. Open Google Chrome.
2. Visit the official [uBO Scope listing on the Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click **Add to Chrome**.
4. When prompted, review the permissions and click **Add extension**.
5. Wait for the extension to download and install â€“ you will see the uBO Scope icon appear in the toolbar.

### Firefox Installation

1. Open Mozilla Firefox.
2. Go to the [uBO Scope page on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click **Add to Firefox**.
4. Review permissions and press **Add**.
5. The uBO Scope icon will appear in the toolbar once installed.

### Safari Installation

1. Open Safari on macOS.
2. Navigate to the Safari Extensions page in the App Store or visit [uBO Scope on the Safari Extensions gallery](https://apps.apple.com).
3. Click **Get** or **Install**.
4. Follow system prompts to allow the extension.
5. Enable uBO Scope under Safari Preferences > Extensions.


<Note>
Safari extensions have more manual enablement steps compared to Chrome and Firefox. Make sure uBO Scope is enabled in Safari's Extensions settings after installation.
</Note>

---

## 3. Verifying a Successful Installation

Confirm uBO Scope is correctly installed and operational by following these steps:

1. **Locate the uBO Scope icon** in your browser's toolbar. It should display an icon similar to the extension's logo.
2. **Click the icon** to open the popup panel.
3. Load any normal website and observe that the popup shows connection data such as "domains connected" and categorized lists labeled **not blocked**, **stealth-blocked**, and **blocked**.
4. Ensure the badge on the toolbar icon updates to indicate the count of distinct third-party remote servers connected.

<Tip>
If the badge does not appear or the popup shows 'NO DATA', try refreshing the active tab or restarting the browser.
</Tip>

---

## 4. Post-Installation Tips

- **Keep the extension updated** by allowing your browser to install automatic updates from the official store.
- **Check permissions** if uBO Scope does not report data. The extension requires network access to monitor web requests.
- **Review browser support** if you encounter unexpected behavior. Older browsers or unofficial builds might lack required APIs.

---

## 5. Troubleshooting Installation Issues

If you experience problems during installation or the extension is not functioning as expected, consider the following:

- **Browser version too old:** Update your browser to the latest version.
- **Permission denied:** Ensure you allowed all necessary permissions during installation.
- **Conflicts with other extensions:** Temporarily disable other content blockers or privacy extensions to test compatibility.
- **Extension icon missing:** Try restarting the browser or reinstalling uBO Scope.

For more detailed troubleshooting, see the [Troubleshooting Installation & Setup](https://example.com/getting-started/installation-setup/troubleshooting-installation) guide.
