# Installing uBO Scope

This guide walks you through the straightforward process of installing uBO Scope — a powerful browser extension that reveals every network connection your browser attempts in real time. You will learn how to install uBO Scope via official web stores on both Chromium-based browsers and Firefox, with direct links and practical tips to ensure a smooth setup.

---

## Supported Browsers

Before you begin, ensure your browser is supported:

- **Chromium-based browsers** (such as Google Chrome, Microsoft Edge, Brave) version 122 or higher
- **Mozilla Firefox** version 128 or higher

uBO Scope is designed to integrate seamlessly using each browser's native extension APIs.

---

## Step-by-Step Installation

### Installing uBO Scope on Chromium Browsers

1. **Open the Chrome Web Store**

   Click or enter the following URL in your Chromium-based browser’s address bar:

   [https://chrome.google.com/webstore/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp](https://chrome.google.com/webstore/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

2. **Click "Add to Chrome"**

   This button is located prominently near the top right of the page.

3. **Confirm Installation**

   A popup will appear requesting permissions required by uBO Scope. These include monitoring network requests and storing data locally. Click "Add extension" to proceed.

4. **Wait for Installation to Complete**

   The browser will confirm the installation with a success message and display the uBO Scope icon in your toolbar.

### Installing uBO Scope on Firefox

1. **Open Firefox Add-ons Site**

   Navigate to the following link in your Firefox browser:

   [https://addons.mozilla.org/firefox/addon/ubo-scope/](https://addons.mozilla.org/firefox/addon/ubo-scope/)

2. **Click "Add to Firefox"**

   This button will initiate the add-on download and installation process.

3. **Grant Permissions**

   Agree to the permissions prompted by Firefox so uBO Scope can monitor network requests and store necessary data.

4. **Confirm Installation**

   Firefox will notify you when installation finishes and display the uBO Scope icon near the address bar.

## Post-Installation Confirmation

Once installed, confirm uBO Scope is active:

- Look for the uBO Scope icon in your browser’s toolbar — a stylized purple shield with a magnifying glass.
- Click the icon to open its popup interface; it will display network connection summaries for your active tab.
- Initially, data will appear as you browse webpages, showing allowed, blocked, and stealth-blocked third-party domains.

<Tip>
If you don’t see the icon, check your browser’s extensions or add-ons menu. You may need to enable the extension manually or pin it to the toolbar for easy access.
</Tip>

---

## Best Practices and Tips for a Smooth Installation

- **Use Official Stores Only:** Always install uBO Scope from the official Chrome Web Store or Firefox Add-ons to ensure security and get automatic updates.
- **Permissions Awareness:** uBO Scope requires permissions to access webRequest data and storage. These are essential for its core function of monitoring network connections.
- **Browser Version:** Verify your browser version meets minimum requirements to avoid incompatibility issues.
- **Restart Browser if Needed:** If the extension does not appear right after installation, try restarting your browser.

## Troubleshooting Common Installation Issues

<AccordionGroup title="Installation Troubleshooting Tips">
<Accordion title="Extension Not Visible in Toolbar">
- Open your browser’s extensions or add-ons management page.
- Locate uBO Scope and choose the option to pin or enable it.
- If necessary, restart the browser.
</Accordion>
<Accordion title="Installation Failed or Blocked">
- Verify you are using a supported browser version.
- Ensure your network allows access to the extension store domains.
- Check if security software or corporate policies block extension installations.
</Accordion>
<Accordion title="Permissions Not Granted or Revoked">
- Reinstall the extension to re-trigger permissions requests.
- Manually adjust permissions in your browser’s extension settings.
</Accordion>
</AccordionGroup>
