# Installing uBO Scope on Chrome, Firefox, and Safari

Welcome to the installation guide for uBO Scope, a powerful browser extension that reports all remote server connections your browser attempts or completes. This guide will walk you through installing uBO Scope from the official extension stores for Chrome, Firefox, and Safari with clear, browser-specific steps.

---

## 1. Prerequisites

Before you begin installation, ensure your environment meets the following:

- **Supported Browsers & Minimum Versions:**
  - **Chrome:** Version 122.0 or higher
  - **Firefox:** Version 128.0 or higher
  - **Safari:** Version 18.5 or higher

- **Network:** Access to the relevant browser extension store.
- **Permissions:** uBO Scope requires permission to access webRequests, active tab, and storage, which will be requested during installation.


## 2. Installation Instructions by Browser

Each browser has its own official store and installation method. Follow the steps below specific to your preferred browser.

### 2.1 Installing on Chrome

Chrome users will install uBO Scope from the Chrome Web Store.

<Steps>
<Step title="Open the Chrome Web Store">
Navigate to the Chrome Web Store: [uBO Scope on Chrome Web Store](https://chrome.google.com/webstore/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
</Step>
<Step title="Locate uBO Scope">
Use the search bar or follow the direct link above to find 'uBO Scope' by Raymond Hill.
</Step>
<Step title="Add to Chrome">
Click the 'Add to Chrome' button to begin installation.
</Step>
<Step title="Confirm Permissions">
A permissions dialog appears detailing required access for:
- Access to active tabs
- Network web requests
- Storage
Click 'Add extension' to confirm.
</Step>
<Step title="Installation Completion">
Wait a moment as Chrome installs the extension. You’ll see the uBO Scope icon appear in your toolbar.
</Step>
</Steps>

<Tip>
If the extension icon does not appear immediately after installation, click the puzzle-piece icon in the toolbar and pin uBO Scope for easier access.
</Tip>

### 2.2 Installing on Firefox

For Firefox users, install uBO Scope from the official Firefox Add-ons site.

<Steps>
<Step title="Open Firefox Add-ons">
Go to [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
</Step>
<Step title="Find the Extension">
Search for 'uBO Scope' by Raymond Hill or use the direct link above.
</Step>
<Step title="Add to Firefox">
Click the 'Add to Firefox' button.
</Step>
<Step title="Review Permissions">
The extension requests permission to monitor web requests, use active tab data, and store settings. Click 'Add' to approve.
</Step>
<Step title="Finish Installation">
Wait for Firefox to add the extension. You’ll see the uBO Scope icon on the toolbar once done.
</Step>
</Steps>

<Note>
If you use Firefox on Android, ensure your Firefox version is 128.0 or higher for compatibility.
</Note>

### 2.3 Installing on Safari

Safari extension installation is available for Safari 18.5 or newer via the Mac App Store.

<Steps>
<Step title="Open the Mac App Store">
Launch the Mac App Store application.
</Step>
<Step title="Search for uBO Scope">
Search for 'uBO Scope' extension or visit the official Safari Extensions Gallery if available.
</Step>
<Step title="Download the Extension">
Click 'Get' or the download button to install the extension.
</Step>
<Step title="Enable the Extension in Safari">
In Safari, go to **Safari > Settings > Extensions**, find 'uBO Scope' and check the box to enable it.
</Step>
<Step title="Grant Permissions">
Safari may prompt for permissions necessary for connection monitoring. Approve these to allow the extension to function properly.
</Step>
</Steps>

<Warning>
If Safari version is below 18.5, uBO Scope will not work due to minimum version requirements.
</Warning>

## 3. Verifying Installation

After installing the extension, verify it is properly installed:

- Look for the uBO Scope icon in your browser’s toolbar.
- Click the icon. You should see the popup interface titled 'uBO-Scope' showing connection data (initially may show 'NO DATA' until you browse).

<Tip>
If the popup does not open or shows error, try restarting your browser or checking extension permissions.
</Tip>

## 4. First Use: Quick Start

Once installed, try these quick steps:

1. Open a new tab and browse to your favorite website.
2. Click the uBO Scope toolbar icon.
3. Observe the summary showing 'domains connected' - this reports the number of distinct third-party remote servers contacted.
4. Review the categorized lists below, indicating domains "not blocked," "stealth-blocked," or "blocked."

This immediate feedback gives you real insight into network connections made by your browser.

## 5. Troubleshooting Common Issues

<AccordionGroup title="Installation Troubleshooting">
<Accordion title="Extension icon not appearing after installation">
- Refresh your browser or restart it.
- Verify that the extension is enabled in your browser’s extension/add-ons page.
- Check if the extension is hidden behind the overflow menu (puzzle icon).
</Accordion>
<Accordion title="Permissions not granted or extension not functioning">
- Reinstall the extension to trigger permission requests again.
- Manually enable needed permissions via your browser settings.
- Ensure your browser version meets minimum requirements.
</Accordion>
<Accordion title="Safari specific issues">
- Ensure you have enabled uBO Scope in Safari's Extensions preferences.
- Update Safari to version 18.5 or newer.
- Restart Safari after extension installation.
</Accordion>
</AccordionGroup>
