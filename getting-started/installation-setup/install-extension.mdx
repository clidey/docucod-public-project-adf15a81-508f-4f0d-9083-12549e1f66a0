# Installing uBO Scope

Follow these clear, straightforward steps to install uBO Scope on your preferred browser—Chrome, Firefox, or Safari. This guide uses official extension marketplaces to ensure security and easy updates, with visual cues and direct links for a smooth onboarding experience.

---

## 1. Supported Browsers & Prerequisites

Before installing, confirm your browser version meets uBO Scope’s minimum requirements:

- **Google Chrome**: Version 122 or higher
- **Mozilla Firefox**: Version 128 or higher
- **Apple Safari**: Version 18.5 or higher

<u>Permissions Highlight:</u> uBO Scope requires these permissions:

- Access to webRequest API to monitor network connections
- Ability to read active tabs
- Storage for extension data

If these are disabled or restricted, the extension may not work correctly.

<Check>
Verify your browser version by navigating to your browser’s About page (e.g., `chrome://settings/help` on Chrome, `about:support` on Firefox, or Safari > About Safari).
</Check>

## 2. Installing uBO Scope

<u>Your goal:</u> Add uBO Scope safely from the official browser extension store for reliable updates and security.

### Step-by-Step Installation

<Steps>
<Step title="Install on Google Chrome">

1. Open [uBO Scope Chrome Web Store page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click the **Add to Chrome** button.
3. Confirm the permissions prompt by clicking **Add extension**.
4. Wait for the installation confirmation; the uBO Scope icon appears next to the address bar.

</Step>
<Step title="Install on Mozilla Firefox">

1. Open the [uBO Scope Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click the **Add to Firefox** button.
3. Accept requested permissions and click **Add**.
4. Look for the uBO Scope icon in the toolbar to confirm installation.

</Step>
<Step title="Install on Safari">

1. Open the Mac App Store and search for uBO Scope Safari extension (if published there), or check the official uBO Scope Safari extension download page if provided.
2. Follow Apple’s official Safari extensions install prompts.
3. After installing, enable the extension via Safari Preferences > Extensions.

> **Note:** Safari’s minimum version required is 18.5 for uBO Scope compatibility.

</Step>
</Steps>

<Tip>
For best security and updates, always install uBO Scope from official browser extension stores and keep your browser up to date.
</Tip>

## 3. Verifying Installation

After installation:

- Look for the uBO Scope icon (an eye symbol) in your browser’s toolbar.
- Click the icon to open the popup interface, which will initially show "NO DATA" until you browse a website.

### Quick Verification

1. Open a website in a new tab.
2. Click the uBO Scope toolbar icon.
3. You should see a count of connected third-party domains, categorized under "not blocked," "stealth-blocked," and "blocked." 

This immediate visual confirms that uBO Scope is actively monitoring your network connections.

<Check>
If the icon does not appear or the popup stays empty, retry installation or check the Troubleshooting guide.
</Check>

## 4. Post-Installation Tips

- **Pin the Extension Icon:** Pin uBO Scope to the toolbar for easy access.
- **Permissions Review:** If you run other browser security extensions or strict privacy settings, ensure they don’t block uBO Scope’s required permissions.

## 5. Common Installation Troubleshooting

<AccordionGroup title="Installation Troubleshooting">
<Accordion title="uBO Scope icon not visible after installation">
- Ensure the extension is enabled in your browser’s extensions menu.
- Restart the browser to load newly installed extensions.
- Check if your browser version meets minimum requirements.
</Accordion>
<Accordion title="Installation fails with permission error">
- Verify you are installing from the official extension store.
- Check if your organization or antivirus software is blocking extension installations.
- Try temporarily disabling strict privacy or content-blocking extensions during installation.
</Accordion>
<Accordion title="Popup shows NO DATA after browsing">
- Make sure you have visited at least one website after installing.
- Some browsers restrict the webRequest API on private/incognito modes by default; ensure the extension has permission if browsing privately.
</Accordion>
</AccordionGroup>
