# Installing uBO Scope

Welcome to the installation guide for **uBO Scope**, a powerful browser extension designed to provide you with detailed visibility into all remote server connections made by your web pages. This guide walks you through installing uBO Scope on the three major supported browsers: **Chrome**, **Firefox**, and **Safari**. Follow the step-by-step instructions below to complete your installation quickly and start uncovering network activity.

---

## 1. Supported Browsers

Before proceeding with installation, ensure your browser matches the supported versions:

- **Google Chrome** (Chromium-based): Version 122.0 or newer
- **Mozilla Firefox**: Version 128 or newer
- **Safari**: Version 18.5 or newer

If you haven't verified this yet, please see the [Supported Browsers & Prerequisites](https://docs.uboscope.org/getting-started/installation-setup/supported-browsers-prerequisites) page for details on requirements and permissions.

---

## 2. Installing uBO Scope

Each supported browser provides an official extension marketplace listing for secure and seamless installation.

### Chrome (Chromium-based browsers)

1. Open the [Chrome Web Store page for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome**.
3. Confirm by clicking **Add Extension** in the popup.
4. Wait for the installation to complete — the extension's icon will appear in the toolbar.

### Firefox

1. Visit the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. Confirm by clicking **Add** when prompted.
4. The extension icon will show in the Firefox toolbar when installation finishes.

### Safari

1. Open Safari and navigate to the [App Store page for uBO Scope](https://apps.apple.com/app/ubo-scope/idYOUR_SAFARI_APP_ID) *(replace the placeholder with the actual link if available)*.
2. Click **Get** or **Install** to download.
3. After installation, enable the extension in Safari > Preferences > Extensions.
4. Confirm the uBO Scope icon is visible on your Safari toolbar.

---

## 3. Verifying Your Installation

Once installed, confirm uBO Scope is active and ready to use:

- Look for the **uBO Scope icon** in your browser's toolbar. It appears as a small badge icon representing network connection counts.
- Click the icon to open the extension popup panel. On the initial load, you should see the heading with your current tab's hostname and sections for domains labeled "not blocked," "stealth-blocked," and "blocked." If you see "NO DATA," try refreshing the page.

## 4. Tips for a Successful Installation

- **Grant necessary permissions** when prompted during installation to ensure uBO Scope can access network request data.
- If the extension icon does not appear, try restarting your browser.
- For Safari users, remember to explicitly enable the extension after installation.
- Extension updates happen rarely but check the marketplace periodically for new versions.

---

## 5. Common Installation Issues & Troubleshooting

<AccordionGroup title="Installation Troubleshooting">  
<Accordion title="Extension Icon Not Visible">  
- Ensure your browser version meets the minimum requirements.
- Confirm the extension is enabled in your browser’s extension/add-ons manager.
- Restart the browser after installation.
- Check for extension conflicts by temporarily disabling other extensions.
</Accordion>  
<Accordion title="Cannot Install due to Permissions or Restrictions">  
- Make sure you are installing from the official browser store.
- Some enterprise-managed devices restrict extension installations; check with your administrator.
- Confirm that pop-up blockers or security software aren’t preventing installation.
</Accordion>  
<Accordion title="Popup Displays 'NO DATA' After Installation">  
- Refresh the active tab to allow uBO Scope to collect data.
- Visit a website known to connect to third-party servers for visible results.
- Confirm network requests are not blocked by browser settings that might prevent reporting.
</Accordion>  
</AccordionGroup>
