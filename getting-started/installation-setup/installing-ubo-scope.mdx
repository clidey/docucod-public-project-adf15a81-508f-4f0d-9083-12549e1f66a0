# Installing uBO Scope

This guide walks you through the process of acquiring and installing uBO Scope on your preferred browser. It provides trusted direct links to official extension stores for Chrome, Firefox, and Safari, and offers precise step-by-step instructions to ensure a smooth and secure installation experience.

---

## Supported Browsers

uBO Scope supports the following browsers with the minimum required versions:

| Browser         | Minimum Version |
| --------------- | --------------- |
| Google Chrome   | 122+            |
| Mozilla Firefox | 128+            |
| Safari          | 18.5+           |


## Installation Overview

uBO Scope is distributed through official browser extension stores to guarantee authenticity and security. Installing from these trusted sources ensures you receive genuine updates and protections.

---

## Step-by-Step Installation Instructions

### 1. Google Chrome

To install uBO Scope on Chromium-based browsers (Chrome, Edge, Brave):

1. Open your browser and navigate to the Chrome Web Store page for uBO Scope:

   [Install uBO Scope for Chrome](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

2. Click the **Add to Chrome** button.
3. When prompted, review and accept the extension permissions.
4. Once installed, you will see the uBO Scope icon appear in your toolbar.

<Callout>
**Tip:** Enable the extension's toolbar icon for quick access by clicking the puzzle piece icon, then pin uBO Scope for convenience.
</Callout>

---

### 2. Mozilla Firefox

Follow these steps to install uBO Scope on Firefox:

1. Open Firefox and navigate to the official Mozilla Add-ons page:

   [Install uBO Scope for Firefox](https://addons.mozilla.org/firefox/addon/ubo-scope/)

2. Click on the **Add to Firefox** button.
3. Review the permissions and click **Add** to confirm.
4. Upon installation, the uBO Scope icon will appear in the Firefox toolbar.

<Warning>
Ensure you install from the official Mozilla Add-ons page to avoid counterfeit extensions.
</Warning>

---

### 3. Safari

To install uBO Scope on Safari, follow the platform-specific process:

1. Click or open the Safari Extensions Gallery or Mac App Store link to find uBO Scope.
2. Alternatively, install via Safari's built-in extension management by:
  - Opening Safari Preferences
  - Navigating to the Extensions tab
  - Searching for "uBO Scope"
  - Following prompts to download and enable the extension

Note: Ensure your Safari version is 18.5 or later, as required by uBO Scope.

<Note>
The extension’s icon and popup are fully functional within Safari once enabled.
</Note>

---

## Post-Installation Steps

After installation, complete these actions:

1. Reload any active browser tabs to allow uBO Scope to start monitoring network requests.
2. Click the uBO Scope icon in your toolbar to open the popup panel and view active connections.
3. Confirm that the badge counter updates when browsing websites.

<Check>
If the badge remains empty or shows no data, verify that the extension has the necessary permissions (active tab, storage, webRequest).
</Check>

---

## Security and Permissions

- uBO Scope requires permissions to observe network requests (`webRequest`) and access tabs to provide accurate domain connection information.
- It does not block or interfere with requests; instead, it reports their outcomes.
- Only install from official extension stores to ensure a safe environment.

---

## Verifying Installation

To confirm successful installation:

- Visit any popular website.
- Observe the badge count on the uBO Scope toolbar icon; this indicates the number of distinct third-party domains connected.
- Click the icon to review detailed connection information in the popup.

Successful data display confirms the extension is active and monitoring.

## Troubleshooting Installation Issues

If you encounter difficulties:

- Refresh your browser and retry installation.
- Ensure no other extension conflicts by temporarily disabling other privacy or network monitoring tools.
- Verify your browser version meets minimum requirements.
- Check your browser’s extension permissions and settings.

For detailed troubleshooting, see [Troubleshooting Common Installation Issues](/getting-started/first-use-validation/troubleshooting-installation).
