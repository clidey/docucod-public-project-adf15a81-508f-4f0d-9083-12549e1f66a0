# Installing the Extension

Installing uBO Scope is straightforward and tailored to your browser. This guide walks you through how to obtain and install the extension securely on Chromium-based browsers, Firefox, and offers notes for Safari users.

---

## 1. Supported Browsers and Requirements

Before installation, ensure your browser meets the minimum version requirements:

- **Chromium (Google Chrome, Edge, Brave, etc.)**: Version 122.0 or newer
- **Firefox**: Version 128.0 or newer
- **Safari**: Version 18.5 or newer

The extension requires permissions to monitor network requests (`webRequest` API) and storage access within the browser environment.

<Tip>
If you are unsure of your browser version, consult your browser’s "About" page or update it through official channels to meet the requirements.
</Tip>

---

## 2. Installation Steps by Browser

### Chromium-Based Browsers

1. Open Chrome, Edge, Brave, or any Chromium-based browser.
2. Access the Chrome Web Store page for uBO Scope:

   [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click **Add to Chrome** (or equivalent button). Confirm by clicking **Add extension** in the prompt.
4. Once installed, uBO Scope will appear as an icon near the address bar with a default badge.

<Warning>
Installing only from the official Chrome Web Store URL above ensures the extension's integrity and safety.
</Warning>

### Firefox

1. Launch Firefox.
2. Navigate to the Mozilla Add-ons page for uBO Scope:

   [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click **Add to Firefox** and allow the permissions requested.
4. Confirm installation by clicking **Add**.
5. The extension icon will be visible in your toolbar.

<Warning>
Always install Firefox extensions from the official Mozilla Add-ons site to avoid impersonators.
</Warning>

### Safari

Safari support for uBO Scope requires at least version 18.5. Installation may not be available through a public app store yet.

- To use uBO Scope on Safari, you may need to build and install the extension locally or use developer installation workflows.
- Refer to the `platform/safari/manifest.json` for platform-specific compatibility and requirements.

<Note>
Due to Safari’s distribution policies, the extension might not be in the official App Store. Consult the GitHub repository for latest updates or instructions on manual installation.
</Note>

---

## 3. Verification After Installation

After installing, confirm uBO Scope is active:

- The extension icon labeled **uBO Scope** or **uBO** should appear near the browser’s address bar.
- Visit any website and observe the icon badge updating to reflect third-party remote server connections.
- Click the icon to open the popup panel summarizing allowed, stealth-blocked, and blocked connections.

For detailed steps on verifying your installation, see the [Verifying Your Installation](https://docs.uboscope.example/getting-started/installation-setup/verifying-installation) page.

---

## 4. Best Practices and Security Tips

- Always install extensions from linked official store pages.
- Avoid installing from unknown third-party sites or unsolicited links.
- Keep your browser updated to maintain compatibility and security.
- Regularly check the extension’s permissions and updates.

<Tip>
Consider pinning the uBO Scope icon to your toolbar for quick visibility of connection activity.
</Tip>
