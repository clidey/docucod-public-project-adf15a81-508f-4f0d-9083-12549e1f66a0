# Installation Guide

Welcome to the uBO Scope installation guide. This document walks you through **simple, step-by-step instructions** to install uBO Scope on the most popular browsers: **Chrome**, **Firefox**, and **Safari**. Installation is performed via the official browser extension stores to ensure security and ease of updates.

---

## 1. Supported Browsers and Versions

Before starting, ensure your browser meets the minimum version requirements:

- **Google Chrome / Chromium**: Version **122.0** or newer
- **Mozilla Firefox**: Version **128.0** or newer
- **Apple Safari**: Version **18.5** or newer

uBO Scope requires permissions such as `activeTab`, `storage`, and `webRequest` to operate effectively. These are automatically requested during installation.


---

## 2. Installing uBO Scope

Install uBO Scope easily from your browser's official extension/add-on stores. Follow the steps below for each browser.

### 2.1 Installing on Google Chrome / Chromium

1. Open the Chrome Web Store page for uBO Scope:

   [uBO Scope on Chrome Web Store](https://chrome.google.com/webstore/detail/ubo-scope/)

2. Click the **Add to Chrome** button.
3. In the confirmation dialog, click **Add Extension**.
4. Wait for the extension to download and install automatically.

<Note>
Ensure you are logged into Chrome with a Google account to access the Chrome Web Store.
</Note>

### 2.2 Installing on Mozilla Firefox

1. Open the Firefox Add-ons page for uBO Scope:

   [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

2. Click **Add to Firefox**.
3. In the prompt that appears, click **Add**.
4. Allow any requested permissions.
5. The extension will install automatically.

<Warning>
If your Firefox version is older than 128.0, upgrade Firefox before installing.
</Warning>

### 2.3 Installing on Apple Safari

1. Open the Mac App Store and search for 'uBO Scope' or visit the link:

   [uBO Scope on Mac App Store](https://apps.apple.com/app/ubo-scope/)

2. Click **Get** and then **Install**.
3. After installation, open Safari and go to **Preferences** (Safari menu > Preferences).
4. Select the **Extensions** tab, then find **uBO Scope** and enable the checkbox to activate the extension.

<Note>
Safari requires you to grant permission explicitly for extensions after installation.
</Note>

---

## 3. Post-Installation Steps

After installing uBO Scope, confirm everything is set up correctly and configure permissions if required.

### 3.1 Confirm uBO Scope is Installed

- Look for the uBO Scope icon in your browser toolbar. By default, it is a small icon labeled "uBO Scope" with the recognized logo.

### 3.2 Granting Necessary Permissions

- Some browsers (especially Safari) require you to grant additional permissions the first time you use the extension.
- When prompted, allow the extension to access the active tab and network request information.

<Tip>
If the extension icon is hidden, check your browser’s extension management UI to pin or enable it for easier access.
</Tip>

---

## 4. Verifying Installation

Ensure that uBO Scope is operating correctly by following these quick verification steps.

1. Navigate to any website that loads external resources.
2. Click the uBO Scope toolbar icon to open the popup panel.
3. Observe the connection summary:
   - You should see domains categorized into _allowed_, _blocked_, or _stealth-blocked_.
   - If "NO DATA" is shown, reload the page and try again.

<Warning>
If no data appears even after reload, verify the extension has the required permissions and is enabled.
</Warning>

---

## 5. Troubleshooting Installation Issues

Encounter a problem? Try the following solutions:

- **Extension not appearing in toolbar:**
  - Check your browser’s extensions settings to confirm uBO Scope is enabled.
  - Pin the extension icon to your toolbar if it is hidden.

- **Permissions not granted:**
  - For Safari, visit Preferences > Extensions and enable all permissions for uBO Scope.
  - For Firefox and Chrome, reinstall the extension to re-trigger permissions requests.

- **Browser version too low:**
  - Update your browser to meet the minimum version stated above.

- **Extension not capturing data:**
  - Disable other extensions that might conflict, such as privacy or security blockers.
  - Restart your browser and try again.

<Info>
For more troubleshooting tips, visit the [Troubleshooting & Validation](../getting-started/first-steps/troubleshooting) guide.
</Info>
