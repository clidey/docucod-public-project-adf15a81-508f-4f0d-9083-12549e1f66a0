## 1. Prerequisites & System Requirements

Before installing uBO Scope, ensure your environment meets the following:

- **Supported Browsers:**
  - **Chrome:** Version 122 or later
  - **Firefox:** Version 128 or later
  - **Safari:** Version 18.5 or later
- **Operating Systems:** Compatible with OS versions supported by the above browsers
- **Permissions:** uBO Scope requires permission to monitor network requests via the browser's `webRequest` API and access to the active tab.
- **Internet Connection:** Needed to download the extension

For detailed system requirements, see [System Requirements & Supported Browsers](https://example.com/getting-started/installation-basics/system-requirements).

## 2. Installing uBO Scope

Choose your preferred installation method based on your browser:

### A. Install from Official Browser Stores

1. **Chrome:**
    - Visit the Chrome Web Store page for uBO Scope: [Chrome Web Store - uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)
    - Click **Add to Chrome**
    - Confirm by clicking **Add extension**

2. **Firefox:**
    - Open Firefox Add-ons: [Firefox Add-ons - uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/)
    - Click **Add to Firefox**
    - Confirm by clicking **Add**

3. **Safari:**
    - Open the Mac App Store or Safari Extensions Gallery
    - Search for 'uBO Scope' or follow the links from the GitHub repository's Safari section
    - Install the extension

<Tip>
Using the official stores ensures you receive automatic updates and security reviews.
</Tip>

### B. Manual Installation for Advanced Users

You may manually install uBO Scope directly from the [uBO Scope GitHub repository](https://github.com/gorhill/uBO-Scope) if you prefer:

1. Clone or download the repository.
2. Build the extension for your platform (e.g., via provided build scripts).
3. Load the unpacked extension in your browser's developer mode/extensions page:

   - **Chrome:** Navigate to `chrome://extensions`, enable **Developer mode**, then click **Load unpacked** and select the appropriate build folder.
   - **Firefox:** Navigate to `about:debugging#/runtime/this-firefox`, click **Load Temporary Add-on**, and select any file inside the build folder.
   - **Safari:** Use Safari's Extension Builder to load the unpacked extension.

<Warning>
Manual installation requires technical knowledge and does not provide automatic updates.
</Warning>

## 3. Verifying Installation Success

After installation, verify uBO Scope is correctly installed and ready to use:

1. Look for the uBO Scope icon in your browser toolbar. It uses the distinct multi-sized icon set with a magnifying glass or radar-style design.

2. Click the icon to open the extension popup.

3. The popup should initially display "NO DATA" until you activate network activity in your browser.

4. Navigate to any website, then click the icon again to see the domains connected by your active tab, categorized by allowed, stealth-blocked, and blocked.

<Check>
If you do not see the extension icon:
- On Chrome and Firefox, ensure the extension is enabled in the extensions page.
- On Safari, ensure the extension is enabled in Safari Preferences > Extensions.
- In case of missing icon, try reinstalling or relaunching the browser.
</Check>

## 5. Troubleshooting Common Installation Issues

<AccordionGroup title="Installation Troubleshooting">
<Accordion title="The Extension Icon Does Not Appear">
- Ensure the extension is enabled in your browser's extensions page.
- Restart your browser.
- Check browser compatibility/version.
</Accordion>
<Accordion title="Installation from Store Fails or Is Blocked">
- Verify network connectivity.
- Temporarily disable other extensions that might block store access.
- Try manual installation.
</Accordion>
<Accordion title="uBO Scope Does Not Show Any Data After Installation">
- Ensure you have granted necessary permissions.
- Make sure you are visiting websites (network activity required).
- Reload the web page and click the icon again.
</Accordion>
</AccordionGroup>
