# Installing uBO Scope

This guide walks you through the step-by-step process of installing the uBO Scope browser extension on Chrome, Firefox, and Safari. You'll find direct links to each browser’s web store and clear instructions on what to expect during installation, helping you get started quickly and confidently.

---

## 1. Overview of Installation

uBO Scope is designed to transparently report remote server connections made by web pages. To leverage this powerful insight, you need to install the extension on your preferred browser. This section focuses exclusively on installation, ensuring you take the right steps to get the extension running smoothly.


## 2. Supported Browsers and Installation Links

The extension supports the latest stable versions of major browsers with the following minimum version requirements:

| Browser  | Minimum Version |
|----------|-----------------|
| Chrome   | 122             |
| Firefox  | 128             |
| Safari   | 18.5            |

Follow the links below to access the official and trusted extension downloads:

- **Chrome:** [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)
- **Firefox:** [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)
- **Safari:** Installation requires installing the Safari Web Extension from the App Store or manually enabling the extension after download (instructions below).


## 3. Installing uBO Scope on Chrome

### Step 1: Open the Chrome Web Store
- Visit the [uBO Scope extension page on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).

### Step 2: Add the Extension
- Click the **Add to Chrome** button.
- Confirm the permissions when prompted to allow uBO Scope access to monitor web requests.

### Step 3: Verify Installation
- After installation, you will see the uBO Scope icon in the Chrome toolbar.
- The toolbar badge initially shows zero or no count until you start browsing websites.

### Tips & Troubleshooting
- Ensure your Chrome version is 122 or above to avoid compatibility issues.
- If installation fails, retry after restarting Chrome or checking your network connection.


## 4. Installing uBO Scope on Firefox

### Step 1: Open Firefox Add-ons Page
- Navigate to uBO Scope’s page on Firefox Add-ons: [uBO Scope for Firefox](https://addons.mozilla.org/firefox/addon/ubo-scope/).

### Step 2: Install the Extension
- Click **Add to Firefox**.
- Review the required permissions, then confirm the installation.

### Step 3: Confirm Successful Installation
- uBO Scope's icon will appear in the Firefox toolbar.
- The badge count updates dynamically as you browse.

### Tips & Troubleshooting
- Firefox must be version 128 or newer.
- If you do not see the icon, check the toolbar customization to ensure the extension is visible.


## 5. Installing uBO Scope on Safari

Safari extensions require a slightly different approach due to App Store distribution and system security controls.

### Step 1: Obtain the Extension
- Download or locate the uBO Scope Safari Web Extension package. (Usually available via the App Store or developer site.)

### Step 2: Enable the Extension
1. Open **Safari**.
2. Go to the **Safari menu > Settings** (or **Preferences**).
3. Navigate to the **Extensions** tab.
4. Find **uBO Scope** in the list.
5. Check the box to enable the extension.

### Step 3: Confirm Permissions
- Safari will request permission for uBO Scope to monitor web traffic; approve these to allow full functionality.

### Step 4: Verify Installation
- The extension icon appears in the Safari toolbar.

### Tips & Troubleshooting
- Ensure your Safari version is 18.5 or later.
- If the extension does not appear, confirm it is installed in your Applications folder and Safari’s Extensions settings.
- Restart Safari if needed to complete installation.


## 6. What to Expect During Installation

Across all platforms, the installation process will request permissions to:

- Access your active tab and browsing data.
- Monitor network requests to detect server connections.
- Store data locally within browser storage.

Upon first install and launch, expect the extension to begin monitoring connections immediately. The icon on your browser’s toolbar will display a badge with a count representing distinct third-party remote servers connected by websites.


## 7. Post-Installation Verification

Once installed:

1. Open your browser and navigate to any website.
2. Click the uBO Scope icon.
3. You should see a popup detailing domains connected, categorized into:
   - **Not blocked:** Connections allowed.
   - **Stealth-blocked:** Connections silently blocked.
   - **Blocked:** Connections explicitly blocked.

These categories help you understand real-time remote server activities.
