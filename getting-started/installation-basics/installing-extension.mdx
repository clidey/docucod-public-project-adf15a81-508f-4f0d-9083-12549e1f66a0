# Installing uBO Scope

Welcome to the installation guide for **uBO Scope**, the simple and transparent browser extension that reveals all attempted and successful remote server connections on any web page. This guide walks you through installing uBO Scope on Chrome, Firefox, and Safari, guiding you to download it safely from official extension stores. By the end, you'll be ready to start monitoring your web connections instantly.

---

## 1. Prerequisites & System Requirements

Before installation, ensure your system meets the following requirements for a smooth experience:

- **Supported Browsers and Versions:**
  - **Google Chrome / Chromium-based browsers:** Version 122 or newer
  - **Mozilla Firefox:** Version 128 or newer
  - **Safari:** Version 18.5 or newer
- **Permissions Needed:**
  - Access to install browser extensions
  - Permission for the extension to monitor network requests (`webRequest` API)
- **Compatibility Notes:**
  - This extension works independently of content blockers; network requests can only be monitored when exposed via the browser’s `webRequest` API.

> For detailed compatibility and system specs, see the [System Requirements & Browser Compatibility](./system-requirements) page.

---

## 2. Installation Instructions by Browser

### Chrome & Chromium Browsers

Follow these steps to install uBO Scope on Chrome or any Chromium-based browser:

1. Open your browser and navigate to the [Chrome Web Store page for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click the **Add to Chrome** button.
3. Confirm the permissions prompt by clicking **Add extension**.
4. Wait a moment while the extension downloads and installs; you will see the uBO Scope icon appear in your browser toolbar.

> <img src="https://raw.githubusercontent.com/gorhill/uBO-Scope/main/img/browsericons/logo128.png" alt="uBO Scope icon" width="48" />  
> **Tip:** Pin the uBO Scope icon to your toolbar for quick access.

---

### Mozilla Firefox

Install uBO Scope on Firefox by following these steps:

1. Open Firefox and visit the [uBO Scope add-ons page on Mozilla Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. Review the permissions, then click **Add**.
4. After installation completes, the uBO Scope icon becomes visible in your toolbar.

---

### Safari

To install uBO Scope on Safari (version 18.5+):

1. Open Safari on your Mac.
2. Go to the [Mac App Store Safari Extensions section](https://apps.apple.com/app/ubo-scope/idYourExtensionIdHere) *(Note: Replace with actual official Safari extension URL if available)*.
3. Search for **uBO Scope**.
4. Click **Get** or **Install** to add the extension.
5. Open Safari Preferences > Extensions.
6. Enable **uBO Scope** by checking the box next to its name.

> <Warning>
> Safari’s extension system requires stricter versions and permissions; ensure your macOS and Safari versions meet or exceed the minimum requirements.
> </Warning>

## 3. Verifying Installation

After installation, you want to be sure uBO Scope is working correctly.

### How to Verify:

1. Look for the **uBO Scope icon** in your browser toolbar. It should display a badge with a count (or remain empty if no third-party connections yet).
2. Visit a webpage such as `https://example.com`.
3. Click the uBO Scope toolbar icon.
4. The popup will display the domains contacted by the web page in three categories:
   - **Not blocked** (allowed connections)
   - **Stealth-blocked**
   - **Blocked**
5. Confirm that you see domain names populate in the lists and that the count updates.

> <Check>
> If the popup shows "NO DATA" or empty results after browsing, refresh the page and try again.
> </Check>

## 4. Troubleshooting Common Installation Issues

<AccordionGroup title="Common Installation Issues and Resolutions">
<Accordion title="Extension Icon Not Visible After Installation">
Ensure you have pinned the extension icon in the toolbar. In Chrome and Firefox, click the puzzle piece icon (extensions menu) and pin uBO Scope. For Safari, verify the extension is enabled in Safari Preferences > Extensions.
</Accordion>
<Accordion title="Extension Not Recording Connections">
Check that you have granted the `webRequest` permission during installation. Also, confirm your browser is updated to a supported version (Chrome 122+, Firefox 128+, Safari 18.5+).
</Accordion>
<Accordion title="Unable to Find uBO Scope in Store">
Double-check you are accessing the official extension store URL to avoid spoofed or unofficial downloads. Refer to this guide’s links directly.
</Accordion>
<Accordion title="Permissions Prompt Fails or Errors Occur">
Temporarily disable other conflicting extensions or try reinstalling uBO Scope. Restart your browser if issues persist.
</Accordion>
</AccordionGroup>
