# Installing uBO Scope

Follow these clear, step-by-step instructions to add the uBO Scope extension to your preferred browser—Chrome, Firefox, or Safari. This page guides you to the official stores, details any browser-specific quirks, and helps you complete installation confidently and successfully.

---

## 1. Choose Your Browser

uBO Scope supports the following browsers, each requiring a slightly different installation approach:

- **Google Chrome** (version 122 or later)
- **Mozilla Firefox** (version 128 or later)
- **Apple Safari** (version 18.5 or later)

Make sure your browser meets the minimum version requirements to ensure full compatibility.

---

## 2. Access Official Extension Store

To guarantee security and seamless updates, always install uBO Scope from its official store listing.

### Chrome Installation

1. Open Chrome’s Web Store:
   [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)
2. Click **Add to Chrome**.
3. Confirm by clicking **Add extension** in the pop-up.

### Firefox Installation

1. Open Firefox Add-ons page:
   [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)
2. Click **Add to Firefox**.
3. When prompted, click **Add**.
4. Confirm permissions by clicking **Okay, Got It**.

### Safari Installation

1. Open the [Safari Extensions Gallery](https://apps.apple.com/app/ubo-scope/id) or the Mac App Store (search for "uBO Scope").
2. Follow the prompts to install.
3. Enable the extension in Safari’s preferences under **Extensions**.

> Note: Safari requires macOS 13.4 or later and has different permission handling; ensure to grant necessary access when prompted.

---

## 3. Post-Installation Checks

After installation, verify the extension is enabled and ready to use.

- **Chrome/Firefox:** Look for the uBO Scope icon next to the address bar.
- **Safari:** Confirm the extension is listed and checked in Safari > Preferences > Extensions.

Click the icon to open the popup interface and confirm it loads without errors.

---

## 4. Browser-Specific Tips & Known Quirks

### Permissions Required

uBO Scope requests these permissions across all browsers to function properly:

- **webRequest:** to monitor network requests
- **storage:** to save session data
- **activeTab:** for interaction with the active tab

### Chrome

- Minimum version 122 is required.
- The extension uses Manifest V3 with service worker background scripts.

### Firefox

- Minimum version 128 is mandatory.
- Uses standard background scripts.

### Safari

- Minimum version 18.5 or higher.
- Hosts permissions are limited to http and https (no websocket permissions).

> Warning: Safari’s extension system may prompt you to explicitly enable network monitoring for this extension.

## 5. Troubleshooting Installation Issues

If you encounter problems during installation, consider the following:

- Confirm your browser version meets the minimum requirement.
- Check network connectivity to the official extension store.
- Restart your browser after installation if the uBO Scope icon does not appear.
- Verify no security or antivirus software is blocking the extension installation.

For details on setup errors or configuration issues after installation, see the [Troubleshooting Common Setup Issues](https://docs.example.com/getting-started/configuring-troubleshooting/troubleshooting-setup) page.
