# Installing uBO Scope

This guide provides clear, step-by-step instructions to install the uBO Scope browser extension on your preferred browser. uBO Scope offers real-time visibility into remote server connections made by your browser, empowering you to monitor and enhance your browsing privacy effortlessly.

---

## 1. Supported Browsers and System Requirements

Before installing uBO Scope, ensure that your system and browser meet the following prerequisites:

- **Browsers Supported:**
  - Google Chrome (version 122.0 or later)
  - Mozilla Firefox (version 128.0 or later)
  - Safari (version 18.5 or later)

- **Operating Systems:** Compatible with all OS versions supported by the above browsers.

- **Permissions Required:** The extension requires permission to access browser tabs, storage, and web requests to monitor network connections effectively.

- **Network:** No special network configurations are necessary, but ensure your browser can access the Web Store or Add-ons site.

<Tip>
Always verify your browser version before proceeding to ensure compatibility and a smooth installation experience.
</Tip>

## 2. Installing on Your Browser

Follow the platform-specific installation instructions below to add uBO Scope to your browser.

### Google Chrome

1. Open Google Chrome.
2. Navigate to the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the "Add to Chrome" button.
4. When prompted, review and accept the requested permissions.
5. Wait for the extension to download and install automatically.

<Check>
On installation completion, the uBO Scope icon will appear in your Chrome toolbar.
</Check>

### Mozilla Firefox

1. Open Mozilla Firefox.
2. Head to the [Firefox Add-ons uBO Scope page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click the "Add to Firefox" button.
4. Grant the permissions requested by the extension.
5. The extension will install and add its icon to Firefox’s toolbar.

<Check>
Confirm the uBO Scope icon is visible on the Firefox toolbar as a sign of successful installation.
</Check>

### Safari

1. Open Safari on macOS.
2. Visit the App Store or Safari Extensions menu.
3. Search for "uBO Scope".
4. Install the extension ensuring your Safari version is at least 18.5.

<Note>
Safari requires a minimum version of 18.5 to run uBO Scope correctly.
</Note>

<Check>
Once installed, the uBO Scope icon will appear near the address bar.
</Check>

## 3. Verifying Installation

After installation, verify uBO Scope is properly set up:

1. Locate the uBO Scope icon in your browser toolbar.
2. Click the icon to open the popup interface.
3. If you see a summary panel with connection data (even if initially minimal), the extension is active.

<Tip>
If you do not see the icon, check your browser’s extensions or add-ons menu to confirm uBO Scope is enabled.
</Tip>

## 4. Post-Installation Tips

- **Pin the Extension:** For quick access, pin the uBO Scope icon on your toolbar.
- **Reload Browser Tabs:** For the extension to start monitoring, refresh any open tabs or open a new tab.
- **Permissions Check:** If the extension does not show data, verify that all requested permissions were granted during installation.

## 5. Troubleshooting Installation Issues

<AccordionGroup title="Common Installation Problems and Solutions">
<Accordion title="uBO Scope Icon Missing After Installation">
- Check if the extension is enabled in your browser’s extension management page.
- Restart your browser to apply changes.
- Verify browser version meets minimum requirements.
</Accordion>
<Accordion title="Permissions Not Accepted or Revoked">
- Go to your browser’s extension settings.
- Review and re-enable all required permissions: activeTab, storage, and webRequest.
- Reload the browser tab.
</Accordion>
<Accordion title="Cannot Find uBO Scope in Extension Store">
- Ensure your browser is updated to the latest version.
- Confirm you are visiting the official Chrome Web Store or Firefox Add-ons site through provided links.
</Accordion>
</AccordionGroup>
