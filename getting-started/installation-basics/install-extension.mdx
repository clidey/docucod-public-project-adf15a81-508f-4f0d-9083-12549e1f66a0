# Installing uBO Scope

This guide provides a straightforward, step-by-step process to install the uBO Scope browser extension on Chrome and Firefox. Leveraging automated store links, you will quickly add the extension and verify its successful installation using clear visual indicators. Follow these instructions to start gaining immediate insight into third-party network connections your browser initiates.

---

## 1. Supported Browsers Before Installation

Before installing uBO Scope, ensure your browser meets the minimum version requirements:

- **Google Chrome:** Version 122 or later
- **Mozilla Firefox:** Version 128 or later

For full details on compatible browsers and permissions needed, consult the [Supported Browsers & Requirements](https://your-docs-domain/getting-started/installation-basics/supported-platforms-requirements) guide.

> **Tip:** Using the latest stable browser release guarantees the smoothest extension experience.

---

## 2. Installing uBO Scope on Chrome

Follow these steps to install uBO Scope via the Chrome Web Store:

<Steps>
<Step title="Open the Chrome Web Store page for uBO Scope">
1. Click the following link or paste it into your Chrome address bar:

   [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

2. The page should display the uBO Scope extension with its icon and description.
</Step>
<Step title="Add the Extension">
1. Click the “Add to Chrome” button.
2. When prompted, review the permissions uBO Scope requests.
3. Confirm by selecting “Add Extension.”
</Step>
<Step title="Wait for the Installation to Complete">
1. The extension icon (a distinct uBO Scope logo) will appear in the toolbar next to the address bar.
2. You may see a brief notification confirming installation.
</Step>
<Step title="Verify Installation">
1. Look for the uBO Scope icon in the browser toolbar.
2. Click the icon to open the popup panel.
3. The popup should display “NO DATA” initially until you navigate to web pages.
</Step>
</Steps>

> **Note:** If you do not see the extension icon, click the puzzle-piece icon (Chrome Extensions menu) and pin uBO Scope for easier access.

---

## 3. Installing uBO Scope on Firefox

To install uBO Scope via Firefox Add-ons store, follow these steps:

<Steps>
<Step title="Open the Firefox Add-ons page for uBO Scope">
1. Click or paste the following URL into Firefox:

   [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

2. This will open the extension’s page with details and screenshots.
</Step>
<Step title="Add the Extension to Firefox">
1. Click the “Add to Firefox” button.
2. Review the permissions requested by uBO Scope.
3. Choose “Add” to confirm installation.
</Step>
<Step title="Confirm Installation">
1. After installation, the uBO Scope icon will appear on the Firefox toolbar.
2. Click the icon to open the popup and check that it says “NO DATA” until browsing web pages.
</Step>
<Step title="Pin the Extension (Optional but Recommended)">
1. Click the hamburger menu → Extensions and Themes → Manage Extensions.
2. Locate uBO Scope and select “Pin” to keep it readily accessible.
</Step>
</Steps>

> **Tip:** Firefox keeps extensions enabled across sessions — you only need to install once.

---

## 4. Post-Installation Verification

Once the extension is installed, confirm it’s functioning correctly with these checks:

- **Toolbar Icon Badge:** The badge overlays a number indicating how many distinct third-party domains your current tab connects to.

- **Popup Panel Display:** Clicking the icon opens a panel showing real-time domain connection stats categorized as 'not blocked', 'stealth-blocked', and 'blocked'.

- **Navigating to Websites:** Visit typical websites and watch the badge count update accordingly.

---

## 5. Platform-Specific Notes

- **Chrome:** Minimum version 122 required for compatibility with extension background service worker.
- **Firefox:** Minimum version 128 ensures full use of required APIs.
- **Safari:** The extension supports Safari 18.5+, but installation is manual and outside scope here.

---

## 6. Troubleshooting Common Installation Issues

<AccordionGroup title="Installation Troubleshooting">
<Accordion title="Extension Icon Not Visible">
- Check if the extension is disabled or hidden in your browser’s extension manager.
- On Chrome, pin the extension using the Extensions icon menu.
- Restart the browser if needed.
</Accordion>
<Accordion title="Installation Failed or Permissions Not Accepted">
- Ensure your browser version is at or above minimum requirements.
- Confirm you are installing from the official store links.
- If issues persist, clear your browser cache and try again.
</Accordion>
<Accordion title="Popup Shows No Data After Installation">
- This is expected until you navigate to websites generating network connections.
- Try reloading a standard web page like https://example.com.
- The badge count should update reflecting third-party connections.
</Accordion>
</AccordionGroup>
