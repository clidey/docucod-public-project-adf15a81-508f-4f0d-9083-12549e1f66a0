# Installing uBO Scope

Get started with uBO Scope quickly and effortlessly by installing it on your preferred browser. This guide walks you through the precise steps to install uBO Scope on **Google Chrome**, **Mozilla Firefox**, and **Safari**, ensuring you can immediately monitor remote server connections made by active webpages.

---

## Supported Browsers

Before installation, verify that your environment meets these minimum requirements:

- **Google Chrome** (version 122.0 or later)
- **Mozilla Firefox** (version 128.0 or later)
- **Safari** (version 18.5 or later)

Ensure your browser is updated to at least these versions to guarantee full functionality of uBO Scope.

---

## Installation Options

### 1. Google Chrome

#### Step-by-Step Installation from Chrome Web Store

1. Open Google Chrome and navigate to the [uBO Scope Chrome Web Store page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click the **Add to Chrome** button.
3. In the prompt, confirm by clicking **Add extension**.
4. Once installed, the uBO Scope icon (colored badge) will appear in your browser toolbar.

#### Verification
- Confirm the extension icon is visible near the address bar.
- You should see a small numeric badge reflecting the count of distinct third-party connections.

#### Troubleshooting
- If the **Add to Chrome** button is disabled, verify your Chrome version (must be 122 or newer).
- If an error appears during installation, ensure you have a stable internet connection and sufficient permissions.

---

### 2. Mozilla Firefox

#### Step-by-Step Installation from Mozilla Add-ons

1. Open Firefox and go to the [uBO Scope Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. When prompted, click **Add** to allow the extension.
4. The uBO Scope extension icon will then appear in your Firefox toolbar.

#### Verification
- Check for the uBO Scope icon with a badge on the toolbar.
- The badge will update dynamically as you browse.

#### Troubleshooting
- If unable to install, verify Firefox is version 128 or above.
- Restart Firefox if the extension icon does not appear immediately.

---

### 3. Safari

#### Step-by-Step Installation via Safari Extensions Gallery or Developer-Side Loading

**Safari requires macOS 13 Ventura or later and Safari v18.5 or higher.**

1. Open Safari.
2. Go to the Safari Extensions Gallery or Apple’s Safari Extensions in the App Store.
3. Search for 'uBO Scope' or use developer instructions if installing manually.

Alternatively, advanced users can package and install the extension manually using the provided package scripts.

#### Manual Packaging for Safari (for developers)

```bash
./tools/make-package.sh safari
```

Then follow Apple’s guidelines to load the unpacked extension in Safari’s Developer menu.

#### Verification
- Confirm the uBO Scope icon appears in Safari’s toolbar.

#### Troubleshooting
- If the extension does not load, ensure macOS and Safari meet minimum version requirements.
- Check Safari’s Preferences under Extensions to enable uBO Scope.

---

## Post-Installation: What to Expect

- Once installed, your browser toolbar will display the uBO Scope icon with a numeric badge.
- This badge shows the number of distinct third-party remote servers your current page is connecting to.
- Click the icon to open the popup.html interface, which categorizes these domains into:

  - **Not blocked** — connections allowed
  - **Stealth-blocked** — connections redirected or stealth-blocked
  - **Blocked** — connections that failed or were outright blocked

- The popup also displays counts for each category and detailed domain information.

---

## Practical Tips

- **Pin the extension icon** to your toolbar for easy access across browsing sessions.
- During first use, visit websites with varied content (e.g., news, shopping) to observe the dynamic badge count changes.
- Use the popup panel regularly to audit third-party connections and assess network transparency.
