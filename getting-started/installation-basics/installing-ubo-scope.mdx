# Installing uBO Scope

This guide provides clear, actionable instructions to install uBO Scope on your preferred browser—Chromium-based browsers, Firefox, and Safari. Follow the steps below to add the extension from your browser’s official extension store, ensuring you get the authentic and most compatible version.

---

## 1. Supported Browsers

Before starting, ensure you are using one of the supported browsers and meet the minimum required versions:

- **Chromium Browsers (e.g., Google Chrome, Microsoft Edge)**: Version 122.0 or later
- **Mozilla Firefox**: Version 128.0 or later
- **Safari**: Version 18.5 or later

> For all supported browsers, uBO Scope requires permissions to monitor network requests. These permissions are requested during installation to enable accurate reporting.


## 2. Installing from the Official Browser Extension Stores

Below are step-by-step instructions for each supported browser. Choose the section matching your environment.

### 2.1 Chromium Browsers

1. Open your Chromium-based browser.
2. Visit the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the **Add to Chrome** button.
4. In the confirmation dialog, review the requested permissions:
   - Access to your active tab
   - Storage access
   - webRequest permissions to monitor network requests
5. Click **Add extension** to confirm.
6. Wait for the browser to download and install the extension.
7. Confirm uBO Scope’s icon appears on the toolbar – it should show a badge count after browsing.

### 2.2 Mozilla Firefox

1. Open Firefox.
2. Navigate to the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click the **Add to Firefox** button.
4. A permissions dialogue will display asking to allow the extension’s required rights, including network request monitoring.
5. Click **Add**.
6. After installation, look for the uBO Scope icon in your toolbar or the overflow menu.
7. The badge count on the icon updates when browsing.

### 2.3 Safari

1. Open Safari on macOS.
2. Access the Safari Extensions Marketplace or the dedicated Safari extensions location.
3. Search for **“uBO Scope”** in the extensions list.
4. Click **Install** or **Get** next to uBO Scope.
5. Safari may ask for permission to enable the extension; enable it in Safari Preferences > Extensions.
6. Confirm the uBO Scope icon is visible in the Safari toolbar.


## 3. Confirming the Installation

After installation, validate that uBO Scope is correctly added and active:

- Ensure the uBO Scope icon appears at the toolbar of your browser.
- Start browsing a few websites.
- Click the uBO Scope icon to open its popup interface displaying connected domains.
- The badge count should reflect the number of distinct third-party remote servers connected to, with a number or empty if none.


## 4. Tips and Recommendations

- **Pin the Icon:** For easier access, pin the uBO Scope icon to your browser’s toolbar.
- **Check Permissions:** If the badge count does not appear, verify that the extension has necessary permissions in your browser settings.
- **Browser Updates:** Keep your browser up to date to ensure full compatibility.


## 5. Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting Installation Issues">
<Accordion title="Extension Icon Does Not Appear After Installation">
- Refresh the browser and check the extension menu/overflow.
- Pin the extension icon manually via your browser’s toolbar customization.
- Restart your browser to refresh extensions.
</Accordion>
<Accordion title="Permissions Prompt Was Declined">
- Extensions require network permissions to function. Reinstall or adjust extension permissions via your browser's settings.
</Accordion>
<Accordion title="Unable to Install from Store">
- Verify your browser version meets the minimum required versions.
- Check internet connectivity and retry.
- Clear browser cache or try installation in a private window.
</Accordion>
</AccordionGroup>
