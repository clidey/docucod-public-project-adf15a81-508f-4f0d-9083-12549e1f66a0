# Installing uBO Scope

This guide provides clear, step-by-step instructions to install uBO Scope on Chrome, Firefox, and Safari using their official extension channels. Follow these instructions to ensure a quick, secure installation and verify the extension is working correctly.

---

## 1. Prerequisites & System Requirements

Before installing uBO Scope, ensure your environment meets the following requirements:

- **Supported Browsers:**
  - **Google Chrome:** Version 122 or higher
  - **Mozilla Firefox:** Version 128 or higher
  - **Safari:** Version 18.5 or higher

- **Permissions:**
  - The extension requires permissions for `webRequest`, `activeTab`, and `storage` to monitor network connections effectively.

- **Network:**
  - The extension monitors HTTP, HTTPS, WebSocket (ws:// and wss://) connections.
  - Ensure your browser and network do not block access to the extension stores.

- **User Rights:**
  - You need permission to install browser extensions on your device.

<Tip>
If you are using a managed device (work or school), check with your administrator as installation of extensions might be restricted.
</Tip>

---

## 2. Installation Instructions

Follow the steps for your browser to install uBO Scope from the official extension stores. Installing from official channels guarantees security and automatic updates.

### Google Chrome

1. Open Chrome and navigate to the [uBO Scope page on the Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click the **Add to Chrome** button.
3. In the confirmation dialog, click **Add extension**.
4. Wait a few seconds as Chrome downloads and installs the extension.
5. After installation, uBO Scope’s icon appears in the toolbar.

### Mozilla Firefox

1. Open Firefox and go to the [uBO Scope page on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click the **Add to Firefox** button.
3. In the permission prompt, click **Add**.
4. Firefox installs the extension, and you’ll see uBO Scope’s icon in the browser toolbar.

### Safari

1. Open Safari, then open the Mac App Store.
2. Search for 'uBO Scope' or visit the official listing if available.
3. Click **Get** or **Install**.
4. After installation, open Safari Preferences > Extensions.
5. Enable uBO Scope by checking its checkbox.
6. The uBO Scope icon will appear near the Safari toolbar.

<Warning>
Safari requires manually enabling extensions after installation due to privacy restrictions.
</Warning>

## 3. Verifying Successful Installation

Once installed, verify that uBO Scope is working correctly to ensure you can start monitoring network connections immediately.

### Steps to Verify

1. Locate the uBO Scope toolbar icon in your browser:
   - In Chrome and Firefox, it appears near the address bar.
   - In Safari, it appears after enabling in Extensions preferences.

2. Visit a website (e.g., https://example.com).
3. Click the uBO Scope icon to open the popup panel.
4. Confirm the panel displays:
   - The domain name of the current tab.
   - The count of connected remote domains under “domains connected”.
   - Lists under sections 'not blocked', 'stealth-blocked', and 'blocked' showing domains and connection counts.

5. Look for a badge count on the extension icon indicating the number of third-party remote servers connected.

<Tip>
If the popup shows "NO DATA" or empty lists, try refreshing the page or visiting a different site to generate network activity.
</Tip>

## 4. Troubleshooting Common Installation Issues

### Extension Icon Not Visible
- Make sure the extension is enabled in your browser’s extensions or add-ons manager.
- In Chrome, pin the extension icon by clicking the puzzle piece icon and selecting the pin next to uBO Scope.
- In Firefox, check if the extension is disabled and enable it.

### Popup Displays No Data
- After installation, network data might not be captured immediately.
- Reload active tabs or open new tabs with different sites.
- Verify that the browser version meets minimum requirements.

### Installation Fails
- Confirm you have the necessary permissions to install extensions.
- Check internet connectivity and access to the extension store.

<Note>
If problems persist, see the [Resolving Common Installation & Activation Issues](https://docs.example.com/getting-started/troubleshooting/common-issues) guide.
</Note>
