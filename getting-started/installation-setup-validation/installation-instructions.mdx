# Install uBO Scope

This guide walks you through the step-by-step process of installing uBO Scope, a browser extension that reveals all remote server connections your browser attempts or completes. You will learn how to install the extension on Chrome (and other Chromium-based browsers), Firefox, and Safari using official browser extension marketplaces, including important details on permissions and compatibility.

---

## 1. Prerequisites

Before installation, ensure your browser meets the minimum requirements to run uBO Scope effectively:

- **Chrome and other Chromium-based browsers:** Version 122 or higher.
- **Firefox:** Version 128 or higher.
- **Safari:** Version 18.5 or higher.

<u>**Permissions:**</u> uBO Scope requires access to certain browser APIs and host permissions:

- Access to "activeTab" allows the extension to interact with your current tab.
- Storage permission enables saving preferences and state.
- `webRequest` permission to monitor network requests.
- Host permissions for all HTTP, HTTPS, WebSocket and secure WebSocket URLs:
  - Chromium: `http://*/*`, `https://*/*`, `ws://*/*`, `wss://*/*`
  - Firefox: same as Chromium
  - Safari: `http://*/*`, `https://*/*`

Ensure you have the ability to install extensions from the browser’s official extension store.

---

## 2. Installation Instructions

### Chrome and Chromium-Based Browsers

1. Open your Chromium-based browser (e.g., Google Chrome, Edge, Brave).
2. Navigate to the uBO Scope listing on the [Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click **Add to Chrome**.
4. In the permissions dialog that appears, review the requested permissions and click **Add extension** to confirm.
5. Wait for the installation to complete. The uBO Scope icon will appear in your toolbar.

### Mozilla Firefox

1. Open Firefox browser.
2. Go to the uBO Scope page on [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click **Add to Firefox**.
4. Verify the permissions and click **Add**.
5. Confirm by clicking **OK, Got it** after installation.
6. The uBO Scope icon will show up in your browser’s toolbar.

### Safari

1. Open Safari (version 18.5+).
2. Open the [App Store on your Mac](https://apps.apple.com) or Safari Extensions page when available.
3. Search for "uBO Scope".
4. Install the extension directly from the App Store listing.
5. After installation, enable uBO Scope via Safari Preferences > Extensions.

<u>Note:</u> Safari permissions include `activeTab`, `storage`, and `webRequest`, with host permissions limited to HTTP and HTTPS URLs.

---

## 3. Required Permissions & What They Mean

- **activeTab**: Allows the extension to access the currently focused tab.
- **storage**: Saves your configuration and state securely.
- **webRequest**: Monitors network requests to report allowed, blocked, and stealth-blocked connections.
- **host permissions**: Covers all web and websocket URLs to capture all network requests. 

These permissions enable uBO Scope to provide accurate and comprehensive visibility of your browsing activity related to network connections.

---

## 4. Verification After Installation

Once installed:

- Look for the uBO Scope icon in your browser toolbar.
- Click the icon to open the popup interface.
- Observe the badge count indicating distinct third-party remote servers connected per active tab.
- Visit a website and refresh the page to see the network connection counts update.

## 5. Troubleshooting Common Issues

### Extension Icon Does Not Appear
- Ensure the extension is enabled in the browser's extensions or add-ons manager.
- Restart the browser if necessary.

### Permissions Not Accepted
- Review requested permissions carefully. The extension requires `webRequest` and host permissions to function properly.
- If uncertain, revisit the extension listing for detailed permission information.

### Browser Version Compatibility
- Verify your browser meets minimum version requirements (Chromium 122+, Firefox 128+, Safari 18.5+).
- Update your browser if needed before installing uBO Scope.
