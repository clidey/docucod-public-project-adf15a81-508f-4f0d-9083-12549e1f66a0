# Installing uBO Scope

Follow these clear, step-by-step instructions to install uBO Scope on your preferred browser. This guide covers how to locate the official extension sources and install uBO Scope for Chrome, Firefox, and Safari, ensuring a smooth start to leveraging its network connection insights.

---

## 1. Prerequisites & Compatibility

Before installation, verify your system meets these requirements:

- **Supported Browsers:**
  - Google Chrome (version 122.0 or later)
  - Mozilla Firefox (version 128.0 or later)
  - Safari (version 18.5 or later)
- **Permissions:** The extension requires permission to monitor active tabs, web requests, and to store data.
- **Network Access:** Ensure your browser allows extension access to network activity through the `webRequest` API.

<Tip>
If you are using browsers other than the above or older versions, uBO Scope may not function properly.
</Tip>

---

## 2. Where to Get uBO Scope

Only install uBO Scope from the official, trusted browser extension stores to guarantee authenticity and security.

| Browser | Official Source | Link |
|---------|-----------------|------|

| Firefox | Firefox Add-ons | [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/) |
| Safari  | Safari Extensions Gallery | Available via Safari's Extensions preference pane (macOS Sonoma 14+) |

---

## 3. Installation Instructions

Follow these steps based on your browser to install uBO Scope:

### For Google Chrome

1. Open the Chrome Web Store link: [uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click the **Add to Chrome** button.
3. Confirm installation by clicking **Add extension** in the prompt.
4. Wait for the extension to download and install; you will see the uBO Scope icon appear in the toolbar.

### For Mozilla Firefox

1. Navigate to the Firefox Add-ons page: [uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click the **Add to Firefox** button.
3. Review the permissions and click **Add** to continue.
4. Upon completion, the uBO Scope icon will appear in your browser toolbar.

### For Safari (macOS Sonoma 14 and later)

1. Open **Safari**, then go to **Preferences > Extensions**.
2. Search for **uBO Scope** in the Extensions Gallery.
3. Click **Install** and follow the on-screen prompts.
4. Once installed, enable uBO Scope by checking its box in the Extensions pane.

<Warning>
Safari requires macOS Sonoma 14 or later for installing the uBO Scope extension.
</Warning>

---

## 4. Enabling and Verifying Installation

After installing:

1. **Enable the Extension** if not automatically enabled:
   - Chrome and Firefox: Usually enabled by default.
   - Safari: Enable in **Preferences > Extensions**.
2. **Locate the uBO Scope Icon** in your browser’s toolbar.
3. **Click the Icon** to open the popup UI.
4. You should see a placeholder interface displaying domains connected information once browsing.

<Tip>
If the icon is missing, check your browser’s extension manager to confirm that uBO Scope is installed and enabled.
</Tip>

---

## 5. First Use: Seeing Network Connections

Once installed and enabled, uBO Scope starts tracking all outgoing network connections from the active tab:

1. Browse to any website.
2. Click the uBO Scope toolbar icon.
3. Observe the popup showing three categories:
   - **Not Blocked:** Connections allowed to remote servers.
   - **Stealth Blocked:** Connections stealthily blocked by content blockers.
   - **Blocked:** Connections explicitly blocked.
4. The badge on the toolbar icon shows the count of distinct third-party remote servers connected.

<Info>
Lower badge counts typically indicate fewer distinct third-party connections, generally aligned with more selective blocking.
</Info>

## 6. Troubleshooting Installation Issues

<AccordionGroup title="Common Installation Issues and Solutions">
<Accordion title="uBO Scope icon does not appear after installation">
- Verify the extension is enabled in your browser’s extension manager.
- Restart your browser to ensure all components reload.
- For Safari, double-check macOS version compatibility and enable it manually under Safari Extensions.
</Accordion>
<Accordion title="Extension popup shows 'NO DATA' or no connections listed">
- Ensure you have permission to access active tab data.
- Visit some websites to generate network requests before viewing the popup.
- Check for conflicting extensions that restrict webRequest API access.
</Accordion>
<Accordion title="Badge count remains empty despite browsing">
- uBO Scope counts distinct third-party remote servers. Use websites with known third-party content.
- Some network requests might be invisible if blocked at the OS or network level outside the browser.
</Accordion>
</AccordionGroup>

<Note>
For persistent issues, consult the official repository and community support for advanced help: https://github.com/gorhill/uBO-Scope
</Note>
