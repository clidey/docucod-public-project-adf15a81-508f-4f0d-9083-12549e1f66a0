# Installing uBO Scope

Get started with uBO Scope by installing the extension on your preferred browser. This guide provides clear, step-by-step instructions for Chromium-based browsers (like Google Chrome and Edge), Mozilla Firefox, and Apple Safari. Each section includes direct links to the official browser extension stores and outlines any prompts or choices you will encounter during installation.

---

## 1. Supported Browsers and Versions

Before proceeding, ensure your browser meets the minimum requirements:

- **Chromium Browsers (Google Chrome, Edge, etc.)**: Version 122.0 or newer
- **Mozilla Firefox**: Version 128.0 or newer
- **Safari**: Version 18.5 or newer

Refer to the [System Requirements and Supported Browsers](../setup-and-installation/prerequisites-browsers) page for detailed system compatibility.

---

## 2. Installation Instructions

### 2.1 Installing on Chromium-based Browsers

1. Open your Chromium-based browser.
2. Navigate to the official uBO Scope page in the Chrome Web Store:

   [Install uBO Scope for Chromium](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click the **Add to Chrome** button.
4. When prompted, confirm by selecting **Add Extension**.
5. Wait a few moments while the extension installs.
6. Once installed, the uBO Scope icon will appear in your browser's toolbar.

### 2.2 Installing on Mozilla Firefox

1. Open Mozilla Firefox.
2. Visit the official uBO Scope add-on page:

   [Install uBO Scope for Firefox](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click the **Add to Firefox** button.
4. In the confirmation popup, click **Add**.
5. Optionally, review and allow any permission requests.
6. Once installation completes, the uBO Scope icon will be visible in the Firefox toolbar.

### 2.3 Installing on Apple Safari

> **Important**: Make sure your Mac is running Safari 18.5 or later to support uBO Scope.

1. Open Safari on your Mac.
2. Access the App Store and search for **uBO Scope** or use the direct link below if available.
3. Download and install the uBO Scope Safari extension.
4. After installation, open Safari’s **Settings** > **Extensions**.
5. Enable the uBO Scope extension by checking its box.
6. Confirm any permission prompts to allow the extension to access website data.


---

## 3. Post-Installation Steps and Tips

- After installation, no special configuration is required to begin using uBO Scope.
- The extension icon will indicate how many distinct third-party remote servers your browser has contacted on any active tab.
- If the icon does not appear, check your browser’s extension management page and confirm uBO Scope is enabled.

---

## 4. Common Installation Prompts Explained

- **Permission requests**: The extension requires permissions such as access to active tabs, storage, and monitoring web requests to function. Always allow these permissions for full functionality.
- **Extension warnings**: Some browsers may warn that the extension can monitor your browsing activity. This is necessary because uBO Scope needs to observe network requests to report server connections.

<Note>
When installing from official stores, you ensure you receive authentic updates and security patches for uBO Scope.
</Note>

---

## 5. Verifying Successful Installation

To confirm uBO Scope is installed and active:

- Check that the extension icon appears in your browser’s toolbar.
- Click the icon to open the popup interface which shows domains connected on the current tab.
- Visit a website and observe the badge count on the extension icon updating with the number of distinct remote servers.

If this behavior works as described, the installation is successful.

---

## 6. Troubleshooting Installation Issues

### Issue: uBO Scope icon does not appear after installation.
- Verify the extension is enabled in your browser’s extensions or add-ons manager.
- Restart your browser; some browsers require a restart after extension installation.

### Issue: Unable to find uBO Scope in the browser store.
- Confirm your browser is updated to the minimum supported version.
- Use the direct links provided above to access correct extension pages.

### Issue: Prompt for permissions not showing or being denied.
- Ensure your browser’s extension permissions or policies do not restrict installation.
- Manually enable required permissions from the browser extension management interface.

For further assistance, refer to [Troubleshooting Common Setup Issues](../first-run-usage/troubleshooting-install) documentation.
