# Installing uBO Scope

Welcome to the guide for installing **uBO Scope**, a browser extension that transparently monitors remote server connections from your browser. This page provides straightforward, step-by-step instructions to install uBO Scope from the official extension stores for supported browsers including **Chrome**, **Firefox**, and **Safari** (if available).

---

## 1. Supported Browsers and Requirements

Before installing, ensure your environment meets the following requirements:

- **Browsers Supported:**
  - Google Chrome (version 122.0 or later)
  - Mozilla Firefox (version 128.0 or later)
  - Safari (version 18.5 or later, if supported)

- **Permissions Needed:** uBO Scope requires permissions to monitor active tabs, access storage, and intercept web requests to report connection information properly.

- **Network:** No special network configuration is required but the extension needs to operate with standard internet access.

<Tip>
Make sure your browser is up to date to ensure compatibility with uBO Scope’s latest version and features.
</Tip>

---

## 2. Installation Instructions

Follow the instructions below according to your browser:

<Steps>
<Step title="Install on Google Chrome">
1. Open the [Chrome Web Store page for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome**.
3. Confirm the installation in the popup dialog by clicking **Add extension**.
4. Wait a few moments for the extension to install – the uBO Scope icon will appear in your toolbar.
</Step>
<Step title="Install on Mozilla Firefox">
1. Open the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click the **Add to Firefox** button.
3. When prompted, click **Add** to confirm permissions and complete installation.
4. The extension icon will become visible in your browser toolbar after installation.
</Step>
<Step title="Install on Safari (If Available)">
1. Visit the Mac App Store or the Safari Extensions page where uBO Scope might be listed.
2. Locate uBO Scope and click **Get** or **Install**.
3. Follow any on-screen prompts, including enabling the extension in Safari’s preferences under the Extensions tab.
4. Confirm that uBO Scope’s icon appears on the Safari toolbar.
</Step>
</Steps>

<Tip>
If you do not find uBO Scope in Safari's official extension gallery, it may not yet be available for your Safari version.
</Tip>

---

## 3. Verifying a Successful Installation

Once installed, validate that uBO Scope is working correctly:

1. Look for the uBO Scope icon in your browser’s toolbar (usually near the address bar).
2. Click the icon to open the popup panel.
3. The popup should display a default message or current tab hostname and network connection summary.
4. Browse some popular websites to see counts of third-party remote server connections update dynamically.

<Check>
If the popup shows "NO DATA" or the icon does not respond, reload the browser or restart it and try again.
</Check>

---

## 4. Practical Tips for Smooth Installation

- **Browser Profiles:** If you use multiple profiles, install uBO Scope separately for each profile.
- **Extension Conflicts:** Disable or remove other extensions that may block extension APIs if you encounter issues.
- **Permissions Review:** Confirm the extension has necessary permissions in your browser’s extensions settings.

---

## 5. Troubleshooting Common Installation Issues

<AccordionGroup title="Common Installation Problems and Solutions">
<Accordion title="Extension Icon Does Not Appear">
- Ensure your browser version meets minimum requirements.
- Try restarting the browser.
- Check the browser’s extensions management page to confirm the extension is enabled.
</Accordion>
<Accordion title="Pop-up Shows No Data After Installation">
- Close and reopen the browser tab.
- Visit different websites to trigger network requests.
- Verify the extension has the necessary permissions (especially `webRequest`).
</Accordion>
<Accordion title="Extension Fails to Install or Add Button is Disabled">
- Ensure you are logged into your browser’s account (for Chrome and Firefox).
- Check internet connectivity and browser settings that may restrict extension installation.
- Temporarily disable security software that may block extension downloads.
</Accordion>
</AccordionGroup>
