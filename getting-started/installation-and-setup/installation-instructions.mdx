# Installing uBO Scope

Follow these step-by-step instructions to install uBO Scope securely and smoothly from the official stores for Chromium-based browsers, Firefox, and Safari. Each platform has specific official sources and version requirements, ensuring the extension runs optimally and safely.

---

## 1. Supported Browsers & Minimum Versions

Before installation, verify your browser meets the following minimum version requirements:

- **Chromium-based browsers** (Google Chrome, Microsoft Edge, Brave, etc.): version 122 or later
- **Firefox**: version 128 or later
- **Safari**: version 18.5 or later

uBO Scope requires these versions to leverage the necessary browser APIs and deliver consistent operation.

---

## 2. Installing on Chromium

1. Open your Chromium-based browser.
2. Visit the Chrome Web Store page for uBO Scope:

   [uBO Scope on Chrome Web Store](https://chrome.google.com/webstore/detail/ublock-origin-scope/mhigpdcjgiojemgpoepecgmolgaifoed) _(replace with actual if different)_

3. Click **Add to Chrome** (or **Add to [Browser Name]**).
4. Confirm by clicking **Add extension** in the popup dialog.
5. Wait for the extension to install; the uBO Scope icon will appear in your toolbar.

### Verification:
- After installation, the extension icon should appear in the browser tools area.

### Tips:
- Ensure you are installing the extension published by Raymond Hill (uBlock Origin’s original author).
- If you do not see the icon, check your browser’s extensions menu or toolbar customization settings.

---

## 3. Installing on Firefox

1. Open Firefox browser.
2. Navigate to the official Mozilla Add-ons page for uBO Scope:

   [uBO Scope on Mozilla Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/) _(replace with actual if different)_

3. Click **Add to Firefox**.
4. In the permission prompt, click **Add** to confirm installation.
5. Restart Firefox if prompted.
6. Confirm installation by locating the uBO Scope icon near the address bar.

### Verification:
- The extension name should be visible in the Add-ons Manager (`about:addons`).

### Tips:
- Be sure to have Firefox updated to version 128 or newer for full support.
- When prompted for permissions, reading and accepting them is necessary for the extension to monitor network requests.

---

## 4. Installing on Safari

1. Open Safari browser on macOS.
2. Open the [Mac App Store](https://apps.apple.com) or the Safari Extensions section.
3. Search for the uBO Scope Safari extension.
4. Click **Get** or the download button next to uBO Scope.
5. After installation, open Safari > Preferences > Extensions and enable uBO Scope.

### Verification:
- Check that uBO Scope is listed and enabled under Safari's Extensions settings.

### Tips:
- Ensure your macOS and Safari versions meet the minimum requirement (Safari 18.5 or later).
- Permissions requested by the extension must be accepted for full functionality.

---

## 5. Post-Installation: Ensuring a Secure Setup

- Only install uBO Scope from the official browser stores linked above; avoid third-party sites.
- Confirm the extension author’s name "Raymond Hill" and the extension version (currently 1.0.1) in the extension details.
- Review requested permissions such as access to active tabs, storage, and webRequest.
- Keep your browser and the extension updated regularly for security and feature improvements.

---

## 6. Troubleshooting Installation Issues

<AccordionGroup title="Common Installation Issues & Solutions">
<Accordion title="Extension Icon Not Visible After Installation">
- Check your browser’s extensions/add-ons page to confirm uBO Scope is listed and enabled.
- In Chromium, open the Extensions menu and pin the uBO Scope icon to the toolbar.
- Restart your browser and try again.
</Accordion>

<Accordion title="Browser Shows Extension Not Compatible">
- Verify your browser meets the minimum version requirement (Chromium 122+, Firefox 128+, Safari 18.5+).
- Update your browser to the latest stable version.
- If on a managed or corporate device, check with IT policies restricting extensions.
</Accordion>

<Accordion title="Permissions Denied During Installation">
- uBO Scope requires permissions to monitor network connections; denying them will limit functionality.
- Carefully review and allow requested permissions.
- If uncertain, reinstall and carefully accept permissions.
</Accordion>

<Accordion title="Extension Installation Suspended or Blocked by Security Software">
- Temporarily disable any security or antivirus software that might block browser extensions.
- Add trusted exceptions for uBO Scope if possible.
- Consult the security software documentation for guidance on allowing browser extensions.
</Accordion>
</AccordionGroup>
