# Install uBO Scope

Follow these detailed instructions to install uBO Scope, the browser extension that reveals third-party remote server connections. Installation steps cover the three primary supported browsers: Chromium-based browsers (including Chrome, Edge), Firefox, and Safari. Choose your preferred method below for a quick and reliable setup.

---

## 1. Prerequisites & System Requirements

Before installing uBO Scope, ensure your environment meets the following:

- **Browsers Supported:**
  - Chromium-based browsers (Chrome, Edge) version 122.0 or above
  - Firefox version 128.0 or above
  - Safari version 18.5 or above

- **Permissions Required:**
  uBO Scope requires these browser permissions to function correctly:
  - `activeTab`
  - `storage`
  - `webRequest`

- **Network Requirements:**
  The extension must be able to observe network requests via the browser's `webRequest` API for accurate reporting. Firewalls or browser policies blocking this API will limit functionality.

- **Operating System:**
  Compatible with OS versions supported by the above browser versions.

<Tip>
Verify your browser version before proceeding by navigating to your browser's `About` page.
</Tip>

---

## 2. Installation Instructions

### Chromium-Based Browsers (Chrome, Edge)

#### Method 1: Install via Chrome Web Store (Recommended)

1. Open the Chrome Web Store page for uBO Scope:

   [Install uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

2. Click the **Add to Chrome** (or **Add to Edge**) button.
3. Confirm any prompts to add the extension.
4. After installation, the uBO Scope icon will appear in your browser toolbar.


#### Method 2: Manual Installation (Development or Offline Use)

1. Clone or download the uBO Scope repository:
   ```bash
   git clone https://github.com/gorhill/uBO-Scope.git
   cd uBO-Scope
   ```
2. Build the Chromium package (requires Linux environment and bash):
   ```bash
   ./tools/make-chromium.sh
   ```
   This will create a package under `build/uBO-Scope.chromium`.
3. Open Chrome or Edge and navigate to `chrome://extensions/`.
4. Enable **Developer mode** (toggle on the top-right).
5. Click **Load unpacked** and select the `build/uBO-Scope.chromium` folder.
6. Confirm the extension is loaded and the icon appears in the toolbar.

<Warning>
Manual installation requires enabling Developer mode and may not auto-update your extension.
</Warning>

---

### Firefox

#### Method 1: Install via Firefox Add-ons Store (Recommended)

1. Visit the Firefox Add-ons page:

   [Install uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

2. Click **Add to Firefox**.
3. Confirm any permission prompts.
4. Upon installation, uBO Scope's icon will be visible near the address bar.

#### Method 2: Manual Installation

1. Clone or download the uBO Scope repository:
   ```bash
   git clone https://github.com/gorhill/uBO-Scope.git
   cd uBO-Scope
   ```
2. Build the Firefox package:
   ```bash
   ./tools/make-firefox.sh
   ```
   This generates an `.xpi` file in the repository root.
3. In Firefox, open the **Add-ons** page (`about:addons`).
4. Click the gear icon ⚙️, then select **Install Add-on From File...**
5. Choose the generated `.xpi` file.
6. Confirm installation and enable the extension.

---

### Safari

Due to Safari's extension management, installation requires building the package manually.

1. Clone or download the uBO Scope repository:
   ```bash
   git clone https://github.com/gorhill/uBO-Scope.git
   cd uBO-Scope
   ```
2. Build the Safari package:
   ```bash
   ./tools/make-safari.sh
   ```
   This generates a ZIP package under `build/uBO-Scope.safari.zip`.
3. Follow Safari’s instructions to install a developer extension package:
   - Open Safari, go to **Preferences > Extensions**.
   - Enable **Developer Mode** (if needed).
   - Use **Install Extension** to select the ZIP package.
4. After installation, ensure uBO Scope is enabled in Safari's extensions.

<Warning>
Safari support requires version 18.5 or above and manual developer extension installation.
</Warning>

## 3. Verification After Installation

Confirm that uBO Scope is correctly installed and active:

- Look for the uBO Scope icon in your browser toolbar.
- Click the icon to open the popup interface (a small window showing network info).
- Visit any typical website; the extension should display domains connected categorized as 'not blocked', 'stealth-blocked', or 'blocked'.

<Check>
If the toolbar icon does not appear or the popup shows 'NO DATA' persistently, confirm you allowed necessary permissions and that the extension is enabled.
</Check>

## 4. Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting Installation Problems">
<Accordion title="uBO Scope Icon Does Not Appear">
- Verify the browser version meets minimum requirements.
- Check that the extension is enabled in the browser's extension/add-ons page.
- Reload the browser or restart it.
- For manual install, ensure the unpacked folder or XPI file was correctly loaded.
</Accordion>
<Accordion title="Popup Shows 'NO DATA' After Installation">
- Ensure you have visited websites that make typical network requests.
- Confirm `webRequest` permission is granted.
- Disable conflicting extensions temporarily to check for interference.
- Try reinstalling or updating the extension.
</Accordion>
<Accordion title="Manual Install Fails with Errors">
- Confirm your environment matches prerequisites (Linux bash shell for build scripts).
- Verify that required tools such as `jq` (JSON processor) are installed.
- Check for correct folder permissions.
- Follow platform-specific developer mode instructions carefully.
</Accordion>
</AccordionGroup>
