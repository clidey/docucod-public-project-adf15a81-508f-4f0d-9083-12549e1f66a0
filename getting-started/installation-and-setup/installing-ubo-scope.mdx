# Installing uBO Scope

Step-by-step, platform-specific installation instructions for Chrome, Firefox, and Safari. Learn where and how to get the extension, including direct web store links and quick validation of a successful install via the browser toolbar.

---

## 1. Prerequisites & System Requirements

Before installing uBO Scope, ensure your system and browser meet the following:

- **Supported Browsers:**
  - Google Chrome 122.0 or higher
  - Mozilla Firefox 128.0 or higher
  - Safari 18.5 or higher
- **Operating Systems:** Windows, macOS, Linux for Chrome and Firefox; macOS for Safari
- **Permissions:** The extension requires permission to access data on websites (via `webRequest` API), storage access, and active tab control.

<Tip>
Make sure your browser is updated to the minimum required version to ensure compatibility and correct extension functionality.
</Tip>

Refer to the [System & Browser Requirements](https://your-documentation-domain/getting-started/installation-and-setup/system-requirements) page for more details.

---

## 2. Installation Instructions

Choose your browser below and follow the respective steps.

### Google Chrome

1. Open Chrome and navigate to the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome**.
3. In the confirmation popup, click **Add extension**.
4. Wait for the extension to install.

### Mozilla Firefox

1. Open Firefox and go to the [Firefox Add-ons uBO Scope page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. Confirm by clicking **Add** in the permissions dialog.
4. The extension installs automatically.

### Safari

1. Open the Safari Extensions Gallery or Safari's Extensions preferences.
2. Search for 'uBO Scope' or open the extension file if downloaded.
3. Enable the extension in Safari’s Extensions preferences.

<Warning>
Due to Safari’s security model, uBO Scope requires Safari version 18.5 or later.
</Warning>

---

## 3. Verifying Installation

After installation, ensure that uBO Scope is correctly installed and enabled:

1. Look for the uBO Scope icon in your browser’s toolbar. The icon will be a small shield with the uBO Scope logo.
2. Click the icon; a popup window should appear showing connection information (initially "NO DATA" until you visit a webpage).
3. Verify permissions:
   - Chrome/Firefox: Check extension details to ensure `webRequest` and storage permissions are granted.
   - Safari: Verify the extension is enabled in Safari preferences.

<Tip>
If you do not see the icon after installation, check your browser’s extensions or add-ons menu and ensure uBO Scope is activated.
</Tip>

---

## 4. Post-Installation Quick Steps

Once installed, you can quickly validate uBO Scope’s basic operation:

1. Open a new browser tab and visit any website.
2. Click the uBO Scope icon in the toolbar.
3. Observe the popup badge count and details; you should see the number of distinct third-party remote servers connected for the page.

  - **Badge count:** Reflects distinct third-party domains connected.
  - **Popup sections:** Categorized views for allowed, stealth-blocked, and blocked domains.

<Note>
The badge count is a direct indicator of third-party connections detected on the current tab. A lower count generally indicates fewer connections to remote servers.
</Note>

## 5. Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting Common Installation Issues">
<Accordion title="uBO Scope icon does not appear after installation">
- Ensure your browser version meets the minimum requirement.
- Check the extension or add-on settings to confirm it is enabled.
- For Chrome or Firefox, try restarting your browser.
- Verify no other extension is conflicting (try disabling others temporarily).
</Accordion>
<Accordion title="Permissions prompt did not appear or was denied">
- For full functionality, uBO Scope needs permissions such as webRequest and storage.
- If denied, uninstall and reinstall, approving all permission prompts.
- Chrome and Firefox will show these prompts automatically; Safari requires manual enabling.
</Accordion>
<Accordion title="Safari installation is not straightforward or fails">
- Make sure Safari version is 18.5 or later.
- Installation may require downloading from the App Store or Apple's developer portal.
- Verify the extension is enabled under Safari Preferences > Extensions.
- If issues persist, consult Safari’s documentation or support.
</Accordion>
</AccordionGroup>
