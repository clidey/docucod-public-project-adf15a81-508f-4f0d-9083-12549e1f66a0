# Installing uBO Scope

Start exploring your browser's third-party network connections by installing uBO Scope. This guide delivers precise, step-by-step instructions to install the extension on supported browsers: Chrome, Firefox, and Safari. Follow these tailored instructions to ensure a smooth installation process.

---

## Supported Browsers & Compatibility

Before proceeding, verify that your browser version meets the minimum requirements to run uBO Scope:

- **Google Chrome (Chromium-based browsers):** Version 122 or higher
- **Mozilla Firefox (desktop and Android):** Version 128 or higher
- **Safari:** Version 18.5 or higher

Ensure you are running a compatible browser version to avoid installation or runtime issues.

---

## Step-by-Step Installation Instructions

### 1. Installing on Google Chrome or Chromium-based Browsers

1. Open your Chromium-based browser (e.g., Chrome, Microsoft Edge).
2. Navigate to the official Chrome Web Store page for uBO Scope:

   [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click the **Add to Chrome** button.
4. In the confirmation prompt, select **Add extension**.
5. Wait for the browser to download and install the extension.
6. After installation, the uBO Scope icon will appear in your toolbar near the address bar.

<Tip>
If the icon does not appear immediately, click the browser's extension menu (puzzle piece icon) and pin uBO Scope for quick access.
</Tip>

### 2. Installing on Mozilla Firefox

1. Open the Firefox browser.
2. Visit the Mozilla Add-ons page for uBO Scope:

   [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click the **Add to Firefox** button.
4. When prompted, click **Add** to confirm.
5. Upon successful installation, the uBO Scope icon will be visible in the toolbar.

<Warning>
Make sure you are using Firefox version 128 or above, otherwise installation will be blocked due to compatibility restrictions.
</Warning>

### 3. Installing on Safari

uBO Scope supports Safari version 18.5 and later.

1. Open Safari on your Mac.
2. Go to the Mac App Store or the Safari Extensions Gallery.
3. Search for "uBO Scope" or visit the direct Safari extension link (on confirmation).
4. Click **Get** or **Install** to download.
5. After installation, enable the extension from Safari's Preferences:
   - Open Safari Preferences (`Safari` > `Settings` or `Preferences` depending on your macOS version).
   - Switch to the **Extensions** tab.
   - Find **uBO Scope** in the list.
   - Check the box to enable the extension.

<Note>
Safari limits some extension capabilities for security reasons. Ensure all permissions requested by uBO Scope are granted for optimal operation.
</Note>

## Verifying Your Installation

After installing uBO Scope, follow these steps to confirm it is correctly installed and ready to use:

1. Locate the uBO Scope icon in your browser’s toolbar.
2. Click the icon to open its popup panel.
3. The panel should display "NO DATA" initially — this indicates uBO Scope is ready but no browsing activity has been recorded yet.
4. Visit any website (e.g., https://example.com) to generate network requests.
5. Click the uBO Scope icon again, and you should see a list of connected domains along with counts under categories such as **not blocked**, **stealth-blocked**, and **blocked**.

<Check>
The badge near the icon should show a count representing the distinct third-party domains connected by the active tab.
</Check>

## Best Practices and Tips

- **Grant Necessary Permissions:** During installation, uBO Scope requires permissions like `activeTab`, `webRequest`, and `storage` to monitor network requests accurately.
- **Browser Updates:** Regularly update your browser to ensure seamless compatibility with the extension.
- **Pin the Extension:** Pin uBO Scope’s icon for immediate access while browsing.
- **Reload Tabs:** After installation, reload open tabs to start capturing connection data immediately.

---

## Common Pitfalls

- **Browser Version Too Low:** Older browser versions may refuse to install the extension or cause malfunction. Confirm your browser version before installation.
- **Extension Not Enabled:** On Safari, make sure uBO Scope is enabled in the Extensions Preferences.
- **Permissions Denied:** If permissions are denied during installation, the extension cannot function properly. Reinstall and allow the requested permissions.
- **Icon Missing:** If the icon doesn’t show up, check the extensions menu and pin it.
