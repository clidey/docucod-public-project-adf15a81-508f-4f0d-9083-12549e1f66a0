# Install uBO Scope from Official Sources

This guide walks you through installing uBO Scope, a lightweight browser extension that reveals all third-party remote server connections your browser makes, directly from trusted official platforms. Choose the installation method matching your browser: Chrome (Chromium-based), Firefox, or Safari. Following these steps ensures you get the authentic extension maintained by Raymond Hill.

---

## 1. Supported Browsers and Platforms

uBO Scope supports these browsers with minimum version requirements:

- **Google Chrome, Microsoft Edge, and other Chromium-based browsers** (version 122 or later)
- **Mozilla Firefox** (version 128 or later)
- **Apple Safari** (version 18.5 or later)

Make sure your browser version meets these to guarantee smooth installation and operation.

---

## 2. Installation Options

You have three official sources to install uBO Scope, based on your browser:

### 2.1 Chrome Web Store (Chromium Browsers)

1. Open your Chromium-based browser (including Google Chrome, Microsoft Edge, Brave, etc.).
2. Visit the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click **'Add to Chrome'** (or corresponding button for your browser).
4. Confirm the permission prompt by clicking **'Add extension'**.

> The extension icon will appear next to your address bar once installed.

### 2.2 Firefox Add-ons (Mozilla Firefox)

1. Open Mozilla Firefox.
2. Navigate to the [uBO Scope Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click **'Add to Firefox'**.
4. Review permissions and click **'Add'**.
5. On the confirmation prompt, click **'Okay, Got It'**.

> You will see the uBO Scope icon near the browser toolbar after installation.

### 2.3 Safari Extensions Store (Apple Safari)

1. Open Safari on macOS 14 (Sonoma) or later.
2. Go to the Mac App Store and search for "uBO Scope" or visit the official listing if available.
3. Click **'Get'** and proceed to install.

> Note: Safari extension support requires macOS 14+ and Safari 18.5+.

---

## 3. Verifying the Installation

After installing uBO Scope, verify it is correctly added, enabled, and ready to use:

1. Ensure the extension icon is visible next to your browser's address bar.
2. Click the icon to open the popup panel; a basic interface should display connection statistics for the active tab.
3. Check that the extension has the necessary permissions (usually requested during installation).

If you want to confirm you installed the authentic extension:

- On **Chrome/Chromium**, review the extension details via `chrome://extensions/` or `edge://extensions/`.
- On **Firefox**, check `about:addons` under Extensions.
- On **Safari**, review installed extensions under Safari Preferences > Extensions.

Verify the author as **Raymond Hill** and the version matches the latest listed on official stores.

---

## 4. Platform-Specific Tips

### Chromium Browsers
- Signed and delivered via Chrome Web Store ensure safe updates.
- If using a Chromium derivative that doesn't support the Web Store, consider manual side-loading from [GitHub Releases](https://github.com/gorhill/uBO-Scope/releases) following developer guidelines.

### Firefox
- The extension requires Firefox 128 or later.
- Firefox mobile supports uBO Scope on Android with version 128+

### Safari
- Requires macOS 14 (Sonoma) or later with Safari 18.5+
- Extension updates are managed via the Mac App Store.
