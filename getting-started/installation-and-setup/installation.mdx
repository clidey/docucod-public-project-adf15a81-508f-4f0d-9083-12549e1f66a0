# Installing uBO Scope

Welcome to the installation guide for uBO Scope! This page provides clear, step-by-step instructions to install uBO Scope on your browser of choice, whether you use Chrome, Firefox, or Safari. Each section is tailored to the specific browser experience, ensuring a smooth and hassle-free installation.

---

## Supported Platforms

uBO Scope supports the following browsers with their minimum required versions:

- **Google Chrome**: Version 122.0 or higher
- **Mozilla Firefox**: Version 128.0 or higher
- **Safari**: Version 18.5 or higher

> Ensure your browser is updated to meet these minimum version requirements for full compatibility.

---

## Installing uBO Scope for Chrome

### Steps to Install from Chrome Web Store

1. **Open Google Chrome** on your computer.
2. Navigate to the official [uBO Scope page on the Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the **Add to Chrome** button.
4. In the confirmation dialog, select **Add extension**.

### Verification

- After installation, the uBO Scope icon (a shield symbol) will appear in the browser toolbar.
- Click the icon to open the popup UI and confirm it loads without errors.

<Tip>
If the extension icon does not appear, check the Chrome extensions menu (puzzle piece icon), and pin uBO Scope for easy access.
</Tip>

---

## Installing uBO Scope for Firefox

### Steps to Install from Firefox Add-ons

1. **Open Mozilla Firefox** on your device.
2. Go to the official [uBO Scope page on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click the **Add to Firefox** button.
4. Select **Allow** in the permission popup, then **Add**.

### Verification

- Look for the uBO Scope icon in the Firefox toolbar.
- Click it to verify the popup interface is accessible.

<Note>
Firefox on Android also supports uBO Scope, provided the browser version is 128.0 or higher.
</Note>

---

## Installing uBO Scope for Safari

### Steps to Install from the Safari Extensions Gallery

1. **Open Safari** on macOS with version 18.5 or higher.
2. Open **Safari Extensions** by navigating to the menu: Safari > Settings (or Preferences) > Extensions.
3. Search for **uBO Scope** in the Extensions Gallery.
4. Click **Install** to add it to Safari.
5. After installation, enable uBO Scope in the Extensions tab.

### Verification

- Confirm the uBO Scope icon appears in the Safari toolbar.
- Click the icon to launch the popup interface.

<Warning>
Safari requires explicit enabling of extensions after installation. Make sure uBO Scope is enabled to work properly.
</Warning>

## Common Installation Tips

- **Restart Browser**: After installation, restart your browser if the extension does not appear immediately.
- **Toolbar Icon Visibility**: If uBO Scope is installed but the icon is hidden, use your browser’s toolbar customization feature to make it visible.
- **Permissions**: The extension requires permissions to monitor network requests. Ensure you accept these during installation.
