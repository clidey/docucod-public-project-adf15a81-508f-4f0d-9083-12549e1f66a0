# Installing uBO Scope

Welcome to the installation guide for **uBO Scope**, a browser extension that transparently reports your browser’s remote server connections to help you understand network activity.

This guide provides clear, step-by-step instructions to install uBO Scope on **Google Chrome**, **Mozilla Firefox**, and **Apple Safari**. Follow these instructions carefully to ensure a safe, complete installation from trusted sources.

---

## 1. Prerequisites & Browser Compatibility

Before installing, confirm your environment meets the following:

- **Supported Browsers and Minimum Versions:**
  - Google Chrome: version **122.0** or above
  - Mozilla Firefox: version **128.0** or above
  - Apple Safari: version **18.5** or above

- **Permissions Required:** uBO Scope requires these browser permissions:
  - `activeTab`: to access the current tab information
  - `storage`: for saving configuration and state locally
  - `webRequest`: to monitor network requests

- **Installation Source:** Always install from the official browser extension stores to avoid tampering and guarantee authenticity.

<Tip>
Check your browser version by visiting your browser’s "About" page before proceeding.
</Tip>

---

## 2. Installing on Google Chrome

Google Chrome requires version 122.0 or higher to support uBO Scope fully.

### Step 1: Navigate to the Chrome Web Store

- Open Chrome and go to the [Chrome Web Store uBO Scope page](https://chrome.google.com/webstore).
- Search for "uBO Scope" or use the direct link if provided.

### Step 2: Add the Extension

- Click on **Add to Chrome**.
- When prompted, review the permissions and click **Add extension**.

### Step 3: Verify Installation

- After installation, the uBO Scope icon appears in Chrome's toolbar.
- Click the icon to confirm the popup window opens.

<Tip>
If you don’t see the icon, click the puzzle piece icon to pin uBO Scope to the toolbar.
</Tip>

---

## 3. Installing on Mozilla Firefox

Firefox users require version 128.0 or above.

### Step 1: Visit the Firefox Add-ons Site

- Open Firefox and go to the [Mozilla Add-ons uBO Scope page](https://addons.mozilla.org).

### Step 2: Install the Extension

- Click **Add to Firefox**.
- Accept the permissions to complete installation.

### Step 3: Confirm Successful Installation

- The uBO Scope icon appears next to the address bar.
- Click this icon to open the extension’s popup interface.

### Optional: Manual Installation

If you need to install manually (e.g., for development or testing):

1. Download the `.xpi` package via the build script or the repository.
2. Open Firefox menus → Add-ons and Themes → Gear icon → Install Add-on From File.
3. Select the downloaded `.xpi` and confirm.

<Tip>
Use the provided `tools/make-firefox.sh` script in the repository root (Linux environment) to generate the installation `.xpi` file if needed.
</Tip>

---

## 4. Installing on Apple Safari

Safari requires version 18.5 or newer.

### Step 1: Access the Safari Extensions Gallery

- Open Safari and navigate to the [Safari Extensions](https://apps.apple.com).

### Step 2: Search and Install

- Search for "uBO Scope".
- Click **Get** and follow the prompts to install.

### Step 3: Enable the Extension

- Go to Safari Preferences → Extensions tab.
- Check the box to enable uBO Scope.

### Optional: Manual Packaging

For manual packaging or distribution:

1. Use the `tools/make-safari.sh` script to build a ZIP package.
2. Install via Safari’s extension developer tools.

<Tip>
Ensure you have the correct signing and entitlements configured for manual Safari extension installation.
</Tip>

---

## 5. Post-Installation Validation

After installing uBO Scope, verify it is properly active:

1. Look for the uBO Scope icon in the browser toolbar.
2. Click the icon to open the popup interface (based on `popup.html`), which should display connection data when browsing.
3. Visit any active tab and watch the ‘domains connected’ count update dynamically.

<Tip>
If the popup shows “NO DATA” initially, load or reload a website in your browser tab to generate network activity.
</Tip>

## 6. Common Installation Issues & Troubleshooting Tips

- **Extension icon missing:**
  - Pin the extension via your browser’s extension menu.
  - Restart your browser.

- **Popup interface not loading:**
  - Verify permissions were accepted.
  - Ensure your browser version meets the minimum requirement.

- **Manual install scripts failing:**
  - Confirm you have a Linux environment for `make-firefox.sh` and `make-safari.sh`.
  - Install `jq` on your system for JSON version updates.

- **Browser restrictions on corporate/managed devices:**
  - Consult your system administrator to ensure extension installation policies allow uBO Scope.
  
<Tip>
For detailed troubleshooting, see the [Troubleshooting Installation & Setup guide](/getting-started/installation-and-setup/troubleshooting).
</Tip>
