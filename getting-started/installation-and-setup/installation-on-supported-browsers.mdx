# Installation on Supported Browsers

This guide provides clear, step-by-step instructions to install uBO Scope via official browser extension stores on Chromium-based browsers, Mozilla Firefox, and Apple Safari. Each section includes direct links, visual cues, and tips to verify that installation has been completed successfully.

---

## 1. Supported Browsers & Version Requirements

Before installation, ensure your browser meets the minimum compatibility requirements:

- **Chromium-based browsers (Chrome, Edge, Brave, etc.)**: Version 122.0 or newer
- **Mozilla Firefox**: Version 128.0 or newer
- **Apple Safari**: Version 18.5 or newer

<u>Note:</u> uBO Scope requires browsers that support Manifest Version 3 extensions and the `webRequest` API.

---

## 2. Installing uBO Scope

### 2.1 Chromium-based Browsers (Chrome, Edge, Brave)

1. Open your Chromium-based browser.
2. Navigate to the official uBO Scope listing in the Chrome Web Store:

   [Install uBO Scope from Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click the **Add to Chrome** button.
4. Confirm any permission prompts by clicking **Add extension**.
5. Upon successful installation, the uBO Scope icon (a badge with distinct colors) will appear in your browser toolbar.

   ![uBO Scope Chromium Installation Screenshot](https://github.com/user-attachments/assets/d5033882-0c94-424f-9e8b-e00ed832acf7)

### 2.2 Mozilla Firefox

1. Open Firefox browser.
2. Navigate to the official uBO Scope page on the Firefox Add-ons site:

   [Install uBO Scope from Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click the **Add to Firefox** button.
4. Allow permissions when prompted by clicking **Add**.
5. The uBO Scope icon will now be visible in the toolbar.

   ![uBO Scope Firefox Installation Screenshot](https://github.com/user-attachments/assets/8a33b8ba-57ee-4a54-a83c-7d21f9b2dafb)

### 2.3 Apple Safari

1. Open Safari browser.
2. Navigate to the Mac App Store or Safari Extensions Gallery and search for "uBO Scope".
3. Click **Get** or **Install** to add the extension.
4. Once installed, open Safari Preferences (⌘ + ,), go to the **Extensions** tab, and enable uBO Scope.

<u>Note:</u> uBO Scope requires Safari 18.5 or later due to Manifest V3 support.

---

## 3. Verifying Successful Installation

After installation on any supported browser, use the following checklist to confirm uBO Scope is properly set up:

- The uBO Scope icon appears in the browser toolbar.
- Click the icon to open the popup panel.
- The popup initially displays the domains connected for the current active tab.
- The number badge on the icon counts distinct third-party remote servers connected.

<Tip>
If the uBO Scope icon does not appear immediately after installation, try restarting your browser or checking the browser's extension manager to ensure the extension is enabled.
</Tip>

---

## 4. Tips for a Smooth Installation Experience

- **Permissions:** When installing, you will be prompted to allow network request monitoring permissions. These are essential for uBO Scope’s functionality.
- **Updates:** The extension will update automatically through the browser’s extension update system. No manual updates needed.
- **Incognito/Private Mode:** To enable uBO Scope in private browsing windows, manually enable this option in your browser’s extension settings.

<Warning>
Do not install unofficial or third-party distributions of uBO Scope. Always use official browser extension stores to ensure security and integrity.
</Warning>

---

## 5. Troubleshooting Installation Issues

<AccordionGroup title="Common Installation Issues and Solutions">
<Accordion title="uBO Scope Icon Not Visible After Installation">
1. Check your browser’s extension management page and confirm uBO Scope is enabled.
2. Restart the browser to refresh toolbar icons.
3. For Chromium-based browsers, right-click the extensions icon and pin uBO Scope to the toolbar if hidden.
</Accordion>
<Accordion title="Permissions Prompt Not Appearing or Denied">
1. Reinstall the extension to trigger permission dialogs again.
2. Verify your browser’s settings do not restrict extension permissions.
3. For corporate or managed environments, check with your IT administrator if policy restrictions apply.
</Accordion>
<Accordion title="Installation Fails or Extension Isn't Listed in the Store">
1. Confirm your browser version meets minimum requirements.
2. Clear browser cache and retry.
3. Visit the official GitHub repository for the latest release information or known issues: https://github.com/gorhill/uBO-Scope
</Accordion>
</AccordionGroup>
