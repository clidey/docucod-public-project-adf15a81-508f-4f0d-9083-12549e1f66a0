# Installing on Chrome, Firefox, and Safari

This guide provides clear, step-by-step instructions to install the **uBO Scope** browser extension on your preferred browser: Chrome, Firefox, or Safari. Whether you want to install directly from the official extension stores or use a manual method for advanced scenarios, these instructions will get you started quickly.

---

## Supported Browsers and Minimum Requirements

- **Google Chrome**: Version 122 or later.
- **Mozilla Firefox**: Version 128 or later.
- **Safari**: Version 18.5 or later.

Ensure your browser is updated to meet these requirements for full compatibility with uBO Scope.

---

## Installation Overview

uBO Scope is a lightweight browser extension that reveals third-party connections made by webpages to remote servers. Installing it on your browser allows real-time monitoring and detailed network insight. This guide covers:

- Installing from the official extension stores
- Manual installation for advanced users

---

## 1. Installing uBO Scope from Official Stores

### Chrome Web Store

1. Open Chrome and visit the official uBO Scope page:  
   [Chrome Web Store - uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)
2. Click **Add to Chrome**.
3. Confirm the permissions prompt.
4. Once installed, you will see the uBO Scope icon in your browser toolbar.

### Firefox Add-ons

1. Open Firefox and navigate to:  
   [Firefox Add-ons - uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/)
2. Click **Add to Firefox**.
3. Confirm the permissions and installation.
4. The uBO Scope icon will appear alongside your installed extensions.

### Safari Extensions Gallery (Mac App Store)

1. Open the App Store on your Mac.
2. Search for **uBO Scope** or browse Safari Extensions.
3. Follow the prompts to install the extension.
4. Enable uBO Scope in Safari's preferences under the Extensions tab.

---

## 2. Manual Installation for Advanced Users

If you want to install a custom build version or are developing uBO Scope, manual installation allows loading the extension unpacked.

### Chrome (Developer Mode)

1. Download the source or build package for Chrome.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked**.
5. Select the directory containing the unpacked uBO Scope extension files.
6. The extension will load, and you can test your build immediately.

### Firefox (Temporary Add-on)

1. Download the source or build package for Firefox.
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
3. Click **Load Temporary Add-on**.
4. Select the `manifest.json` file inside the unpacked extension directory.
5. The extension will appear in your add-ons list for this session.

### Safari (Developer Mode)

1. Download the source or build package for Safari.
2. Open Safari and navigate to **Preferences > Extensions**.
3. Enable the **Develop** menu in Safari's **Advanced** preferences.
4. From the Develop menu, choose **Show Extension Builder**.
5. Add your unpacked Safari extension directory.
6. Build and run the extension directly in Safari.

<Note>
Manual installation methods are best suited for developers, testers, or users who require the bleeding-edge or customized version of uBO Scope. For standard users, installing from the official stores is recommended for automatic updates and security.
</Note>

---

## 3. Post-Installation Steps

1. After installation, locate the uBO Scope toolbar icon in your browser.
2. Click the icon to open the extension popup, which will start displaying data about remote connections on the active tab.
3. To learn how to configure and run your first operation, proceed to the [Initial Configuration](https://documentation-url/getting-started/installation-and-setup/initial-configuration) guide.

<Tip>
Make sure to grant any requested permissions such as access to active tabs and web requests to allow uBO Scope to function properly.
</Tip>
