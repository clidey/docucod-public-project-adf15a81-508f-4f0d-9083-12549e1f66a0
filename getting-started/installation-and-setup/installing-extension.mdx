# Installing the Extension

Installing uBO Scope is a straightforward process tailored for your preferred browser. Choose from Chromium-based browsers (like Google Chrome), Mozilla Firefox, or Safari, and follow the step-by-step instructions below to get uBO Scope up and running from official and trusted marketplaces. This ensures a reliable and secure installation experience.

---

## Supported Browsers

- **Chromium-based browsers** (Google Chrome, Microsoft Edge, Brave, etc.) — Minimum version: 122.0
- **Mozilla Firefox** — Minimum version: 128.0
- **Safari** — Minimum version: 18.5

<u>These version requirements guarantee full compatibility with uBO Scope’s core features and permissions.</u>

---

## Installation Steps by Browser

### 1. Installing on Chromium-based Browsers

Follow these steps to install uBO Scope from the Chrome Web Store:

<Steps>
<Step title="Open the Chrome Web Store">
Navigate to the official uBO Scope page in the Chrome Web Store:

[https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

</Step>
<Step title="Add to Browser">
Click **Add to Chrome** button.

- Confirm any permissions requested (such as access to active tabs and storage).
- Wait for the extension to download and install.

</Step>
<Step title="Verify Installation">
Look for the uBO Scope icon in the browser toolbar.

- The icon will display a badge count once you start browsing.
- If the icon is hidden, click the puzzle piece icon in the toolbar and pin uBO Scope for quick access.

</Step>
</Steps>

<Note>
<u>Tip:</u> Make sure your browser is updated to version 122 or later to avoid compatibility issues.
</Note>

---

### 2. Installing on Mozilla Firefox

Install uBO Scope via the official Mozilla Add-ons site:

<Steps>
<Step title="Open Firefox Add-ons">
Visit the uBO Scope Firefox Add-ons page:

[https://addons.mozilla.org/firefox/addon/ubo-scope/](https://addons.mozilla.org/firefox/addon/ubo-scope/)

</Step>
<Step title="Add to Firefox">
Click **Add to Firefox**.

- Review permissions, such as webRequest access.
- Confirm installation when prompted.

</Step>
<Step title="Check the Toolbar Icon">
Confirm the uBO Scope icon appears in the toolbar.

- If missing, enable it from the Customize Toolbar menu.

</Step>
</Steps>

<Warning>
<u>Minimum Firefox version 128.0 is required for uBO Scope to function properly.</u>
</Warning>

---

### 3. Installing on Safari

To install uBO Scope on Safari, follow these steps:

<Steps>
<Step title="Access Safari Extensions">
Open the Safari Extensions gallery or the App Store on your Mac.

- Search for “uBO Scope.”

</Step>
<Step title="Download and Install">
Download and install the extension.

- Safari requires version 18.5 or higher for this extension.

</Step>
<Step title="Enable the Extension">
In Safari, go to Preferences > Extensions.

- Enable uBO Scope.
- Review requested permissions.

</Step>
</Steps>

<Note>
Safari’s permission model may restrict available functionality; consult system version compatibility if you encounter issues.
</Note>

## After Installation

Once installed, uBO Scope monitors network connections from webpages and updates the toolbar badge accordingly.

- The badge count indicates the number of distinct third-party remote servers contacted.
- Click the toolbar icon to open the popup interface showing detailed domain-level connection data.

For information on your first run and configuring uBO Scope, see [First Run & Minimal Configuration](/getting-started/installation-and-setup/first-run-configuration).

---

## Security and Privacy Considerations

- uBO Scope requests only essential permissions: access to active tabs, storage, and network requests.
- No personal data is collected or transmitted by the extension.
- Always install from official marketplaces to avoid tampered versions.

---

## Troubleshooting Installation Issues

If you run into problems during installation:

- Verify your browser meets the version requirements.
- Restart your browser after installation to ensure proper initialization.
- Disable conflicting extensions that might interfere with uBO Scope.

For detailed help, refer to [Troubleshooting Common Installation Issues](/getting-started/installation-and-setup/troubleshooting-installation).
