# Installing uBO Scope

Welcome to the installation guide for **uBO Scope**, a lightweight browser extension designed to reveal all third-party remote server connections initiated during your browsing sessions. This guide provides clear, step-by-step instructions for installing uBO Scope from the official browser extension stores for **Google Chrome** and **Mozilla Firefox**. Follow these instructions to quickly add the extension, ensure it’s set up correctly, and manage it for optimal use.

---

## 1. Prerequisites & Compatibility

Before proceeding, verify the following to guarantee a smooth installation:

- **Supported Browsers:**
  - Google Chrome (version 122.0 or later)
  - Mozilla Firefox (version 128.0 or later)

- **Operating Systems:** Any OS supporting the above browsers, including Windows, macOS, and Linux.

- **Permissions Required:**
  - uBO Scope requests browser permissions for `webRequest`, `storage`, and `activeTab` to monitor network requests and save extension data.

- **Internet Connectivity:** Required for downloading the extension from the stores.

<Tip>
Ensure your browser is updated to the minimum required version for compatibility and security.
</Tip>

---

## 2. Installation Instructions

### 2.1 Installing uBO Scope on Google Chrome

1. Open Google Chrome.
2. Navigate directly to the uBO Scope extension page on the Chrome Web Store via this link:

   [Install uBO Scope for Chrome](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

3. Click the **Add to Chrome** button.
4. In the permission prompt, review the permissions and click **Add extension** to confirm.
5. Once the installation completes, you will see the uBO Scope icon appear in your browser toolbar.

### 2.2 Installing uBO Scope on Mozilla Firefox

1. Open Mozilla Firefox.
2. Navigate directly to the uBO Scope add-on page on the Firefox Add-ons store via this link:

   [Install uBO Scope for Firefox](https://addons.mozilla.org/firefox/addon/ubo-scope/)

3. Click the **Add to Firefox** button.
4. Review the requested permissions and click **Add**.
5. You may need to confirm by clicking **Okay, Got it** in a follow-up prompt.
6. After installation, the uBO Scope icon will appear in your Firefox toolbar.

<Note>
For both browsers, if you do not see the icon immediately, check your browser’s extension menu or toolbar customization settings to enable it.
</Note>

## 3. Post-Installation Quick Reference

### 3.1 Updating uBO Scope

- **Automatic Updates:** Both Chrome Web Store and Firefox Add-ons automatically update extensions. Keep your browser updated to receive the latest version of uBO Scope.
- **Manual Update Check:**
  - In Chrome, visit `chrome://extensions/`, enable **Developer mode**, and click **Update** to force extension updates.
  - In Firefox, go to `about:addons`, click the gear icon, and select **Check for Updates**.

### 3.2 Uninstalling uBO Scope

- **In Chrome:**
  1. Open `chrome://extensions/`.
  2. Find uBO Scope.
  3. Click **Remove** and confirm.

- **In Firefox:**
  1. Open `about:addons`.
  2. Locate uBO Scope.
  3. Click the three dots menu next to it and select **Remove**.

<Warning>
Uninstalling uBO Scope will remove all stored session data and reset any related settings.
</Warning>

## 4. Verification After Installation

1. Look for the uBO Scope icon in your browser toolbar (a simple shield icon).
2. Click the icon to open the popup interface.
3. Visit any website and observe the badge count on the toolbar icon; this indicates the number of distinct third-party servers contacted.
4. Use the popup to review connection details:
   - **Not Blocked:** Domains from allowed network requests.
   - **Stealth-Blocked:** Domains that were redirected or otherwise processed stealthily.
   - **Blocked:** Domains for which requests failed or were actively blocked.

<Tip>
If the badge shows no count on browsing sites where third-party content is expected, verify that the extension has permission and is enabled.
</Tip>

## 5. Troubleshooting Common Issues

### Issue: Extension Icon Not Visible

- Check your browser's extensions menu and pin the icon to your toolbar.
- Restart the browser.

### Issue: Badge Count Not Updating

- Ensure you are on a supported browser version.
- Disable conflicting content blockers temporarily to test.
- Refresh the tab and verify permissions.

### Issue: Unable to Install from Store

- Verify internet connectivity.
- Ensure your browser version meets minimum requirements.
- Clear browser cache or restart browser.
