# Installation Steps for uBO Scope

Follow this comprehensive, step-by-step guide to install uBO Scope on your preferred browser. uBO Scope supports the leading browsers—Chrome, Firefox, and Safari—with dedicated installation methods to ensure smooth and reliable setup.

---

## 1. Prerequisites & System Requirements

Before installing, verify your system meets the following minimum requirements to guarantee full compatibility:

- **Supported Browsers:**
  - **Google Chrome:** version 122.0 or higher
  - **Mozilla Firefox:** version 128.0 or higher
  - **Apple Safari:** version 18.5 or higher

- **Operating Systems:**
  - Windows, macOS, Linux (any OS supporting the above browsers)

- **Permissions Required:**
  - The extension requires permissions for `activeTab`, `storage`, and `webRequest` to monitor network connections effectively.

- **Network Requirements:**
  - Internet connection to access extension stores

> Make sure your browser version meets or exceeds these thresholds to prevent compatibility issues.

---

## 2. Installation Instructions

Install uBO Scope using the browser-specific extension stores or manual methods where applicable.

### A. Install on Google Chrome

1. Open the Chrome browser.
2. Navigate to the [uBO Scope page on the Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the **Add to Chrome** button.
4. Confirm by clicking **Add extension** in the prompt.
5. Wait for the installation to complete. You will see the uBO Scope icon in the toolbar.

### B. Install on Mozilla Firefox

1. Open Firefox.
2. Visit the [uBO Scope add-ons page on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click **Add to Firefox**.
4. Confirm any permission dialogs by selecting **Add**.
5. Once installed, the toolbar icon will appear.

### C. Install on Apple Safari

1. Open Safari.
2. Open the Safari Extensions Gallery or the App Store.
3. Search for **uBO Scope**.
4. Select the extension and click **Install**.
5. Ensure Safari is version 18.5 or later to support the extension.

> Safari requires macOS with a compatible Safari version. For detailed compatibility, check your Safari’s About page.

---

## 3. Verifying Successful Installation

Once installed, confirm the extension is working properly by following these steps:

1. Look for the uBO Scope icon in your browser’s toolbar.
2. Click the icon to open the popup panel.
3. The popup should display **no data** initially until you browse websites.
4. Navigate to any website.
5. Observe the badge icon number update, representing the count of distinct third-party remote server connections detected.

If the icon or popup does not appear:

- Verify browser permissions for extensions
- Ensure you have installed the extension from the official store
- Refresh or restart the browser

---

## 4. Troubleshooting Common Installation Issues

<Tip>
If the extension fails to install or is missing features:

- **Check browser version:** Update to the latest version.
- **Confirm network connectivity:** Connectivity issues can affect installation.
- **Verify extension permissions:** Sometimes permission denial blocks installation.
- **Disable conflicting extensions:** Other content blockers might interfere.

If problems persist, consult the Troubleshooting section in the documentation.
</Tip>
