# Installing uBO Scope

Welcome to the installation guide for uBO Scope! This step-by-step document will walk you through installing the extension on your preferred browser—Chromium-based browsers, Firefox, or Safari. uBO Scope is a privacy-focused browser extension that transparently reports remote server connections made by your browser.

---

## Supported Browsers and Versions

Before proceeding, ensure your browser meets the minimum version requirements for uBO Scope:

- **Chromium-based browsers (Google Chrome, Edge, Brave, etc.)**: Version 122.0 or later
- **Mozilla Firefox**: Version 128.0 or later
- **Safari**: Version 18.5 or later

If you are running an earlier version, please update your browser to access the latest features and stability improvements.

---

## Installation Steps

### 1. Installing on Chromium-Based Browsers

Follow these steps to install uBO Scope from the Chrome Web Store:

<Steps>
<Step title="Open the Chrome Web Store">Navigate to the official uBO Scope extension page on the Chrome Web Store: [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)</Step>
<Step title="Click Add to Chrome">Click the 'Add to Chrome' button and confirm the installation in the popup dialog.</Step>
<Step title="Verify Installation">Once installed, you will see the uBO Scope icon (a red shield logo) appear in your browser's toolbar.</Step>
</Steps>

### 2. Installing on Mozilla Firefox

To install uBO Scope on Firefox, use the official Mozilla Add-ons store:

<Steps>
<Step title="Open Firefox Add-ons site">Go to the uBO Scope page on Mozilla Add-ons: [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)</Step>
<Step title="Click Add to Firefox">Click 'Add to Firefox' and grant the required permissions when prompted.</Step>
<Step title="Confirm Installation">You will see the uBO Scope icon in the Firefox toolbar once installation completes.</Step>
</Steps>

#### Visual Cue:

![Get uBlock Origin Lite for Firefox](https://github.com/user-attachments/assets/8a33b8ba-57ee-4a54-a83c-7d21f9b2dafb)

### 3. Installing on Safari

For Safari users, uBO Scope requires at least Safari version 18.5. Installation requires manual enabling in Safari Extensions preferences:

<Steps>
<Step title="Download uBO Scope for Safari">Download the uBO Scope Safari extension package from the official repository or trusted source.</Step>
<Step title="Open Safari Preferences">Launch Safari and navigate to `Safari > Settings` (or `Preferences`) > Extensions.</Step>
<Step title="Enable uBO Scope">Find uBO Scope in the list and check the box to enable it.</Step>
</Steps>

<Note>
Safari’s extension ecosystem differs from Chromium and Firefox. uBO Scope may require manual installation or download from Apple’s Extension Gallery if available in the future.
</Note>

## Post-Installation

After installing the extension, you will see the uBO Scope icon in your browser’s toolbar, typically as a shield with a badge counter. This badge shows the number of distinct remote third-party domains your browser connected to from the current tab.

<uBoScopePopupInterface /> will display detailed connection results — including which domains were allowed, stealth-blocked, or blocked — when you click the toolbar icon.

---

## Permissions Requested

Upon installation, uBO Scope requests the following browser permissions essential for its operation:

- **activeTab**: To monitor the currently focused tab for network connections.
- **storage**: To save session and configuration data.
- **webRequest**: To observe network requests and their outcomes in real-time.
- **host permissions**: Access to HTTP(s) and WebSocket URLs (`https://*/*`, `http://*/*`, `wss://*/*`, `ws://*/*`) to report all remote connections.

<Tip>
Granting these permissions is necessary for accurate and comprehensive connection reporting.
</Tip>

---

## Verification of Installation

To confirm that uBO Scope is properly installed and active:

1. Open a new browser tab and navigate to a website.
2. Click the uBO Scope toolbar icon.
3. In the popup, you should see a summary count of connected domains and categorized domain lists.

If you see "NO DATA" or no count badge, reload the tab or verify permissions.

## Troubleshooting Common Installation Issues

<AccordionGroup title="Installation Troubleshooting">
<Accordion title="uBO Scope icon does not appear after installation">
- Ensure your browser version meets the minimum requirements.
- Restart your browser after installation.
- Check the browser’s extensions or add-on management page to confirm uBO Scope is enabled.
</Accordion>
<Accordion title="Extension permissions are denied or missing">
- Extensions cannot function without granted permissions. Check your browser’s permission dialogs.
- Reinstall or update uBO Scope to prompt permissions again.
</Accordion>
<Accordion title="Badge count does not update">
- Reload the active tab to refresh network activity.
- Ensure the browser supports `webRequest` API fully.
</Accordion>
</AccordionGroup>
