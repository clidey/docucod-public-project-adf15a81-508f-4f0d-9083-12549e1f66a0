# Installing uBO Scope

Get started by installing uBO Scope on your preferred browser. This guide provides clear, actionable steps to download, install, and enable the uBO Scope extension on Chrome, Firefox, and Safari, helping you achieve immediate visibility into your network connections.

---

## 1. Supported Browsers and Versions

uBO Scope is compatible with the following browsers:

- **Google Chrome:** Version 122.0 and above
- **Mozilla Firefox:** Version 128.0 and above
- **Safari:** Version 18.5 and above

Ensure your browser is up-to-date to avoid compatibility issues.

<Note>
The extension relies on the browser's `webRequest` API, so successful installation and operation depend on your browser supporting this API fully.
</Note>

---

## 2. Installation Steps by Browser

### Google Chrome

1. Open the [Chrome Web Store page for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome**.
3. In the confirmation dialog, click **Add extension**.
4. Wait for the installation to complete. You will see the uBO Scope icon in your browser toolbar.

### Mozilla Firefox

1. Visit the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. Confirm by clicking **Add**.
4. The extension installs and you will see the uBO Scope icon on your toolbar.

### Safari

1. Open the **App Store** on your Mac.
2. Search for **uBO Scope**.
3. Click **Get** and then **Install**.
4. Once installed, enable the extension:
   - Open Safari Preferences (`Safari` > `Settings` or `Preferences`).
   - Go to the **Extensions** tab.
   - Check the box next to **uBO Scope** to enable it.

<Warning>
Safari requires manually enabling extensions after installation to activate functionality.
</Warning>

---

## 3. Understanding Required Permissions

uBO Scope requests the following permissions:

- **webRequest:** To capture and monitor network requests initiated by webpages.
- **activeTab:** To access data about the currently active tab for real-time reporting.
- **storage:** To save session data and maintain state between browser sessions.

<Info>
These permissions are essential for uBO Scope to provide accurate, tab-specific network visibility.
</Info>

<Note>
The extension requests access to URLs covering `http://*/*`, `https://*/*`, and websocket protocols where applicable.
</Note>

---

## 4. Post-Installation Checklist

Follow these steps immediately after installation to verify correct setup:

1. Look for the uBO Scope icon in your browser's toolbar.
2. Visit any popular website (e.g., https://example.com).
3. Click the uBO Scope icon to open its popup and observe detected connections.

<Check>
If you see a list of connected domains, the extension is installed and functioning correctly.
</Check>

<Warning>
If the icon does not appear or shows no data after browsing a site, refer to the "Troubleshooting Common Issues" guide.
</Warning>

---

## 5. Tips and Best Practices

- **Pin the extension icon** to your toolbar for quick access.
- **Keep your browser updated** to maintain compatibility with the latest API features.
- **Review permissions** occasionally in your browser's extension management page.
- **Use alongside other content blockers** without conflicts: uBO Scope works independently and transparently via the `webRequest` API.

---

## 6. Ready to Start Monitoring?

Once installed, you can:

- Click the toolbar icon any time to see real-time connection summaries for the active tab.
- Understand which third-party domains are actively contacted by visited web pages.
- Begin auditing network requests to improve your privacy and security awareness.
