# Installation Instructions for uBO Scope

Get ready to unlock clear, real-time visibility into your browser's network connections by installing uBO Scope. This guide walks you through precise, step-by-step instructions to install uBO Scope from your browser’s official extension or add-ons store for Chrome, Firefox, and Safari (if applicable). We provide visual cues and verification tips so you can confidently start using the extension.

---

## 1. Supported Browsers and Requirements

Before installation, ensure your browser meets the minimum version requirements for smooth operation:

- **Google Chrome**: Version 122 or later
- **Mozilla Firefox**: Version 128 or later
- **Apple Safari**: Version 18.5 or later (if available)

<u>Note:</u> uBO Scope requires permissions for `webRequest`, `activeTab`, and `storage` to accurately monitor network requests.

---

## 2. Installing uBO Scope

### Option 1: Install from Chrome Web Store (for Chromium-based Browsers)

1. Open Google Chrome or any Chromium-based browser (e.g., Edge, Brave).
2. Navigate to the [uBO Scope Chrome Web Store page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the **Add to Chrome** button.
4. In the confirmation dialog, click **Add extension**.
5. Wait briefly as the extension downloads and installs.

### Option 2: Install from Firefox Add-ons Site

1. Open Mozilla Firefox.
2. Visit the [uBO Scope Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click **Add to Firefox**.
4. When prompted, click **Add** to grant required permissions.
5. Allow the extension to install.

### Option 3: Install from Safari Extensions Gallery

_If uBO Scope is available for Safari:_

1. Open Safari browser.
2. Go to Safari > Settings (or Preferences) > Extensions.
3. Use the Safari Extensions Gallery link or the App Store to search for **uBO Scope**.
4. Install the extension following the on-screen instructions.

---

## 3. Verifying Installation Success

After installation, confirm uBO Scope is active and ready:

- Look for the **uBO Scope icon** (a shield with uBO branding) in your browser’s toolbar near the address bar.
- Click the icon to open the popup. It should show a default message or rapidly update once you browse web pages.
- On Chrome/Chromium browsers, ensure the badge number on the icon reflects distinct third-party connections as you navigate websites (starting from zero).
- In Firefox, check the Add-ons menu under **Extensions** to confirm uBO Scope is enabled.
- For Safari, ensure the extension is enabled in the Extensions settings and visible in your toolbar.

---

## 4. Post-Installation Tips

- If the icon or badge does not appear, try restarting your browser.
- Ensure you have granted the extension appropriate permissions when prompted during installation.
- uBO Scope requires network access permissions (`webRequest`) which are essential for its functionality.
- For detailed verification of correct operation, see the guide on [Validating That uBO Scope Is Working](/getting-started/first-run-validation/validating-functionality).

---

## 5. Troubleshooting Common Installation Issues

<AccordionGroup title="Common Installation Issues and Solutions">
<Accordion title="uBO Scope Icon Does Not Appear">
- Confirm the extension is enabled in your browser’s extensions or add-ons page.
- Restart the browser completely.
- Check that your browser version meets the minimal requirements.
- In Chrome, verify that the extension is not hidden behind the puzzle piece icon. Pin it to the toolbar if needed.
</Accordion>

<Accordion title="Installation Fails or Permissions Are Denied">
- Ensure you are installing from an official browser store (Chrome Web Store, Firefox Add-ons, Safari Extensions).
- Refresh the store page and retry installing.
- If using corporate or managed devices, verify browser policies allow installing extensions.
- Check for antivirus or security software that may block extension installs.
</Accordion>

<Accordion title="Badge Count Does Not Update">
- Some sites may not have third-party connections immediately; try navigating to popular sites to test.
- Check for conflicting extensions that might interfere with `webRequest` API.
- Review the permissions you've granted to uBO Scope; all are required for accurate monitoring.
</Accordion>
</AccordionGroup>
