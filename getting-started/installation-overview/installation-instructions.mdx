# Installing uBO Scope

This guide walks you through the straightforward process of installing the uBO Scope browser extension on Chrome, Firefox, and Safari. uBO Scope offers transparent visibility into the network connections a webpage makes, helping you better understand and enhance your online privacy.

---

## 1. Overview

Each supported browser uses a different installation source and has specific minimum version requirements. This guide provides direct download links, clear instructions, and tailored tips for each browser to ensure a smooth installation experience.


## 2. Prerequisites

Before proceeding, make sure your system meets these basic requirements:

- **Supported Browsers and Minimum Versions:**
  - **Chromium-based (Chrome, Edge, Brave, etc.)**: Version 122 or newer
  - **Firefox**: Version 128 or newer
  - **Safari**: Version 18.5 or newer
- **Permissions Required:**
  - Network request monitoring capabilities through `webRequest` permission
  - Access to active tabs
  - Storage permission for local data retention

If your browser does not meet these minimum versions, please update to the latest stable release before continuing.

---

## 3. Installing on Chromium-based Browsers (Chrome, Edge, Brave)

### Step 1: Open the Chrome Web Store

- Navigate to the official uBO Scope page on the [Chrome Web Store](https://chrome.google.com/webstore/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).

### Step 2: Add uBO Scope to your Browser

- Click the **Add to Chrome** button.
- Confirm the installation by clicking **Add extension** in the popup dialog.

### Step 3: Confirm Successful Installation

- Watch for the uBO Scope icon to appear in your browser toolbar.
- Hover over the icon to see its tooltip ‘uBO Scope’.

<Callout type="info">
If you do not see the icon, check your browser’s extension menu or pin uBO Scope from the extensions dropdown.
</Callout>

---

## 4. Installing on Firefox

### Step 1: Open Firefox Add-ons Store

- Visit the [uBO Scope page on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).

### Step 2: Install the Extension

- Click **Add to Firefox**.
- Confirm any permission dialogs by clicking **Add**.

### Step 3: Verify Installation

- Look for the uBO Scope icon in the Firefox toolbar.
- You can manage or pin the icon from the Extensions menu to keep it visible.

<Callout type="info">
Firefox supports additional host permissions including websocket connections, enhancing the scope of network visibility.
</Callout>

---

## 5. Installing on Safari

### Step 1: Ensure Safari Version

- Verify your Safari is version 18.5 or later by going to Safari > About Safari.

### Step 2: Obtain uBO Scope

- Download uBO Scope from the official Apple Safari Extensions Gallery or trusted sources publishing the extension packaged for Safari.

### Step 3: Enable the Extension

- Open Safari Preferences...
- Go to the **Extensions** tab.
- Locate uBO Scope and check the box to enable it.

### Step 4: Confirm the Toolbar Icon

- The uBO Scope icon will appear in the Safari toolbar once enabled.

<Warning>
Safari extensions require explicit user enablement; if the icon does not appear, double-check you enabled uBO Scope in Extensions preferences.
</Warning>

---

## 6. Post-Installation Checklist

- **Icon Presence:** Confirm the uBO Scope icon is visible in your browser toolbar.
- **Initial Activation:** Click the icon to open the popup panel, showing ‘NO DATA’ initially until you browse web pages.
- **Permissions Granted:** Ensure the extension has required permissions to monitor network requests.

<Check>
If you do not see network data after opening web pages, verify you granted all permissions and that no other security settings block the extension’s operation.
</Check>

---

## 7. Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting Installation Issues">
<Accordion title="uBO Scope Icon Missing After Install">
- Pin the extension manually from your browser’s extensions menu.
- Restart the browser to refresh the extension state.
</Accordion>

<Accordion title="Browser Version Not Supported">
- Update your browser to the indicated minimum version or later.
- Refer to your browser’s official update mechanism.
</Accordion>

<Accordion title="Extension Fails to Monitor Network Requests">
- Check that all requested permissions (`webRequest`, `activeTab`, `storage`) were granted.
- Verify no other privacy/security extensions interfere.
- Consult the troubleshooting documentation on validating network monitoring.
</Accordion>
</AccordionGroup>
