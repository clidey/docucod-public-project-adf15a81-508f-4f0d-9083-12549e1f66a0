# Installing uBO Scope

This guide provides clear, step-by-step instructions to install uBO Scope safely and effectively from the official browser extension stores. Whether you use Chromium-based browsers or Firefox, follow the instructions below to add uBO Scope and validate its successful installation.

---

## 1. Supported Browsers and Platforms

uBO Scope is available for the following browsers:

- **Chromium-based browsers** (Google Chrome, Microsoft Edge, Brave, etc.)
- **Mozilla Firefox** (Desktop and Firefox for Android)

**Important:** Safari support currently requires a manual package build and is not available via the official App Store.


## 2. Installing uBO Scope

Follow the instructions that correspond to your browser.

### For Chromium-based Browsers

1. Click the following button to open the uBO Scope extension in the Chrome Web Store:

   [![Install uBO Scope on Chromium](https://github.com/user-attachments/assets/d5033882-0c94-424f-9e8b-e00ed832acf7)](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

2. Once on the page, click the **Add to Chrome** button.
3. Confirm the installation when prompted by your browser.
4. Wait for the installation to complete.

### For Mozilla Firefox

1. Click the following button to open uBO Scope on Firefox Add-ons:

   [![Install uBO Scope on Firefox](https://github.com/user-attachments/assets/8a33b8ba-57ee-4a54-a83c-7d21f9b2dafb)](https://addons.mozilla.org/firefox/addon/ubo-scope/)

2. Click the **Add to Firefox** button.
3. Accept the permissions requested.
4. Wait for the extension to install.

---

## 3. Verifying Installation

After installation, verify that uBO Scope is active and ready.

### Check for the Toolbar Icon and Badge

- Look for the uBO Scope icon (a shield) in your browser's toolbar.
- Hovering over the icon should display a tooltip with the name 'uBO Scope'.
- Upon browsing to a webpage, the badge number on the icon updates showing the number of distinct third-party domains connected.

### Access the uBO Scope Popup

1. Click the uBO Scope toolbar icon.
2. A popup interface opens, initially showing connection data.
3. You should see at least the 'domains connected' count.

## 4. Tips for a Smooth Installation

- **Use official extension stores** only for installation to avoid tampered versions.
- Make sure your browser is up to date to ensure compatibility.
- uBO Scope requires permissions to access network requests (`webRequest`) and storage; do not disable these.

<Warning>
If you deny any requested permissions, uBO Scope will not function correctly.
</Warning>

---

## 5. Troubleshooting Common Installation Issues

<AccordionGroup title="Troubleshooting Installation">
<Accordion title="The uBO Scope icon does not appear">
- Confirm the extension is enabled in your browser's extension management.
- For Chrome, try pinning the extension from the extensions menu.
- Restart the browser if necessary.
</Accordion>
<Accordion title="Badge does not update on browsing">
- Verify you are visiting pages that make network requests.
- Check that no other extension or network setting prevents `webRequest` API access.
- Clear browser cache and reload the page.
</Accordion>
<Accordion title="Installation fails or permission errors">
- Ensure you have internet access to reach the extension store.
- Confirm that your browser version meets minimum requirements (Chromium 122+ for Chrome).
- Disable conflicting extensions temporarily and retry.
</Accordion>
</AccordionGroup>
