# Installation Guide for uBO Scope

This guide provides clear, actionable steps to install the uBO Scope extension on supported browsers: Chrome (Chromium-based), Firefox, and Safari. You will find quick links, installation flow, screenshots, and verification steps to get started immediately with minimal effort.

---

## 1. Prerequisites

Before starting the installation, ensure your system meets the following requirements:

- **Supported Browsers:**
  - Chrome 122 or later (Chromium-based)
  - Firefox 128 or later
  - Safari 18.5 or later
- **Permissions:** uBO Scope requires permissions to monitor network requests (`webRequest`), access storage (`storage`), and interact with the active tab (`activeTab`). These are requested automatically during installation.
- **Network Access:** Your browser needs internet connectivity to download the extension and to update the Public Suffix List used internally.

<Tip>
Check your browser version before installation using:
- Chrome: `chrome://settings/help`
- Firefox: `about:support`
- Safari: `Safari > About Safari`

Ensure also that your browser’s extension store is accessible.
</Tip>

---

## 2. Installation Instructions

### 2.1 Installing on Chrome (Chromium-based browsers)

1. Open your Chrome browser.
2. Visit the [uBO Scope page on the Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
3. Click the **Add to Chrome** button.
4. Confirm the permission dialog by clicking **Add extension**.
5. Once installed, the uBO Scope icon appears in your toolbar near the address bar.

![Install on Chrome](https://github.com/user-attachments/assets/d5033882-0c94-424f-9e8b-e00ed832acf7 "Get uBO Lite for Chromium")

### 2.2 Installing on Firefox

1. Open your Firefox browser.
2. Go to the [uBO Scope page on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
3. Click the **Add to Firefox** button.
4. Approve permissions by clicking **Add**.
5. The uBO Scope icon will appear in the toolbar.

![Install on Firefox](https://github.com/user-attachments/assets/8a33b8ba-57ee-4a54-a83c-7d21f9b2dafb "Get uBlock Origin Lite for Firefox")

### 2.3 Installing on Safari

1. Ensure your Safari version is 18.5 or later.
2. Safari extensions are installed from the Mac App Store. Currently, uBO Scope does not have a public Mac App Store listing.
3. For users who build the extension locally, please refer to the developer documentation or follow the build tools provided.

<Warning>
Safari users are advised to verify Safari version compatibility and availability of the extension in the Mac App Store. Offline installation requires technical steps beyond typical user installation.
</Warning>

## 3. Verifying Installation

Once you have installed uBO Scope, verify it's properly running:

1. Locate the uBO Scope icon in your browser toolbar.
2. Click the icon to open the popup interface.
3. The popup should display connection tracking information on your current website with counts of domains connected.
4. The icon badge should show a numeric count indicating distinct third-party remote servers connected.

<Check>
If the icon does not appear:
- Ensure the extension is enabled in your browser’s extension management page.
- Restart the browser and retry.
</Check>

---

## 4. What to Expect After Installation

- The toolbar icon updates live with the number of distinct third-party domains contacted by webpages.
- Clicking the icon reveals detailed domain lists categorized as **not blocked**, **stealth-blocked**, and **blocked**.
- This lets you immediately assess network activity and the presence of third-party servers involved in browsing.

## 5. Troubleshooting Installation Issues

<AccordionGroup title="Common Installation Problems and Their Solutions">
<Accordion title="Extension Icon Not Visible">
- Confirm extension installation via your browser’s extensions page.
- Check extension is enabled and permissions granted.
- Restart your browser to refresh extensions.
</Accordion>
<Accordion title="Unable to Find Extension in Store">
- Verify you are using a supported and updated version of your browser.
- Check your region or network policies that might restrict access to the extension stores.
</Accordion>
<Accordion title="Permission Requests Not Shown">
- Some browsers cache permissions. Try removing and reinstalling the extension.
- Make sure no conflicting extensions are blocking permission dialogs.
</Accordion>
</AccordionGroup>

<Note>
If issues persist, consult the [Troubleshooting Setup Issues](https://your.documentation.url/getting-started/configuration-usage/troubleshooting) page or seek community help in the project’s GitHub repository.
</Note>
