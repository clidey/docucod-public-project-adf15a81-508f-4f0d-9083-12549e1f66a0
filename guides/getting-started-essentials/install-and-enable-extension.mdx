# Install and Enable uBO Scope

This guide provides a clear, step-by-step walkthrough for installing and enabling the uBO Scope browser extension across Chromium, Firefox, and Safari browsers. You will learn how to find the extension in official browser stores, grant necessary permissions during installation, and verify that uBO Scope is actively monitoring your browser sessions.

---

## 1. Workflow Overview

**What this guide helps you achieve:**
- Install the uBO Scope extension on supported browsers.
- Enable the extension with required permissions.
- Confirm that uBO Scope is actively tracking third-party connections in your browser.

**Prerequisites:**
- A supported browser version:
  - Chromium-based browsers version 122 or later.
  - Firefox version 128 or later.
  - Safari version 18.5 or later.
- Internet connection to access the browser extensions store.

**Expected Outcome:**
- uBO Scope installed and enabled as a browser extension.
- Toolbar icon with badge dynamically showing third-party domains connected.
- Ability to access the popup panel with live data on network connections.

**Time Estimate:** Approximately 5–10 minutes depending on browser and internet speed.

**Difficulty Level:** Beginner

---

## 2. Step-by-Step Installation Instructions

### Step 1: Choose Your Browser and Open Extension Store

- **Chromium (Chrome, Edge, Brave, etc.):** Navigate to the [Chrome Web Store page for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
- **Firefox:** Visit the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
- **Safari:** Use the Safari Extensions Gallery or App Store to search for "uBO Scope" compatible with Safari 18.5+.

<Tip>
Always install uBO Scope from the official store for your browser to ensure you get the authentic and latest version.
</Tip>

### Step 2: Install the Extension

1. Click the "Add to [Browser]" or "Install" button on the extension page.
2. When prompted, review the permissions required by uBO Scope:
   - Access to browser tabs.
   - Storage permissions
   - Access to webRequest API for monitoring network traffic.
3. Confirm and allow these permissions.

<Tip>
The permissions are essential because uBO Scope needs to monitor network requests to report third-party connections accurately.
</Tip>

### Step 3: Confirm Installation and Enable Extension

- After installation, the uBO Scope icon will appear in your browser's toolbar.
- Ensure the extension is enabled (usually indicated by the icon being colored and clickable).

<Tip>
If the icon does not appear, try checking your browser’s extensions menu or toolbar customization settings to ensure it is visible.
</Tip>

### Step 4: Grant Additional Permissions If Prompted

- Some browsers may request additional permissions on the first run or after updates.
- Approve any permission requests related to webRequest monitoring or tab access promptly.

### Step 5: Verify uBO Scope is Active

To confirm uBO Scope is actively monitoring your browsing session:

1. Visit a website you frequently use.
2. Click the uBO Scope toolbar icon to open its popup panel.
3. Check if domains appear under the "not blocked", "stealth-blocked", or "blocked" categories.
4. Look at the badge count on the toolbar icon showing the number of distinct connected third-party domains.

<Check>
An active, populated panel and a dynamic badge count indicate correct installation and operation.
</Check>

---

## 3. Practical Example: Installing on Chromium Browser

1. Open Chrome and navigate to the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome**.
3. Review permissions: confirm access to active tabs, webRequest API, and storage.
4. Click **Add extension**.
5. Wait for the download and installation to complete.
6. Locate the uBO Scope icon on the toolbar.
7. Visit a website like example.com, click the extension icon, and verify domains are listed and the badge count updates.

## 4. Troubleshooting Installation Issues

<AccordionGroup title="Common Installation and Activation Issues">
<Accordion title="Extension Icon Not Visible After Installation">
- Check if the extension is enabled in your browser's extension or add-on page.
- Ensure the toolbar icon is not hidden behind the extension menu or customization settings.
- Restart the browser if changes don't take effect immediately.
</Accordion>
<Accordion title="Permissions Prompt Not Shown or Denied">
- uBO Scope requires access to webRequest and active tab permissions.
- Manually grant these from your browser’s extension permissions settings.
- Reinstall the extension if permissions were denied accidentally.
</Accordion>
<Accordion title="uBO Scope Not Reporting Any Domains">
- Verify your browser supports the necessary minimum version (Chromium 122, Firefox 128, Safari 18.5).
- Ensure no other privacy extension or browser setting blocks webRequest API access.
- Try refreshing the current tab or restarting the browser.
</Accordion>
</AccordionGroup>

<Tip>
For more detailed troubleshooting, refer to the [Common Installation & Activation Issues guide](/getting-started/troubleshooting/common-issues).
</Tip>

## 5. Best Practices and Tips

- Update your browser regularly to maintain compatibility with uBO Scope.
- Avoid manually modifying extension permissions unless absolutely necessary.
- Use uBO Scope alongside your preferred content blocker to gain visibility into hidden third-party connections.
- Pay attention to the extension badge count to monitor privacy exposure trends.
