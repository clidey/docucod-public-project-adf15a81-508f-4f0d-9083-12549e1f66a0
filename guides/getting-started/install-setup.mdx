# Installing and Setting Up uBO Scope

Welcome to your first step in harnessing the power of uBO Scope. This guide offers a straightforward, beginner-friendly walkthrough for installing and setting up uBO Scope on your preferred browser — Chromium, Firefox, or Safari. By the end, you will have the extension installed, configured properly, and ready for use, with tips on how to pin it for quick access and verify it is running correctly.

---

## 1. What You'll Achieve

- Install uBO Scope from the official browser extension stores or manually load the extension in developer mode.
- Understand platform-specific nuances and requirements.
- Pin the extension icon on your browser toolbar for easy access.
- Launch uBO Scope and verify it is active and capturing network connections.

### Prerequisites

- A compatible browser version: Chromium 122+, Firefox 128+, or Safari 18.5+.
- Internet connection to download the extension.
- Familiarity with basic browser operations like locating extension stores and managing toolbar icons.

### Estimated Time

- About 5 to 10 minutes.

### Difficulty Level

- Beginner

---

## 2. Step-by-Step Installation & Setup

### Step 1: Installing uBO Scope

1. Open your browser:
   - **Chromium-based browsers** (Google Chrome, Edge, Brave, etc.)
   - **Firefox**
   - **Safari**

2. Visit the official extension store page for uBO Scope:

   | Browser        | Link                                                                            |
   |----------------|---------------------------------------------------------------------------------|
   | Chromium       | [Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp) |
   | Firefox        | [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)          |
   
   > **Note:** Safari users currently need to get uBO Scope via the Apple App Store or Apple’s Safari Extensions mechanism applicable for version 18.5 and up.

3. Click **Add to Browser** (or similar button).
4. If prompted, confirm the permissions request involving `webRequest`, `storage`, and `activeTab` permissions.

<Tip>
Always install uBO Scope from the official browser extension stores linked above to ensure the latest, verified version.
</Tip>

### Step 2: Pin the Extension for Easy Access

After installation, the extension icon might be hidden behind the browser’s extensions menu.

- **Chromium:** Click the puzzle-piece icon in the toolbar, find uBO Scope, then click the pin icon to have it always visible.
- **Firefox:** Click the puzzle icon, then the pushpin icon next to uBO Scope to pin it.
- **Safari:** The extension icon appears automatically in the toolbar, or under the extensions menu (the puzzle piece). You can customize the toolbar by right-clicking and selecting `Customize Toolbar` to add uBO Scope.

<Tip>
Pinning uBO Scope means quicker access to its popup for real-time insights into your network connections.
</Tip>

### Step 3: Launch uBO Scope for the First Time

1. Click the uBO Scope icon in your browser toolbar.
2. The popup will open, initially showing 'NO DATA' until it receives network connection information for the current tab.
3. Navigate to a website and let uBO Scope observe the network requests made on the page.
4. Re-open the popup; you should see:
   - The **domain** and subdomain information.
   - Counts of network connections broken down into **not blocked**, **stealth-blocked**, and **blocked** domains.
   - A summary badge count on the toolbar icon showing the number of third-party remote servers connected.

<Check>
If the popup does not display data or the badge does not update, try refreshing the page or restarting the browser.
</Check>

## 3. Platform-Specific Tips & Considerations

| Platform   | Notes                                                                                             |
|------------|-------------------------------------------------------------------------------------------------|
| Chromium   | Minimum version 122. Extension uses service worker as background script.                         |
| Firefox    | Version 128+ required. Supports full webRequest URLs including ws:// and wss://.                 |
| Safari     | Requires Safari 18.5 or later. Background script is a module with specific permissions setup. |

<Tip>
On Firefox, uBO Scope tracks additional websocket connections, providing broader network visibility.
</Tip>

<Tip>
Safari users should ensure to update to the minimum supported version to avoid compatibility issues.
</Tip>

---

## 4. Verifying Installation Is Active

### Toolbar Badge

- After visiting a site, check the extension icon on the toolbar for a badge number.
- The badge shows how many distinct third-party remote servers were contacted.

### Popup Interface Confirmation

- Open the popup and confirm it shows data sections labeled **not blocked**, **stealth-blocked**, and **blocked**.
- Each section lists domains connected to during the active tab session.

<Tip>
The badge count is a reliable indicator of uBO Scope's active monitoring for the current tab.
</Tip>

### Quick Refresh

- If no data appears, refresh the website and wait a few seconds.
- Then check the popup data again.

## 5. Troubleshooting Common Issues

<AccordionGroup title="Troubleshooting Installation and Setup Issues">
<Accordion title="Extension Icon Not Visible">
Make sure you have pinned uBO Scope in your browser toolbar. Check the extensions menu (puzzle icon) to locate it if not pinned.
</Accordion>

<Accordion title="Popup Shows 'NO DATA'">
- Refresh the active tab and wait for network requests.
- Verify uBO Scope permissions are granted.
- Restart the browser if the issue persists.
</Accordion>

<Accordion title="Badge Count Does Not Update">
- Confirm you are visiting sites with third-party network requests.
- Some sites may have very few or no third-party connections.
- Restarting the browser often resolves badge update delays.
</Accordion>
</AccordionGroup>

## 6. Best Practices & Tips

- **Pin It:** Keep uBO Scope visible on your toolbar for quick access to real-time insights.
- **Refresh:** Reload pages after installing to let uBO Scope capture network activity.
- **Interpret Badge:** A lower badge count generally means fewer connections to third-party servers, aligning with privacy preferences.
- **Combine With Other Tools:** Use alongside uBlock Origin for a comprehensive content blocking and tracking awareness setup.
