# Installing uBO Scope on Your Browser

This guide walks you through the installation process for uBO Scope on the major supported browsers: Chromium-based browsers (e.g., Google Chrome, Brave), Firefox, and Safari. You'll learn where to find the extension in official stores, how to install it, and how to verify the extension is correctly active and collecting data.

---

## 1. Choosing the Right Browser Version

uBO Scope supports the following browsers with minimum version requirements:

| Browser                 | Minimum Version |
|-------------------------|-----------------|
| Chromium-based browsers  | 122.0 or later  |
| Firefox                 | 128.0 or later  |
| Safari                  | 18.5 or later   |

Ensure your browser is updated to meet or exceed these versions for compatibility and security.

---

## 2. Finding the Official uBO Scope Extension

The extension is published on official browser marketplaces. Always install from these trusted sources to ensure authenticity and automatic updates:

- **Chromium-based browsers:**
  - [Chrome Web Store – uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)
- **Firefox:**
  - [Firefox Add-ons – uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/)

> **Note:** Safari users should check the [Mac App Store](https://apps.apple.com) or Apple's official Safari Extensions Gallery for availability.

---

## 3. Step-by-Step Installation

Follow these steps to install uBO Scope on your browser.

### For Chromium-based Browsers

<Steps>
<Step title="Open Chrome Web Store">
Navigate to the [uBO Scope page on the Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
</Step>
<Step title="Click 'Add to Chrome'">
Click the 'Add to Chrome' button to start the installation.
</Step>
<Step title="Confirm Permissions">
Review the permissions requested by uBO Scope, typically including access to tabs, storage, and web requests. Click 'Add extension' to proceed.
</Step>
<Step title="Verify Installation">
After installation, look for the uBO Scope icon near your address bar. It should appear without error.
</Step>
</Steps>

### For Firefox

<Steps>
<Step title="Open Firefox Add-ons Store">
Visit the [uBO Scope page on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/).
</Step>
<Step title="Click 'Add to Firefox'">
Click the 'Add to Firefox' button to install.
</Step>
<Step title="Grant Required Permissions">
Accept the permission prompts to allow the extension to monitor network requests.
</Step>
<Step title="Verify Extension Icon">
Look for uBO Scope’s icon in the toolbar to confirm successful installation.
</Step>
</Steps>

### For Safari

1. Open the Safari Extensions section in the Mac App Store or Safari preferences.
2. Search for "uBO Scope" or check the official Apple extensions repository.
3. Follow the prompts to install and enable the extension.
4. Confirm the extension icon is visible in Safari’s toolbar.

> **Tip:** Safari extensions may require Safari to be restarted after installation.

---

## 4. Verifying uBO Scope Is Active

Once installed, ensure uBO Scope is actively monitoring your browser:

- **Extension Icon Presence:** The uBO Scope icon should be visible in the browser’s toolbar.
- **Popup Access:** Click the icon to open the popup dashboard.
- **Real-Time Data:** The popup should display the hostname of the active tab and show counts of connected domains under categories "not blocked", "stealth-blocked", and "blocked".
- **Badge Number:** The toolbar icon badge displays the number of distinct, allowed third-party domains connected by the active page.

If these elements are missing or empty, the extension may not be running correctly.

---

## 5. Troubleshooting Common Installation Issues

<Tip>
If you do not see data in the popup or the badge does not update:
- Confirm your browser permissions allow uBO Scope to monitor tabs and web requests.
- Check if other extensions might conflict with uBO Scope.
- Restart your browser to initialize the extension properly.
- Verify you are on a supported browser version.
- See the [Troubleshooting Installation & Setup](../getting-started/first-use-and-validation/troubleshooting-installation.md) guide for detailed help.
</Tip>
