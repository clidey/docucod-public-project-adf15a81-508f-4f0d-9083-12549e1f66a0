# Installing uBO Scope on Your Browser

## Overview
This guide provides clear, step-by-step instructions to install uBO Scope on supported browsers including Chromium-based browsers (Google Chrome, Microsoft Edge, etc.), Mozilla Firefox, and Apple Safari. You'll find download links, platform-specific considerations, and practical tips to address common installation challenges.

## Prerequisites
Before beginning the installation, ensure you have:

- A supported browser version:
  - Chromium-based browsers: version 122 or later
  - Firefox: version 128 or later
  - Safari: version 18.5 or later (where available)
- An active internet connection to download the extension
- Permissions to install browser extensions

## Expected Outcome
By completing this guide, you will have uBO Scope installed and ready to use in your browser, enabling you to monitor remote server connections made by your tabs through its popup UI and toolbar badge.

## Time Estimate
The entire installation process typically takes 5 minutes or less, depending on your internet speed and browser setup.

---

## Step-by-Step Installation Instructions

### 1. Choose Your Browser

Identify which browser you will be installing uBO Scope on. This guide supports:

- Chromium-based browsers (Google Chrome, Microsoft Edge, Brave, etc.)
- Mozilla Firefox
- Apple Safari

### 2. Access the Official Extension Store

To ensure security and integrity, only install uBO Scope from your browser’s official extension marketplace:

| Browser          | Installation Link                                                                                   |
|------------------|--------------------------------------------------------------------------------------------------|
| Chromium Browsers | [Chrome Web Store - uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp) |
| Firefox          | [Firefox Add-ons - uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/)                |

For Safari, installation depends on availability through the App Store or developer-provided instructions due to Apple's extension policies.

### 3. Installing uBO Scope

#### For Chromium-based Browsers
1. Open the Chrome Web Store link above.
2. Click the **Add to Chrome** button.
3. Confirm the installation by clicking **Add extension** in the prompt.
4. Once installed, the uBO Scope icon will appear in your browser toolbar.

#### For Mozilla Firefox
1. Open the Firefox Add-ons page via the link above.
2. Click **Add to Firefox**.
3. Approve the permissions and confirm **Add**.
4. The uBO Scope icon will then appear in the toolbar.

#### For Safari
1. Refer to your Safari extension management or the official Apple App Store if available.
2. Follow any platform-specific installation instructions.

### 4. Confirm Installation

- Look for the uBO Scope icon in your browser’s toolbar area.
- In Chromium and Firefox, you can right-click the icon and verify extension details.
- Safari users should review extension management from Safari Preferences.

### 5. Initial Use

- Click the uBO Scope toolbar icon to open the popup.
- The popup will present network connection data for the current tab (initially it may appear empty until you browse).

## Platform-Specific Considerations

### Chromium
- Minimum Chrome version 122 is required.
- The extension runs a service worker background script (`background.js`) to monitor network requests.
- WebSocket connections are monitored where possible.

### Firefox
- Requires Firefox version 128 or later.
- Uses traditional background scripts for persistent data.
- Permissions include `webRequest`, `activeTab`, and `storage`.

### Safari
- Requires Safari 18.5 or later.
- Some features may be limited due to platform API support.
- Follow Apple’s extension installation and signing requirements.

## Troubleshooting Installation Issues

If you encounter problems, try the following steps:

### Issue: uBO Scope icon does not appear after installation
- Ensure the installation completed successfully.
- Check your browser’s extension page to confirm uBO Scope is listed and enabled.
- Try to manually enable the extension or pin it to the toolbar.

### Issue: Permissions prompt rejected or not granted
- uBO Scope requires permissions for active tab access, network request monitoring, and storage.
- Accept these permissions when prompted; otherwise, the extension cannot function.

### Issue: Extension not updating or behaving unexpectedly
- Remove and reinstall uBO Scope from your browser’s extension management page.
- Restart your browser.

### Issue: No data shown in popup after installation
- Browse to a website and reload the popup.
- Some network data may take a moment to appear as network requests are tracked.
- Verify that no other extensions block vital browser APIs.

   - Choose `Load Temporary Add-on` and select any file inside the unpacked extension directory.

   - Use Xcode and Safari Extension Builder.
