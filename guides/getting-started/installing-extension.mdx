# Installing uBO Scope on Your Browser

This guide walks you through the clear, practical steps to install uBO Scope on Chromium, Firefox, and Safari browsers. By following this tutorial, you will securely obtain the extension from official sources, verify correct installation, and achieve a ready-to-use setup for reliable third-party connection monitoring.

---

## 1. Overview

uBO Scope is a privacy-centric browser extension that shows all third-party network connections a webpage makes, regardless of any existing content blockers. To benefit from its insights, you must install it on a supported browser and ensure it is active.

### Prerequisites

- Use a supported browser:
  - Chromium-based browsers version 122 or higher (e.g., Chrome, Edge)
  - Firefox version 128 or higher
  - Safari version 18.5 or higher
- Confirm your browser supports extension installation and has access to the WebRequest API.
- Verify you have internet access to download the extension.

### Expected Outcome

- uBO Scope installed and enabled in your browser.
- Toolbar icon appears with uBO Scopeâ€™s badge ready to monitor network connections.
- Ability to open the extension popup for detailed third-party connection data.

### Time Estimate

Approximately 5-10 minutes depending on browser and connection speed.

### Difficulty Level

Beginner-friendly, with step-by-step instructions.

## 2. Installation Steps

### Step 1: Choose Your Browser

Decide which supported browser you will install uBO Scope on. The details below cater to Chromium, Firefox, and Safari.

### Step 2: Access Official Extension Source

- **Chromium (Chrome, Edge, etc.)**
  - Visit the Chrome Web Store page for uBO Scope:
    [uBO Scope on Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)

- **Firefox**
  - Visit the official Mozilla Add-ons page:
    [uBO Scope on Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)

- **Safari**
  - Use the Mac App Store or official Safari Extensions gallery to find uBO Scope.
  - Safari support requires version 18.5 or later.

<Tip>
Always install uBO Scope from official browser stores to avoid security risks and ensure you get the latest stable version.
</Tip>

### Step 3: Install the Extension

- Click the **Add to Chrome** / **Add to Firefox** / **Get** button on the store page.
- Confirm any permission prompts the browser shows, including access to network requests and active tabs.
- Wait for the installation to complete; the browser will notify you once it's done.

<Tip>
The extension requires permissions such as `webRequest` and `storage` to monitor network connections accurately.
</Tip>

### Step 4: Verify Installation

- Look for the uBO Scope icon in your browser's toolbar. The icon features the uBO Scope logo.
- In Chromium and Firefox, pin the extension icon to the toolbar if not automatically shown.
- Check that there is no warning symbol on the icon indicating an installation or permission problem.

### Step 5: Initial Setup

- Click the uBO Scope icon to open its popup.
- On first use, the extension may request permissions or notify about its operation.
- Ensure the badge count is visible and updates when you browse websites.

<Tip>
If the badge remains empty or you see 'NO DATA' in the popup, browse to a website (e.g., https://example.com) and watch the extension update.
</Tip>

## 3. Browser-Specific Notes

### Chromium

- Minimum required version 122.
- Extension uses a service worker for background processes.
- Permissions include `webRequest` to monitor HTTPS and HTTP traffic.

### Firefox

- Minimum required version 128.
- The extension supports WebSocket URLs in addition to HTTP/HTTPS.
- Firefox requires explicit permissions and may prompt for additional confirmation.

### Safari

- Version 18.5 or higher required.
- Uses module scripts for background tasks.
- Extension installed via official Safari extension gallery or Apple App Store.

## Appendix: Example Installation Scenario (Chrome)
