# Installing uBO Scope in Your Browser

## Overview
This guide walks you through installing uBO Scope on the major supported browsers: Google Chrome (Chromium-based), Mozilla Firefox, and Apple Safari. Following these steps will ensure you have the extension correctly installed, with the required permissions, and that it is actively monitoring your network connections.

---

## What You Will Achieve
- Locate the official uBO Scope extension on your browser's storefront
- Understand permission requirements and their purpose
- Install the extension securely
- Verify that uBO Scope is running and ready for use

---

## Prerequisites
Before starting, make sure:
- Your browser version is supported:
  - Chromium browsers version 122 or newer
  - Firefox version 128 or newer
  - Safari version 18.5 or newer
- You have an active internet connection to access the browser's extension store
- You have basic familiarity with installing extensions in your browser

---

## Step-by-Step Installation Instructions

### 1. Choose Your Browser
The installation process varies slightly depending on your browser. Select the relevant section below.

---

### 2. Installing on Chromium-Based Browsers (e.g., Google Chrome, Microsoft Edge)

1. **Open the Chrome Web Store:**  
   Visit the [Chrome Web Store listing for uBO Scope](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).

2. **Add the Extension:**  
   Click the `Add to Chrome` button.

3. **Review Permissions:**  
   The extension requests permissions including `activeTab`, `storage`, and `webRequest` which allow it to monitor network connections without modifying them.

4. **Confirm Installation:**  
   Accept the permissions and wait for the extension to install.

5. **Verify Installation:**
   - You should see the uBO Scope icon in your browser’s toolbar.
   - The icon will have a badge displaying the number of distinct third-party domains connected.

<Info>
Permissions Explained: uBO Scope requires these permissions to monitor your browser's network requests faithfully, track allowed, blocked, and stealth-blocked connections, and to store session data for per-tab analysis.
</Info>

---

### 3. Installing on Mozilla Firefox

1. **Open Firefox Add-ons:**
   Visit the [uBO Scope Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/ubo-scope/).

2. **Add to Firefox:**
   Click the `Add to Firefox` button.

3. **Review Permissions:**
   Firefox will prompt for permissions such as access to tab information, storage, and webRequest APIs necessary to monitor network connections.

4. **Complete Installation:**
   Accept the prompts and let the extension install.

5. **Verify Installation:**
   Confirm the uBO Scope icon appears in the toolbar with the badge count visible after network activity.

---

### 4. Installing on Safari

1. **Access the Safari Extension Gallery:**
   Use the Safari Extensions section of the App Store to find "uBO Scope".

2. **Install the Extension:**
   Click `Get` and then `Install`. You may be prompted for your Apple ID credentials.

3. **Enable in Safari Preferences:**
   After installation, go to Safari > Preferences > Extensions and ensure uBO Scope is enabled.

4. **Review Permissions:**
   Safari will request permission to monitor network requests and access browser data.

5. **Verify Installation:**
   Check that the uBO Scope icon appears in Safari’s toolbar and that it updates as you browse.

---

### 5. Verifying uBO Scope Is Active and Ready

- **Toolbar Badge:** The extension icon displays a badge showing the number of distinct third-party domains connected for the current tab. A count greater than zero indicates it's tracking connections properly.

- **Popup UI:** Clicking the toolbar icon opens the popup window showing detailed lists of allowed, stealth-blocked, and blocked domains connected by the current active tab.

- **Testing:** Visit popular websites and watch for the badge count to update reflecting real network activity.

<Check>
If the badge remains empty or no data is shown, verify that:
- The extension is enabled in your browser’s extensions manager.
- The browser version meets requirements.
- Necessary permissions were granted during installation.
</Check>

## Best Practices

- Always install uBO Scope from the official browser store links to ensure security and authenticity.
- Keep your browser updated to the latest supported versions for compatibility.
- Regularly verify the extension remains enabled after browser updates.

## Common Pitfalls & Troubleshooting

<AccordionGroup title="Common Installation Issues and Solutions">
<Accordion title="Extension Icon Missing After Installation">
- Ensure the extension is enabled in the extension manager.
- Pin the uBO Scope icon to your browser toolbar if hidden.
- Restart your browser and check again.
</Accordion>
<Accordion title="Badge Count Does Not Appear">
- Confirm you have visited pages that generate network connections.
- Some pages with no third-party connections may not show a badge.
- Check if browser restrictions or other extensions interfere with network monitoring.
</Accordion>
<Accordion title="Permissions Prompt Blocked or Denied">
- Reinstall the extension and accept permissions.
- Check browser settings for blocking extension permissions.
- Follow browser-specific help guides for granting extension permissions.
</Accordion>
</AccordionGroup>
