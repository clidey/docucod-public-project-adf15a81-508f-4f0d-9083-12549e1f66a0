# Getting Started: Installing and Accessing uBO-Scope

Welcome to the quick start guide for uBO-Scope! This guide walks you through installing the extension on supported browsers and how to open and access the extension's popup interface. It is perfect for users new to uBO-Scope or browser extensions in general, aiming to get you monitoring third-party network connections as fast and smoothly as possible.

---

## 1. Introduction

uBO-Scope is a browser extension that reveals all remote server connections initiated by websites you visit. It reports distinct third-party connections in real-time and displays this information via a toolbar badge and a detailed popup UI. This guide ensures you can:
- Install uBO-Scope correctly on your preferred browser
- Confirm the extension is active and visible
- Open and explore the popup interface to view connection data

## 2. Prerequisites

Before installing uBO-Scope, ensure the following:

- **Supported Browsers:**
  - Chromium-based browsers (Chrome, Edge, Brave) version 122 or higher
  - Firefox version 128 or higher
  - Safari version 18.5 or higher

- **Operating System:** Compatible with your browser's system requirements

- **Permissions:** The extension requires permissions for active tab access, webRequest monitoring, and storage (managed automatically during installation)

- **Network Access:** Ability to access the Chrome Web Store, Firefox Add-ons, or Safari Extensions Gallery

- **Avoid installing conflicting extensions** that may suppress extension visibility or block extension APIs

For detailed prerequisites, see the [Prerequisites & System Requirements](../../getting-started/installation-and-setup/prerequisites-and-requirements) documentation.

## 3. Installing uBO-Scope

Follow the steps below depending on your browser:

### Chromium-based Browsers

1. Navigate to the [Chrome Web Store uBO-Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome** (or equivalent on your Chromium variant).
3. Confirm the installation by clicking **Add extension** in the prompt.
4. Once installed, the uBO-Scope icon appears alongside your browser's toolbar extensions.

### Firefox

1. Go to the [Firefox Add-ons uBO-Scope page](https://addons.mozilla.org/firefox/addon/ubo-scope/).
2. Click **Add to Firefox**.
3. Confirm by clicking **Add** in the permission prompt.
4. The extension icon will then appear on your Firefox toolbar.

### Safari

1. Access the Safari Extensions Gallery or Mac App Store (for Safari 18.5+).
2. Search for "uBO Scope".
3. Install the extension.
4. Enable uBO-Scope under Safari Preferences > Extensions.

<Tip>
Make sure your browser is up to date to meet minimum version requirements for compatibility.
</Tip>

## 4. Confirming Installation and Access

After installation, verify and access uBO-Scope as follows:

### Step 1: Locate the uBO-Scope Icon

- Find the uBO-Scope icon on your browser's extension toolbar.
- The icon depicts the uBO logo in varying sizes depending on DPI.

### Step 2: Open the Extension Popup

- Click the uBO-Scope icon to open the extension popup window.
- The popup displays the number of distinct remote domains connected to by the active webpage.
- It categorizes domains into **Not Blocked**, **Stealth-Blocked**, and **Blocked**.

### Step 3: Load a Website to See Data

- Navigate to any website (preferably a site with third-party resources, e.g., news or shopping sites).
- Click the uBO-Scope icon again to refresh the popup.
- You should see the domain and count information update live.

<Note>
The badge count on the toolbar icon reflects the number of distinct third-party remote servers your browser connected to while on the active tab. A lower count usually indicates fewer third-party connections.
</Note>

## 6. Understanding the Popup UI

The popup interface categorizes connections as:

- **Not Blocked:** Connections allowed by the browser
- **Stealth-Blocked:** Connections blocked silently without user-visible indication
- **Blocked:** Connections actively blocked by content blocking or network defenses

Each category lists distinct connected domains along with the count of network requests to those domains.

## 7. Tips for a Smooth Setup

- **Pin the Extension:** Pin the uBO-Scope icon to your toolbar for easy access.
- **Use Real Websites:** Test uBO-Scope on popular content-heavy pages to generate meaningful connection data.
- **Allow Permissions:** Accept all extension permission requests; webRequest permission is mandatory for accurate data.

## 8. Troubleshooting Common Installation and Access Issues

<AccordionGroup title="Troubleshooting Guide">
<Accordion title="Extension Icon Missing After Installation">
- Ensure the extension is enabled in your browser's extension manager.
- In Chromium, check chrome://extensions and toggle the extension ON.
- Restart your browser if necessary.
- In Firefox, check about:addons.
</Accordion>
<Accordion title="Popup Shows 'NO DATA' or No Connections">
- Browse to a website first, then open the popup.
- Ensure no conflicting privacy extensions interfere with webRequest API.
- Try refreshing the tab and opening the popup again.
</Accordion>
<Accordion title="Badge Count is Not Updating">
- Verify the extension has the required permissions.
- Confirm your browser is at the supported version.
- Try reinstalling the extension.
</Accordion>
</AccordionGroup>
