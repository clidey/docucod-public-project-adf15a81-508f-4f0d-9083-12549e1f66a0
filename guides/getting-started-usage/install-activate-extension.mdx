# Install and Activate uBO Scope

## Overview
This guide walks you through the essential steps to install uBO Scope on supported browsers (Chrome, Firefox), pin the extension for ready access, and verify that it is properly activated. Following this guide ensures you set up uBO Scope correctly so you can start monitoring third-party server connections immediately as you browse.

---

## 1. Preparing to Install uBO Scope

### Supported Browsers and Versions
- **Chrome:** Version 122.0 or higher
- **Firefox:** Version 128.0 or higher

Ensure your browser version meets or exceeds these minimums for full compatibility.

### Prerequisites
- A stable internet connection
- Access to the browser's extension/add-on store

### Why Pin the Extension?
Pinning the extension’s icon to the browser toolbar ensures quick, convenient access to uBO Scope’s popup panel where you can view connection data live.

---

## 2. Installation Steps

### Step 1: Access the Official Extension Store

| Browser | Official Store Link |
|---------|--------------------|
| Chrome  | [Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp) |
| Firefox | [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/) |

_Do not install uBO Scope from any unofficial source to maintain security._

### Step 2: Add the Extension
1. Click the **Add to Chrome** or **Add to Firefox** button on the store page.
2. Accept all required permissions including `webRequest`, `activeTab`, and `storage`.
3. Wait for the extension to download and install.

### Step 3: Pin the Extension (Recommended)

#### Chrome
- Click the puzzle piece icon (Extensions) next to the address bar.
- Find **uBO Scope** in the list.
- Click the pushpin icon to pin it to your toolbar.

#### Firefox
- Click the menu button (three bars).
- Select **Customize Toolbar**.
- Find **uBO Scope** and drag it to the toolbar.
- Click **Done** when finished.

---

## 3. Confirming Proper Activation

Once installed and pinned, confirm uBO Scope is ready:

### Step 1: Open a Webpage
- Visit any website (for example, `https://example.com`).

### Step 2: Open uBO Scope Popup
- Click the uBO Scope icon in your browser’s toolbar.

### Step 3: Verify Data Display
- The popup header should show the current tab’s hostname and domain.
- The panel lists distinct remote server domains categorized as:
  - **Not Blocked:** Connections that were allowed.
  - **Stealth-Blocked:** Connections that were stealthily redirected or blocked.
  - **Blocked:** Connections that failed.
- The summary section shows the count of distinct connected domains.

### Step 4: Check the Badge Counter
- Look at the extension’s toolbar icon badge.
- It should show a number representing the count of unique third-party remote servers connected from the page.
- No badge or zero indicates no connections detected.

<Tip>
If you see no data or "NO DATA" in the popup, try refreshing the webpage or verifying proper permissions were granted during installation.
</Tip>

## 4. Troubleshooting Common Installation and Activation Issues

<AccordionGroup title="Troubleshooting Common Issues">
<Accordion title="The uBO Scope icon does not appear in the toolbar">
- Check if the extension installed successfully via your browser’s extension management page (`chrome://extensions/` or `about:addons`).
- Pin the extension manually as described above.
- Restart your browser and try again.
</Accordion>
<Accordion title="The popup shows 'NO DATA' or is empty">
- Make sure you have loaded an active webpage (not a new tab or internal browser page).
- Reload the page and retry.
- Confirm you granted all necessary permissions during installation.
- Consult the Troubleshooting guide for installation issues.
</Accordion>
<Accordion title="Badge count is missing or zero unexpectedly">
- Badge only shows counts for pages with third-party connections.
- Try on multiple websites with known external resources.
- Verify the extension is enabled and no other extension blocks its operation.
</Accordion>
</AccordionGroup>

## 5. Best Practices to Start Using uBO Scope

- Always pin the extension for easy access during browsing.
- Visit diverse sites to observe how uBO Scope reports remote connections in real-time.
- Cross-check badge counts with what you see in the popup panel for a more comprehensive view.
