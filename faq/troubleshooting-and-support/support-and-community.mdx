# Where Can I Get Help or Contribute Feedback?

This page guides you to the range of resources available for support, feedback, and contributing to uBO Scope. Whether you need help resolving issues, want to request new features, or wish to get involved in development, the paths below will point you in the right direction.

---

## Support Channels

### Extension Store Pages

For official downloads, updates, and basic user reviews or comments, visit the appropriate extension page for your browser:

- **Chromium-based browsers (Chrome, Edge, etc.):** [Chrome Web Store - uBO Scope](https://chrome.google.com/webstore/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp)
- **Firefox:** [Firefox Add-ons - uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/)
- **Safari:** Refer to Appleâ€™s Safari Extensions Gallery or relevant Safari extension store.

These pages often have user discussions, version updates, and important notices.

---

### GitHub Repository

The primary hub for issue tracking, feature requests, source code, and community contributions is the uBO Scope GitHub repository:

- **Repository URL:** [https://github.com/gorhill/uBO-Scope](https://github.com/gorhill/uBO-Scope)

On GitHub you can:

- Report bugs or issues encountered during use.
- Request new features or improvements.
- View the project's source code.
- Submit pull requests if you want to contribute directly.

It is the authoritative place where active development and community engagement happen.

---

## How to Report Issues Effectively

To help maintainers diagnose problems quickly and efficiently:

- Clearly describe the problem you experienced.
- Include steps to reproduce the issue if applicable.
- Provide your browser and uBO Scope version.
- Attach screenshots or logs if possible.

For installation or setup issues, consider reviewing the troubleshooting documentation first to rule out common problems.

---

## Participating in the Project

If you want to contribute improvements or help maintain uBO Scope:

- Fork the repository on GitHub.
- Review the source code and existing issues.
- Follow the contribution guidelines outlined in the repository (if any).
- Use the `make-package.sh` script provided in the `tools` directory to build extension packages for different platforms.

<u>Note:</u> uBO Scope is a focused extension designed primarily for transparency about third-party connections, and the maintainer does not plan frequent updates but welcomes meaningful contributions.

---

## Additional Community Resources

While there is no official forum or dedicated community chat for uBO Scope, you can:

- Engage on GitHub Discussions (if enabled).
- Participate in broader forums or communities around uBlock Origin and browser privacy tools.
- Ask questions and share insights on platforms like Reddit or Stack Exchange under relevant tags.
