# How Do I Install uBO Scope? Which Browsers Are Supported?

This guide provides clear and practical instructions to install **uBO Scope** on the supported browsers: **Chrome**, **Firefox**, and **Safari**. It highlights platform-specific details to ensure a smooth installation experience and clarifies minimum version requirements to avoid compatibility issues.

---

## Supported Browsers and Minimum Requirements

- **Google Chrome (Chromium-based browsers)**
  - Minimum version: **122.0**
  - Supported on desktop platforms where Chrome extensions are available.

- **Mozilla Firefox**
  - Minimum version: **128.0** (including Firefox for Android)

- **Apple Safari**
  - Minimum version: **18.5**
  - Supports Safari Web Extensions.

<u>Note:</u> Using older browser versions may prevent proper installation or operation of the extension.

---

## Installation Overview

### General Prerequisites

- Ensure you are running a supported browser version.
- Confirm that your device has an active internet connection to access the official browser extension stores.
- Have sufficient permissions to install extensions in your browser profile.

### Installing from Official Sources

#### 1. Google Chrome (Chromium-based Browsers)

- Visit the [Chrome Web Store uBO Scope page](https://chrome.google.com/webstore/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
- Click **Add to Chrome** and accept any permission requests (`activeTab`, `storage`, `webRequest`).
- Once installed, the uBO Scope icon will appear on your toolbar.

#### 2. Mozilla Firefox

- Open the [Firefox Add-ons page for uBO Scope](https://addons.mozilla.org/firefox/addon/ubo-scope/).
- Click **Add to Firefox** and review permissions.
- Approve and finalize installation. The extension icon will be visible in your browser UI.

#### 3. Apple Safari

- Open the Safari Extensions Gallery or App Store and search for **uBO Scope**.
- Follow the system prompts to install the extension.
- Ensure your Safari version is at least 18.5 and that extensions are enabled in Safari's preferences.
- After installation, the extension icon will be available in the Safari toolbar.

## Permissions Explained

- **activeTab**: Allows the extension to monitor the active browser tab.
- **storage**: Enables saving session data and user preferences.
- **webRequest**: Permits interception and analysis of network requests to provide accurate connection insights.

Granting these permissions is essential for uBO Scope to function properly.

---

## Platform-Specific Notes

- **Chrome**: The extension runs as a service worker in the background with support for both HTTP(S) and WebSocket protocols.
- **Firefox**: Supports additional `wss://` and `ws://` protocols; the extension works on desktop and Firefox for Android.
- **Safari**: Due to platform constraints, ensure your Safari is up to date to avoid issues with background scripts.

## Troubleshooting Installation

If you encounter issues during installation or the extension does not appear or function as expected:

- Confirm browser version and update.
- Verify you installed from the official store.
- Check that the extension is enabled in your browser's extensions manager.
- Look for any permission blocks or prompts that may have been denied.
- Restart your browser.

For detailed troubleshooting, refer to [Troubleshooting Common Installation Issues](/getting-started/troubleshoot-validate/troubleshooting-install).
