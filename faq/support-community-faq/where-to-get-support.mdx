# How Do I Get Help or Report Issues?

If you encounter any issues, have questions, or want to provide feedback regarding uBO Scope, this page directs you to the appropriate support channels and resources to get timely and effective assistance. Whether you need help with using the extension, reporting bugs, or discovering additional documentation, this guide will make your journey smoother.

---

## Support Channels and Contact Methods

### 1. Official GitHub Repository

The primary hub for issue reporting, feature requests, and source code inspection is the official GitHub repository:

- **URL:** [https://github.com/gorhill/uBO-Scope](https://github.com/gorhill/uBO-Scope)

You can use the repository’s **Issues** section to:

- Report bugs and unexpected behaviors
- Suggest improvements or new features
- Track ongoing fixes and updates

Be sure to provide detailed information including your browser version, uBO Scope version, and steps to reproduce the issue.

### 2. Community and Forums

While uBO Scope itself is a niche tool, community discussions occur around the broader uBlock Origin ecosystem and browser privacy tools.

- **uBlock Origin Community Forums**: Participating in uBlock Origin forums may help as many principles and troubleshooting tips overlap.
- **Browser Extension Forums and Reddit**: Privacy and extension-related subreddits or forums are other useful places to seek advice.

### 3. Support via Documentation and FAQs

Before submitting an issue, consider the rich documentation available:

- Visit the **[FAQs](https://github.com/gorhill/uBO-Scope/tree/main/docs/faq)** section for common questions
- Review troubleshooting guides such as **[Troubleshooting Common Installation Issues](/getting-started/usage-validation-troubleshooting/troubleshooting-install)**
- Explore usage and setup guides for step-by-step assistance

Using these resources often provides instant answers without waiting for direct support.

---

## Best Practices When Reporting an Issue

To facilitate quick resolution of your problem, follow these practical tips:

- **Reproduce the Problem Step-by-Step:** Write down exact steps you took to observe the issue.

- **Include Environment Details:** Browser name and version (minimum Chrome 122, Firefox 128, Safari 18.5), operating system, and exact version of uBO Scope.

- **Provide Popup Data if Possible:** Screenshots or the data shown in the popup interface help diagnose network connection issues.

- **Check for Conflicting Extensions:** Sometimes other privacy tools or blockers interfere with uBO Scope’s operation.

- **Use the Latest Version:** Confirm you are running the latest release from the official stores or GitHub.

- **Review Log Data:** Advanced users can explore browser console logs or collect debugging info if requested by support.
