# Supported Browsers and Installation

Discover how to quickly set up and start using uBO Scope on your preferred browser. This guide details the supported browsers, installation sources, and essential tips for a smooth installation process.

---

## Supported Browsers

uBO Scope is designed with broad compatibility in mind, supporting the latest major desktop browsers where the required WebExtensions APIs are available and stable.

- **Chromium-based Browsers**
  - Includes Google Chrome (version 122.0 and later) and other Chromium derivatives.
  - Utilizes Manifest Version 3 with a service worker background script for performance and compatibility.

- **Mozilla Firefox**
  - Supported on Firefox version 128.0 and later.
  - Supports Manifest Version 3 with a module-type background script.

- **Safari**
  - Supported on Safari version 18.5 and later (WebExtension compatibility).
  - Uses Manifest Version 3 with module-type background script.


### Why These Browsers?

uBO Scope leverages the browser's `webRequest` API to monitor network requests with high fidelity. These APIs have stabilized on the above browsers and versions, ensuring that uBO Scope functions reliably and securely.

## Installation Instructions

Follow the steps below to install uBO Scope on your browser quickly and securely.

### Step 1: Choose Your Browser

Identify which browser you will use with uBO Scope.

### Step 2: Visit the Official Extension Store

To ensure security and authenticity, always install uBO Scope from the official web store for your browser:

| Browser              | Install From                                                   |
|----------------------|----------------------------------------------------------------|
| Chromium/Chrome      | [Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp) |
| Mozilla Firefox      | [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/) |

> **Note:** Safari users should check the Safari Extensions Gallery or the developer's GitHub for installation instructions, as Safari support is comparatively recent.

### Step 3: Install the Extension

- Click the `Add to Browser` or equivalent button on the extension page.
- Confirm any permission requests, notably access to network requests and tab information.

### Step 4: Verify Installation

- Once installed, the uBO Scope icon appears in the browser toolbar.
- Click the icon to open the popup, which will display data when browsing sites.

### Step 5: Start Browsing

- As you browse, uBO Scope monitors and categorizes network connections in real-time.
- The badge on the toolbar icon will indicate the count of distinct third-party domains connected to by the active tab.


## Important Permissions

uBO Scope requires the following permissions to provide accurate reporting:

- **`webRequest`**: To listen and monitor network requests in real-time.
- **`storage`**: To save session and state data between browser restarts.
- **`activeTab`**: To interact with and obtain information about the current active tab.

These permissions are standard for network monitoring extensions and are essential for uBO Scope's core functionality.

## Best Practices Post-Installation

- **Keep Your Browser Updated:** Ensure your browser version matches or exceeds the supported version for best compatibility.
- **Do Not Disable uBO Scope:** For accurate connection monitoring, uBO Scope must be enabled and allowed to run in your browser.
- **Check Badge Counts Regularly:** They provide immediate insights into third-party network exposure per tab.


## Troubleshooting Installation

If installation issues arise:

- Verify your browser version matches the minimum requirements.
- Check your internet connection during installation.
- Try reinstalling or restarting your browser.

For detailed troubleshooting, consult the [Troubleshooting Common Setup Issues](../getting-started/first-run-and-validation/troubleshooting-install) page.
