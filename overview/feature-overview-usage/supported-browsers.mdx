# Supported Browsers & Compatibility

## Discover Where and How You Can Use uBO Scope

uBO Scope is a lightweight browser extension designed to reveal all connections a web page tries to establish with remote servers. This page clarifies which browsers support uBO Scope, how to install it safely, and what privacy and compatibility assurances you can expect.

### Why This Matters to You

Imagine loading your favorite website and instantly knowing every third-party server your browser is communicating with—without guessing or missing hidden connections. uBO Scope empowers you by making these connections transparent, helping you gain control over your digital privacy.

Whether you're a privacy advocate, developer, or filter list maintainer, knowing the extension’s supported environments and installation methods ensures you can rely on it where it matters most.

---

## Compatible Browsers

uBO Scope supports major modern browsers, focusing on those that provide necessary APIs to observe network requests reliably.

- **Chromium-based browsers (e.g., Google Chrome, Microsoft Edge)**
  - Requires version 122.0 or later due to dependency on Manifest V3 features.
- **Mozilla Firefox (desktop and Android)**
  - Requires version 128.0 or later.
- **Apple Safari**
  - Supports version 18.5 and later.

These versions reflect the minimum requirements to ensure full functionality and security.

<Info>
If you use a different browser or an earlier version, uBO Scope may not operate correctly or could be missing key features.
</Info>

### Browser-Specific Permissions

uBO Scope requires permissions essential for monitoring network activity:

- `webRequest` to capture network request events
- `activeTab` to access information about the current tab
- `storage` for saving session data locally

Additionally, host permissions include broad URL patterns (`http://*/*`, `https://*/*`) to observe requests from all web origins.

## Installing uBO Scope

### Installation Sources

You can install uBO Scope from trusted official stores:

| Browser                   | Installation Link                                                                 |
|---------------------------|----------------------------------------------------------------------------------|
| Chromium-based Browsers    | [Chrome Web Store](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp) |
| Mozilla Firefox           | [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ubo-scope/)            |

These official pages ensure you receive the authentic, latest extension version.

### Installation Steps (Example for Chrome)

1. Navigate to the [Chrome Web Store uBO Scope page](https://chromewebstore.google.com/detail/ubo-scope/bbdpgcaljkaaigfcomhidmneffjjjfgp).
2. Click **Add to Chrome**.
3. Confirm permissions when prompted.
4. The extension icon will appear on your toolbar.

<Note>
For Firefox, installation is similar via the Firefox Add-ons page. For Safari, visit the Mac App Store or Safari Extensions site as appropriate.

### Post-Installation

- Restart the browser if necessary to enable all features.

## Privacy & Security Assurances

uBO Scope is designed with privacy as a top priority:

- **No data leaks:** uBO Scope solely monitors network requests locally within your browser. It does not send any user or browsing data externally.
- **Transparency:** Being open source and lightweight, its behavior can be scrutinized on [GitHub](https://github.com/gorhill/uBO-Scope).
- **Minimal permissions:** Requests only the bare minimum permissions needed to observe network requests for accurate reporting.

<Check>
Your third-party connection data stays private and under your control.
</Check>

## Compatibility Tips & Best Practices

- **Update your browser:** Always run a supported version to get the best performance and security.
- **Avoid conflicting extensions:** Other extensions overriding network requests could impact uBO Scope’s accuracy.
- **Check permissions if issues arise:** Ensure permissions are granted and not blocked by browser policies.

<Tip>
For the best experience, combine uBO Scope with uBlock Origin or other blockers but remember uBO Scope reports network outcomes independently.
</Tip>

## Troubleshooting Installation Issues

- If you do not see the badge icon after installing:
  - Refresh your browser or restart it.
  - Check extension settings and permissions.
  - Confirm your browser version meets the minimum.

- If uBO Scope does not report connections:
  - Ensure no other network-blocking extensions interfere.
  - Verify that the extension is enabled on the active tab.
  - Review your browser's extension management page for error messages.

Detailed troubleshooting guidance is available on the [Troubleshooting Setup Issues](/getting-started/setup-support/troubleshooting-setup) page.

## Quick Reference Installation Code Snippet

For users who prefer developer mode installation or validating version, here is a quick check via manifest snippet:

```json
{
  "manifest_version": 3,
  "minimum_chrome_version": "122.0",
  "permissions": ["activeTab", "storage", "webRequest"],
  "host_permissions": ["http://*/*", "https://*/*", "ws://*/*", "wss://*/*"]
}
```

This ensures the extension uses the WebRequest API for full network monitoring capabilities required by uBO Scope.
