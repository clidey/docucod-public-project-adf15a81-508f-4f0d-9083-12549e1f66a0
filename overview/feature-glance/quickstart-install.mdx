# Quickstart & Installation Links

## Get uBO Scope Up and Running Quickly

uBO Scope offers a straightforward path to begin uncovering all remote server connections your browser makes. This page guides you through installing the extension on your preferred browser and highlights essential first steps so you can start monitoring third-party network connections without delay.

---

### Why Install uBO Scope?

Imagine browsing your favorite website and instantly knowing every remote server it touches—whether those connections were allowed, blocked, or stealthily redirected. uBO Scope puts this transparency at your fingertips with minimal setup.

Key benefits immediately unlocked when installed:

- **Comprehensive Connection Reporting:** See all remote servers your browser reaches out to per tab.
- **Independent of Content Blockers:** Works alongside your current blockers, providing unbiased insight.
- **Compact Toolbar Badge:** Quick glance to assess third-party connections count—lower is better.

## First Steps After Installation

Once you add uBO Scope to your browser, you are ready to start exploring your network connections:

1. **Activate the Extension:** Click the uBO Scope icon in your browser toolbar.
2. **View Popup Panel:** The popup shows an immediate summary of remote domains connected by the active tab, categorized by connection outcome.
3. **Interpret Badge Count:** The badge number on the toolbar icon reflects the count of distinct third-party domains successfully connected.

### What to Expect

- Real-time reporting of network requests as you browse.
- Domains grouped into "not blocked," "stealth-blocked," and "blocked" categories.
- No need for additional configuration for the default usage scenario.

---

## Installation Tips & Best Practices

- **Keep Your Browser Updated:** Ensure your browser meets minimum version requirements to avoid compatibility issues.
- **Review Permissions:** This extension requires permission to monitor web requests and active tabs, which it uses solely to provide connection transparency.
- **Verify Popup Access:** Clicking the toolbar icon should open the popup panel showing connection summaries for the active tab.

<Warning>
If the popup does not display data, refresh the active web page or ensure the extension is enabled. Also, verify that other privacy extensions do not block webRequest API access.
</Warning>

## Troubleshooting Quick Tips

<AccordionGroup title="Common Installation and Startup Issues">
<Accordion title="Extension Icon Not Visible">
- Confirm installation completed successfully.
- Check browser extension management page to enable uBO Scope.
- Restart your browser if necessary.
</Accordion>

<Accordion title="Popup Panel Shows 'NO DATA' or Blank">
- Reload the active tab to trigger new network activity.
- Verify the page you are testing is loading remote resources.
- Ensure browser supports webRequest API fully (some platforms restrict certain URLs).
</Accordion>

<Accordion title="Badge Count Not Updating">
- Close and reopen tabs to reset session tracking.
- Confirm you are browsing websites that initiate network connections.
- Wait a few seconds after opening a tab for the badge to update.
</Accordion>
</AccordionGroup>
